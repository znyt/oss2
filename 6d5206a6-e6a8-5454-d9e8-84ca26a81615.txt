        “把暖气打开！”傅夜沉疾言厉色地命令。
<br/>
<br/>        坐在驾驶座上的司机，来不及去探看后座是什么情况，立马按开了空调键，调到送热风状态。
<br/>
<br/>        童筱颖的身子一直在发抖，束成马尾的长发，像条黑色的水蛇，盘绕在她的颈项、肩头、胸前。
<br/>
<br/>        她的胸口，随着她抽泣的动作，起伏得厉害。
<br/>
<br/>        傅夜沉见状，慌忙脱下自己的外套，将童筱颖的身子裹住，大手一揽，便将她紧紧地搂入了怀中。
<br/>
<br/>        他搂着她的肩头，另一只大手轻轻地按着她的脑袋埋入自己的颈窝。
<br/>
<br/>        童筱颖的眼泪，浸湿了他的衣领，带着她的体温，让他感到无比的心痛。
<br/>
<br/>        “筱颖，对不起……”他在跟她道歉。
<br/>
<br/>        在这之前，他的心很迷茫，在这之后，他的心不再迷茫了。
<br/>
<br/>        三年前，他的成全，并未给她带来幸福。
<br/>
<br/>        与其把她的幸福，交付于别人，倒不如由他来给她创造。
<br/>
<br/>        傅夜沉捧住童筱颖的脸颊，强迫她与自己四目相对。
<br/>
<br/>        她哭得梨花带雨，如一翦秋水般的眸子里映着他英俊的容颜。
<br/>
<br/>        童筱颖吸了吸鼻子，闻到了从傅夜沉身上传来的淡淡薄荷香。
<br/>
<br/>        这种香气，揉合着某种她无法形容的男性气息，那么醇厚、那么浓冽，让她心里很是迷茫。
<br/>
<br/>        她的小手，可怜兮兮地揪着他衣襟，就像一只受了极大委屈的小猫。
<br/>
<br/>        傅夜沉凝视她的眼睛，幽深的墨瞳闪着光，透着一抹无法言说的深情。
<br/>
<br/>        “筱颖，乖，不哭了。再哭下去，都快变成小花猫了。”他低声温柔像哄孩子一样哄着她，大手轻轻扒梳她脸颊上纤柔的发丝。
<br/>
<br/>        可是，童筱颖仍旧觉得心好痛，眼泪不受控地从眼眶里溢出。
<br/>
<br/>        他看着她因为哭泣而惊颤不止的羽睫，看着在她因为哭泣双颊渲染上两抹粉晕。
<br/>
<br/>        心痛、心怜、甚至，情难自禁……
<br/>
<br/>        傅夜沉的臂膀紧了紧，大掌托住她后脑勺，让她更贴近他的胸膛。
<br/>
<br/>        “筱颖，再哭的话，我要吻你了。”
<br/>
<br/>        他的话语一落，不等童筱颖反应过来，那清冽的薄唇已经含住她的唇瓣，舌尖湿润地描绘她柔软的唇缘，跟着诱她分开，汲取唇腔的甜蜜。
<br/>
<br/>        童筱颖瞬间瞪大了眼睛，晕眩地领受着他这番温柔似水的暧昧。
<br/>
<br/>        他的吻很轻，很柔，不带任何情欲，从容不迫地诱导着她慢慢地接受，也成功地止住了她的哭泣。
<br/>
<br/>        驾驶座上的司机，一边开车，一边无意间地看了一眼后视镜，忍不住嘴角微扬。
<br/>
<br/>        想不到平日里雷厉风行又高冷的大Boss，也有如此温柔的一面。
<br/>
<br/>        他还是第一次见自己的上司，对一个女人如此有耐心。
<br/>
<br/>        “再哭，我就继续吻你，吻到你不哭为止。”傅夜沉的峻唇浮起一抹连他自己也未察觉的笑意。
<br/>
<br/>        童筱颖吸着鼻子，轻轻喘息，抿了抿唇，不再哭泣。
<br/>
<br/>        他宠溺地在她鼻梁上轻轻地刮了刮，再次将她的身子搂紧：“我的筱颖真乖！”
<br/>
<br/>        下车后，傅夜沉拽着童筱颖的手腕，一路从地下室乘坐VIP专属电梯，直达的傅氏集团大厦顶楼的总裁办。
<br/>
<br/>        总裁办二楼，是他的私人休息房。
<br/>
<br/>        童筱颖几乎是用小跑步的方式，才有办法跟得上长腿的傅夜沉。
<br/>
<br/>        但此刻，她的思绪混乱，或许是因为淋了雨，受了凉，她只觉脑袋昏沉，像是遭到炮火重击过一般。
<br/>
<br/>        浴室门“啪”地一声，自身后关上，童筱颖仍旧有些晕头转向，双眸呆滞。
<br/>
<br/>        她鬈翘的长睫向上一掀，无力地望着身前这个总是对她意乱情迷的男人。
<br/>
<br/>        他突然强势地抓住她的双肩，将她压在墙上，无意间撞开了身后花洒的开关，瞬间，无数滴热气腾腾的水珠从天而降，将他和她再次淋湿。
<br/>
<br/>        她身上的白色薄衫几近呈半透明状，紧贴着肌肤，变成了朦胧的肉色。
<br/>
<br/>        “你先洗，我去给你拿衣服。”傅夜沉的喉结微微滑动，他闭了闭眼睛，转身拉开了浴室门，只身走了出去。
<br/>
<br/>        童筱颖此刻的心情，稍稍好转，一个热水澡冲去了她身上的疲惫与寒意，却无法抚平她那颗千疮百孔的心。
<br/>
<br/>        她以为，经过这段时间在老家的沉淀，能平复自己的心情，让自己暂时不去想小卫的事情。
<br/>
<br/>        但终究，她还是败给了季思妍的那封电子邮件。
<br/>
<br/>        小卫，已经不再是她的小卫了……
<br/>
<br/>        童筱颖洗完澡后，在浴室里吹干了头发，穿着傅夜沉递给她的他自己的白衬衫走出来的。
<br/>
<br/>        她依稀记得，她好像并不是第一次穿他的白衬衫。
