　　此话，并不属假，大陆近千年历史中，并不泛这种依靠奇遇而成为强者的故事。
<br/>　　这个故事所造成的后果，便是造就了大批每天等在悬崖边，准备跳崖得绝世功法的怀梦之人，当然，这些人大多都是以断胳膊断腿归来。
<br/>　　很显然，眼前这口隐蔽的山洞，便是无名隐士坐化的地方。
<br/>　　只见小医仙脚尖在山崖上借力，弹射而起后，就在她以为要抵达洞口的时候，忽然一道破空声在夜空中响起，使她骤然色变。
<br/>　　“岩蛇！”
<br/>　　黑夜对于慕清霄来说，与白昼无异，一眼就认出了居住在悬崖上的魔兽。
<br/>　　岩蛇，顾名思义，这是一种生活在岩壁中的蛇形魔兽，级别在一阶左右。
<br/>　　这种魔兽，借助着身体扁长如翅的缘故，能够犹如猎鹰一样在空中进行翱翔，而且由于魔兽属性是一种变异的石属性，所以其身体坚硬如石，普通刀剑，根本难以对其造成生命危险。
<br/>　　此时，小医仙脸色惨白，她完全没想到，在这距离洞口两米处，居然有一条岩蛇。
<br/>　　“嗤！”
<br/>　　岩蛇盘旋在空中，三角眼释放着血腥寒光，嘶鸣一声，狭窄的双翼一阵，目露凶光对着小医仙俯冲而下，血腥的口中，尖锐的獠牙泛着森冷的光泽。
<br/>　　岩蛇速度极快，小医仙的小脑袋仿佛停止思考，顿时吓得浑身无力，握着粗绳的玉手也松了开来，娇躯猛地向悬崖下坠去，正巧躲开了岩蛇的攻击。
<br/>　　完了！
<br/>　　小医仙美眸中满是绝望，渐渐闭上双眼，就在她绝望之际，身躯居然定格在空中，却没有继续坠下，掌心也含有一丝温暖。
<br/>　　睁开双眼一看，却见慕清霄不知何时出现在她上面，神骏的脸颊上依旧是一副淡然的笑容，此时正伸手拉着她的胳膊，至于岩蛇早已消失不见。
<br/>　　呆泄两秒钟后，懵逼中小医仙回过神来，生气道：“你跟踪我！”
<br/>　　慕清霄手臂一拉，小医仙身躯一颤，因为她感觉到，一只手掌已经搂在了她的纤细的腰间，绝美的脸颊上瞬间布满红晕，煞是可爱。
<br/>　　体会到怀中柔腻的娇躯，那有些犯规的纤腰，慕清霄微笑道：“我要是不跟过来，你现在可消香玉陨了，直接从这里掉下去，恐怕会直接变成肉饼啊。”
<br/>　　闻言，小医仙一颤，顿时一阵后怕，心间的一丝怒意也消失不见，美眸中有一丝歉意，回过神来，道：“对了，刚才的岩蛇呢？”
<br/>　　指了指深不见底的漆黑悬崖，慕清霄道：“掉下去了吧，我们先上去在说，搂紧了。”
<br/>　　听到慕清霄的话，小医仙略微有些迟疑，不过当慕清霄脚尖在山壁上一蹬，身子瞬间甩动起来。
<br/>　　怀中的佳人惊得赶紧一把抱住慕清霄的腰，将绝美的脸颊埋在后者怀中，动也不敢动一下，暗道：“咦，有股淡淡的清香，真是好闻。”
<br/>　　慕清霄微微借力，握着粗壮的绳子就荡了过去，轻飘飘的落在地上，低头望着怀中脸颊微红的小医仙，坏笑道：“这样抱着很舒服吗？”
<br/>　　“嗯。”
<br/>　　小医仙微微点头，但抬起头来却见慕清霄神骏的脸颊上噙着一丝坏笑，顿时心乱如麻，美眸狠狠的瞪了他一眼，快速挣脱他的怀抱。
<br/>　　“你是如何发现我的？”
<br/>　　“白天就觉得你有些不对劲，在小小的青山镇，将价值四千金币的白兰果送人，本就是疑点，更何况你是顺着悬崖爬上来的。”
<br/>　　说罢，慕清霄望着洞口出密布的碎石与怪木，随手一挥，一股劲风就携带者碎石怪木滚落山涧。
<br/>　　没有了树木与碎石的遮盖，借着淡淡的月光，终于看见了这所隐藏在山崖下的山洞，或者说是洞府。
<br/>　　“现在知道了吧？想吃独食就要量力而行，光是洞口的碎石和树木就够你清扫一阵子，哪怕你从岩蛇嘴中活下来，消耗体能的你，估计到早晨也不可能清扫完毕吧？”
<br/>　　听到慕清霄平淡的话语，不仿佛有魔力参杂其中，小医仙竟乖巧的点了点头，但回过神来却是瞪了他一眼，有些兴奋的打量着洞口。
<br/>　　“看来是前人的洞府，我们进去看看吧，说不定有什么好东西。”
<br/>　　反正已经暴露了，想要吃独食是不可能了，小医仙也只能苦着小脸同意，况且慕清霄刚才救了她的性命。
<br/>　　只是见自顾自的走进漆黑的山洞，小医仙有些踌躇，片刻后，跺了跺脚，咬着银牙跟了上去。
<br/>　　......
<br/>　　......
<br/>　　行走在幽静而黑暗的山洞隧道中，一丝寒意缭绕在周身，安静的隧道中，只有两人细微的脚步声。
<br/>　　周围阴暗的环境，使得小医仙双臂不自主的互相抱了抱，抬头望着走在前面的慕清霄，略微迟疑，随即加快步伐，仅仅的跟在他的后面。
<br/>　　在这种情况下，也只有前面的青年，能够使她多出一丝安全感。
<br/>　　周围死一般寂静的氛围，黑暗的能够使人发疯，在其中行走足足十几分钟，小医仙实在有些受不了，就在这时，慕清霄忽然停住了脚步。
<br/>　　“呀...”
<br/>　　由于没有反应过来，最后撞在慕清霄结实的后背上，小医仙惊叫一声，胸前两团柔软的挺翘，在压力的作用下，顿时在慕清霄背上压缩成了两团柔软的小圆球。
<br/>　　亲密的接触，使得小医仙俏脸绯红的急退一步，羞恼道：“你干嘛呀？”
<br/>　　体会到刚才的两团柔软，慕清霄轻轻吸了口凉气，没想到这小妮子蛮有料的啊，若无其事的指着前面一扇散发着淡黄色光晕的石门，道：“到了。”
<br/>　　闻言，小医仙黛眉微蹙，上前两步，望着石门，沉吟道：“石门后面，估计就是藏宝室了，既然前人在这种地方铸造山洞...
<br/>　　我想，应该不会将来者困死在这里。”
<br/>　　慕清霄走上前来，只是瞥了一眼石门上的黄色光晕，不是阵法，那便是机关术了，于是视线在周围扫视起来。
<br/>　　“周围估计有开启石门的机关，找找看吧。”
<br/>　　“你懂得机关术？”小医仙有些惊讶的问道。
<br/>　　慕清霄微微点头，对于机关术他并不陌生，当初在秦时明月位面的时候，机关术见的太多了。
<br/>　　相比起蜃楼那种机关术与阴阳术的组合，面前的石门简直是小儿科。
