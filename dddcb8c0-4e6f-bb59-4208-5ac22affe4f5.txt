        乘务长顿时一脸诧异地看着宋云汐：“我觉得你挺好相处的啊！怎么可能会性格不合？再说了，你两才见面，没聊几句，怎么就性格不合呢？”
<br/>
<br/>        “其实吧！是我觉得他太优秀了，我配不上。”宋云汐随后又往咖啡里加了点点海盐。
<br/>
<br/>        “太优秀？莫非，以后是要当军长的吗？”乘务长又是一脸崇拜。
<br/>
<br/>        宋云汐泡好咖啡后，端起咖啡杯的托盘，看着乘务长会心一笑：“嗯，所以，我还是有自知之明的。那我先回驾驶舱了。”
<br/>
<br/>        “好，慢走，小心点咖啡，别泼了。”乘务长露出标准的国际微笑，好心叮嘱道。
<br/>
<br/>        宋云汐点了下头，端着咖啡，优雅地回到了驾驶舱。
<br/>
<br/>        此时，第二机长直接从宋云汐的手中夺走了咖啡，二话不说地喝了一口，惊赞道：“我们的宋司花泡的咖啡，就是特别好喝！”
<br/>
<br/>        宋云汐倒是淡定地笑了笑，然后对坐在副驾驶座上的师父说道：“师父，您的咖啡被个泼皮猴抢了，我再去给您倒一杯过来。”
<br/>
<br/>        “好！”副驾驶师父点了下头。
<br/>
<br/>        宋云汐睐了第二机长男一眼，又转身打开舱门，走出了驾驶舱。
<br/>
<br/>        她是个优雅与美貌并存的女人，与同僚的关系也处的不错。
<br/>
<br/>        宋云汐离开驾驶舱后，舱门关上时，副驾驶师父，就跟第二机长男闲聊了一句：“你就别想着追云汐了，云汐曾跟我说过，她是不会嫁给同僚的。”
<br/>
<br/>        第一机长男也忍不住插上一句：“说真的，女机长里，她是我见过的最高冷的一个。是眼光高的高，冷艳的冷。”
<br/>
<br/>        “也是。”第二机长男自讨没趣地应了声。
<br/>
<br/>        宋云汐再次回到茶水间里时，乘务长正在解决乘客与空姐之间的一点小摩擦。
<br/>
<br/>        当她看着咖啡的颜色时，无意间又想起了今天跟她相亲的那个男人。
<br/>
<br/>        其实，是她母亲要她去相亲的，相亲之前，母亲有资料给她看，但是她没看，只听说对方是个当兵的。
<br/>
<br/>        实话说，冷晏琛这个人，给她的第一印象其实还不错。
<br/>
<br/>        如果不是个人收入差异太大，或许她会考虑他的。
<br/>
<br/>        算了，不想了。
<br/>
<br/>        反正从今以后，两人说不定就不会再见面了。
<br/>
<br/>        宋云汐嘴角微扬，继续泡咖啡。
<br/>
<br/>        另一边，冷晏琛回到自己的部队里后，完全没去想宋云汐的事情。
<br/>
<br/>        或许，他们只是在一个不好的时机里相遇。
<br/>
<br/>        翌日。
<br/>
<br/>        宋云汐飞完一趟航线后，又被自己的母亲逼着去相亲了。
<br/>
<br/>        好像在她的母亲眼里，自从她被前男友甩了后，她就再也嫁不出去了一样。
<br/>
<br/>        其实，母亲一开始给她安排相亲，她是很反感的，后来，若不是母亲“一哭二闹三上吊”，以及每天神棍念经一样的电话炮轰，她还真不会如此应付母亲的安排。
<br/>
<br/>        这次相亲的地点，是在北城中心医院附近的一家茶馆。
<br/>
<br/>        时间地点都是对方约的。
<br/>
<br/>        一踏进茶馆的大厅，宋云汐的目光立刻锁定在中央一处雅座上。
<br/>
<br/>        毕竟，此刻这大厅的雅座里，只有一个男人坐在那儿，让她不去注意都挺难。
<br/>
<br/>        男人留着一头利落的板寸头，穿着白色衬衫，外套了一件黑色休闲西装外套，正低着头，看手中的手机。
<br/>
<br/>        宋云汐走过去后，直接在男子对面落座。
<br/>
<br/>        男子这才抬起头来，随即收起了手中的手机。
<br/>
<br/>        只见宋云汐坐姿随性却很优雅，似乎对于今天的这个相亲完全不紧张。
<br/>
<br/>        毕竟，她都不知道自己跟多少男人相过多少亲了，所以，没什么好紧张的。
<br/>
<br/>        不过话说回来，也有可能她只是在应付这件事情。
<br/>
<br/>        男子打量完宋云汐，嘴角礼貌而客气地扬起一抹弧度，主动问候道：“宋小姐，你好。”
<br/>
<br/>        “陈先生，你好。”宋云汐浅浅地笑了笑。
<br/>
<br/>        陈先生接着向服务生招了招手，服务生随即送上两本菜单，分别递给他们俩。
<br/>
<br/>        “我要一杯白开水即可。”陈先生看都不看菜单，直接跟服务生说道，说完后，又看向宋云汐，温文儒雅地迎了迎手，“宋小姐，你随意。”
<br/>
<br/>        既然他都点了一杯白开水了，宋云汐索性直接合上菜单，微笑着对服务生说道：“我也要一杯白开水即可。”
<br/>
<br/>        毕竟，现在不是饭点。
<br/>
<br/>        “我只是觉得和白开水比和任何茶饮对身体有益。”陈先生解释道。
<br/>
<br/>        宋云汐坦然一笑：“我知道，毕竟，陈先生是一名主刀医生。”
<br/>
<br/>        “既然，我们彼此已经知道彼此的职业和工薪了，那我们直接谈一些如果能处在一起时相关的事宜，你看看如何？”陈先生直接开门见山道。
