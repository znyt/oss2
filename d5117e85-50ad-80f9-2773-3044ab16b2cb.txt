　　不过，自己不是老人，还没到只能回忆和追悔的时候，自己还有能力去改变这一切，哪怕是看似不可能做到的。
<br/>　　每看一次照片，叶少阳就多一分对自己那个时代的思念。冷玉，还有小九。
<br/>　　第二天，三个汉子一起去街上买了三匹马（反正现在就是有钱），跟两个姑娘告别，翠云和云儿恋恋不舍地送他们离开，临走时，给三人口袋里装的满满的都炒的化生瓜子核桃之类，让他们路上吃，消磨时间。
<br/>　　三人快马加鞭，在官道上奔驰，一路前往杭州去。
<br/>　　路上停下来休息，三人在路边坐着喝水，叶少阳瞅着毛小方的长衫，凑近看了半天，道：“我记着你这衣服这里不是撕了好长一个口子吗，之前古墓里弄的，什么时候缝上的。”
<br/>　　“这……这是我昨天晚上缝的。”毛小方脸色顿时有点尴尬。
<br/>　　“你缝的？哈哈，我差点就信了。”叶少阳拍着他的肩膀，一脸坏笑，“你什么时候这么好的针线活了，来来正好，我衣服也破了，来帮个忙。”
<br/>　　“去你的，你衣服哪里破了！昨天翠云不是帮你缝了，还来挤兑我！”
<br/>　　“翠云是我干姐姐啊，给我缝衣服不是正常的吗，倒是你……”
<br/>　　“云儿也算我徒弟，给师父效劳，也是理所当然！”毛小方瞪着两眼。
<br/>　　叶少阳一只手搭在他肩膀上，挤眉弄眼地笑道：“是么？”
<br/>　　毛小方不理他，转头对道渊真人笑道：“你看这个人，一点正经没有，他还说他是灵仙牌位，我都不信的，哪里有这个样子的灵仙！”
<br/>　　“你啥意思，想让我在你面前摆架子啊。”叶少阳磕着瓜子，蹲在地上，冲毛小方坏笑着，偏偏身上穿着长衫，活脱脱一个神棍的模样，的确是没有一点得道高人的感觉。
<br/>　　“跟你说正经的，我跟云儿姑娘真的没有什么，她以前被蛤蟆精欺负过……”
<br/>　　“所以你嫌弃她？”
<br/>　　“哪来的话。”毛小方怼了他一拳，“听我说完，我意思是说，因为她以前被蛤蟆精欺负过，对这些邪修的精怪充满仇恨，想到要去降妖除魔，一个劲缠着要拜我为师，所以对我很好，就是这么回事，你这人这么心术不正，啥事都往那方面想。”
<br/>　　叶少阳托着下巴，问道：“那为什么不找我？”
<br/>　　“你看上去没我老实。”
<br/>　　叶少阳嘴角抽搐了一下，“就这？”
<br/>　　“就这！她说看你就是闲不住的人，而且好像有很多事情要去做，不可能停留下来，我不一样，所以找了我，有问题吗？”
<br/>　　“没了。”叶少笑了笑，他相信自己的直觉，云儿对毛小方是真的有点意思，可能也是觉得自己不够干净，所以不敢有什么非分之想，这才推求其次，想要做他的徒弟，也是可以理解。
<br/>　　道渊真人大马金刀地坐在一块石头上，表情严肃，磕着瓜子，说道：“你们两个都有姑娘心疼，就我这孤家寡人一个。”
<br/>　　叶少阳蓦然想起那个时代的道渊真人，好像是没有媳妇的，也不知道是活得太久，媳妇先死了，还是压根就没娶媳妇，也不好跟他透露，随口说道：“你干嘛不去自己找一个。”
<br/>　　道渊真人嘿嘿一笑，“我就是这么一说，我当年立下宏愿，此生修道，绝不娶妻。一个人挺好的，清静。”
<br/>　　“你要是二十年后，还坚持这个想法，我就服你。”毛小方道。
<br/>　　叶少阳知道，他是真的坚持下来了……
<br/>　　三人有说有笑，没有一点隔阂，叶少阳心中却很是感怀，没想到自己穿越到一百年之前，居然还能找到两个这么好的朋友，不，应该说是同患难过的兄弟了。
<br/>　　就算自己将来回去了，也一定会记住他们。
<br/>　　从江西前往杭州，路程遥远，三人虽然骑马，走了两天，也只是到了江西的东北角，著名的景德镇。
<br/>　　在路上，叶少阳就听两个兄弟普及了一下，景德镇在这个时候还是一个县，瓷器是很出名，但是城市规模也不太大，但因为瓷器的缘故，有客商往来县城，因此比周边的县城要繁华一些。当时天色有点晚，三人打算进城休息一晚，明天早上再出发。
<br/>　　进城的时候，三人发现城门下面有很多人围着一张告示在看，这一幕很像电视里经常演的那样，城墙上贴着通缉犯的名字，很多人围观……
<br/>　　叶少阳看到那张纸上有“通缉要犯”四个字，下面有画像，但因为天黑看不清，叶少阳本想去看个热闹，但是人太多不好挤过去，干脆算了，去街上找了一家客栈住下。
<br/>　　这个时代的客栈，一般是不需要登记的，只要报个名字就行，方便收信和有人找。叶少阳三人报了名字，掌柜的听见“叶少阳”三个字，眉头皱了一下，盯着他打量了一会。
<br/>　　“怎么了？”叶少阳摸着自己的脸，问道。
<br/>　　“没什么，没什么，客官，里面请。”掌柜的满脸堆笑，给三人开了房间。
<br/>　　晚上，三人一起随便吃了点东西，回到房间里，叶少阳本想吐纳修炼，结果刚坐下，外面就响起脚步声，叶少阳听出是四个人，其中有三个人走得很轻，一个人脚步声很重，心下正在狐疑，敲门声响了。
<br/>　　“客官，需要续水吗？”外面是客栈掌柜的声音。
<br/>　　续水需要四个人？
<br/>　　叶少阳心中狐疑，难不成自己遇到传说中的黑店了？
<br/>　　叶少阳本来不需要续水，但是为了一探究竟，还是答应了一声，过去打开门，结果门刚打开，外面就冲进来三个彪形大汉，上来就把他按住。
<br/>　　叶少阳本来想反抗的，转头一瞥，这三人都穿着警察制服——这个时代的警察跟百年后当然不同，不过叶少阳之前见过，跟电视里演的差不多。当时犹豫了一下，没有动手，问道：“你们搞什么？”
<br/>　　“老实待着！”其中一个警察提着一盏油灯，照着叶少阳的脸，问掌柜：“可是这人？”
<br/>(本章完)
