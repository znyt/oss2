　　这一吻，时间更久！
<br/>　　直到外面已经响起了敲门声，叶尘这才将怀里的‘萧若曦’缓缓放开。
<br/>　　“曦月，你之前不是一直说，想要来我的家乡看一看吗？这里就是我的家乡，地球！”
<br/>　　“外面的人，就是我的姐姐苏蔓，我先让她进来，和你认识一下！”
<br/>　　‘萧若曦’点了点头，同时好奇的在周围观察起来，眼眸中透着惊喜之色，
<br/>　　如今萧若曦的身体，为曦月所掌控，看到地球上的一切事物，都觉得很是新奇。
<br/>　　之前在修真界，她经常听叶尘提及地球上的事情，一直想来看一看，可惜没能如愿。
<br/>　　叶尘随手一挥，病房的房门，便已经自动打开，苏蔓和吕国胜等人，也走了进来。
<br/>　　“咦！萧若曦，你醒了？”
<br/>　　苏蔓看到‘萧若曦’已经坐了起来，当即快币来到床前，满脸的喜色。
<br/>　　至于吕国胜，早已经目瞪口呆，
<br/>　　“他居然真的治好了？这……这怎么可能！”
<br/>　　吕国胜号称是全华夏最好的脑科专家，用了大半天时间，连病因都没有查出来，结果却被眼前的毛头小子，这么快就给治好了，这简直就是赤裸裸的打脸啊！
<br/>　　虽然对方并没有追究的意思，吕国胜也没脸再呆下去了，当即灰溜溜的离开了。
<br/>　　“这就是我姐姐，苏蔓！”
<br/>　　叶尘通过神念，向曦月介绍道。
<br/>　　曦月当即露出一抹和善的笑容，伸手拉住苏蔓的玉手，笑道：
<br/>　　“苏曼姐姐，你好！”
<br/>　　苏蔓顿时愣住了，甚至都怀疑自己，出现了幻觉！
<br/>　　“你……刚刚……叫我什么来着？”
<br/>　　原来，她们两个人向来不对付，虽然苏蔓要比萧若曦大上一岁，可是萧若曦从来都是直呼其名。
<br/>　　如今，‘萧若曦’竟然喊她苏蔓姐姐！
<br/>　　这简直就是破天荒的事情啊！
<br/>　　曦月又哪里知道这些，还以为自己说错了什么话，顿时有些不知道该如何回答。
<br/>　　不料，苏蔓忽然双手叉腰，扬天大笑，颇为得意，
<br/>　　“冲着你这声姐姐，以前的那些事情，我就不和你计较啦！”
<br/>　　说完这话，苏蔓似乎又想起了什么，
<br/>　　“对了！小尘，曦月，我昨天刚刚在一家外企，应聘成功了一份实习生的职位，今天第一天上班，就旷工了大半天，要是再不回去，恐怕就有点说不过去……”
<br/>　　苏蔓如今上大四，已经开始实习。
<br/>　　叶尘听到这话，正求之不得，当即道：
<br/>　　“蔓姐，你去忙吧，曦……若曦交给我照顾就行了！”
<br/>　　苏蔓顿时眉头微微一挑，隐约发现叶尘对萧若曦的态度，似乎和之前不一样了，心里竟然莫名有些嫉妒……
<br/>　　‘苏蔓啊苏蔓！你想什么呢？他是你的弟弟，如果他真的和萧若曦在一起了，你就应该祝福他们！’
<br/>　　一念及此，苏蔓强行将心头的那一抹嫉妒，生生压制了下去，冲着‘萧若曦’挤了挤眼睛，道：
<br/>　　“萧若曦！我弟弟可不是一般人，你要好好珍惜哦！”
<br/>　　说完这话，便转身离开了。
<br/>　　叶尘顿时一阵好笑，
<br/>　　萧若曦体内藏着曦月的残魂，这件事情实在过于匪夷所思，并非叶尘刻意隐瞒，实在是根本无法解释。
<br/>　　待苏蔓离开之后，叶尘重新坐回曦月的旁边，很自然的想要将她揽入怀中。
<br/>　　不料，曦月却躲开了，气鼓鼓的望着叶尘，
<br/>　　“你老实告诉我，你和我这幅躯体的主人，是什么关系？苏蔓姐姐方才那话，又是什么意思？”
<br/>　　叶尘顿时苦笑不得，
<br/>　　吃醋似乎是所有女人的本能，连超然物外的曦月仙子，也同样不能免俗。
<br/>　　叶尘不由得伸手，在她的小脑袋上戳了戳，
<br/>　　“我的仙子大人，你想什么呢？我心里已经有了你，又怎么可能会喜欢上别的女人？”
<br/>　　叶尘说到这里，微微一顿，眼眸中忽然又闪过一抹邪意，
<br/>　　“曦月，反正你现在肉身也不在了，要不直接夺舍……”
<br/>　　这道邪念，当初叶尘就曾经想过，虽然这么做，有些对不住萧若曦，可是为了自己挚爱的女人，叶尘可管不了这么多！
<br/>　　萧若曦，绝对是曦月夺舍的最佳躯壳！
<br/>　　然而，
<br/>　　曦月却无比坚决的拒绝了，
<br/>　　“不行！我现在占着她的躯体，就已经亏欠她不少了，怎么还能做那种事情？”
<br/>　　“而且，我隐隐感觉到，我的残魂，似乎一直在受规则之力的影响！”
<br/>　　叶尘听到这话，顿时一惊，
<br/>　　“此话怎讲？”
<br/>　　曦月修为微蹙，缓缓道：
<br/>　　“具体的原因，我也说不上来，不过我曾经在古籍中看过，月光宝鉴虽然可以将人带回过去，但只能以灵魂的状态穿越，并且会回到自己的肉身之上……”
<br/>　　“可是现在，我的残魂，却跟随你来到了地球！”
<br/>　　“甚至，我都无法感应到我的肉身！”
<br/>　　“这说明，眼下的地球，和修真界之间，似乎被某种规则之力所隔绝！”
<br/>　　叶尘眉头微微一皱，
<br/>　　“就算是这样又如何？你只要神魂足够强大，然后再找一具合适的肉身夺舍，早晚能恢复昔日的修为！”
<br/>　　不料，曦月却摇了摇头，道：
<br/>　　“不是这样的！我之所以会被规则之力所排斥，就是因为我不属于这个世界，我的实力越强，排斥之力也就越强，早晚会不容于这个世界！”
<br/>　　“而且，根据古籍中关于月光宝鉴的记载，在同一个时空之下，应该不能同时存在两个主体意识！”
<br/>　　叶尘听到这话，顿时脸色一变，
<br/>　　“曦月，你的意思是……”
<br/>　　曦月点了点头，道：
<br/>　　“如果我没有猜错的话，修真界中的那个我，意识恐怕应该已经开始消散了！”
<br/>　　说到这里，曦月又深吸了一口气，泛着智慧的美眸之中，又露出一抹决然之色，
<br/>　　“所以，我的魂魄，必须要回到修真界！”
<br/>　　“否则，另外一个世界的我，将会逐渐丧失意识！”
<br/>　　“而这个世界的我，也会被规则之力所排斥，早晚也是死路一条！”
<br/>　　听完曦月的一番解释，叶尘顿时脸色无比凝重，
<br/>　　“我明白了！”
<br/>　　“曦月，你放心，我已经找到返回修真界的入口，随时都能带你回去！”
