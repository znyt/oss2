　　接下来的这段时间，他们把一切的活动全都推掉了，大家都是将注意力放到了英雄联盟的身上，不光是为了全国人民的期盼，就是连他们自己对于这个冠军也是非常渴望的。
<br/>　　职业生涯毕竟也就仅仅只有这么几年，所以能够在自己的生涯上拿下一个冠军或者是两个，这绝对是非常浓重的一笔的，起码那些拿下冠军的，都会在电竞史上留名。
<br/>　　因为他们已经打过比赛了，所以今天他们倒是咩有机会和皇族打训练赛，一下午的时间叶帝五人都是在排位赛中度过，几场排位赛下来，他们遇到的大多是其他战队的选手。
<br/>　　到了这个时候大家想要保持手感，这种高强度的排位赛自然是在适合不过了。
<br/>　　而就在叶帝结束今天的训练的时候，家里的电话却是给他带来了另外一个好消息，让他久久不能平静。
<br/>　　还有一个多月苏沐婉就要生了，也就是说在世界赛结束后叶帝就要当爸爸了。
<br/>　　当基地内的众人得知这个消息后，自然又是一番庆祝。
<br/>　　“看来我也要抓紧找个女朋友了，这样的话如果我生孩子咱们也能弄个娃娃亲，先说好了啊，大家都别抢。”听到这个消息，蒋念连忙开口说道。
<br/>　　“卧槽，你怎么突然间变的这么机制了，我敢保证你肯定是看上叶哥家的后台了。”香蕉一脸佩服的说道。
<br/>　　“你还是太年轻了，能和小叶子结为亲家，我的孩子这是要少奋斗好几百年了啊。”蒋念一脸得意的说道。
<br/>　　“便宜不能都让蒋念一个人占了啊，不管是男孩还是女孩我都要当干爹。”秦霄也是跟着起哄。
<br/>　　“我当干妈！”柳清月、林雅楠两人也是跟过来凑起了热闹，这个时候大家都很开心。
<br/>　　“居然把这个忘了，不过看时间的话，应该是在领土战前后，不管怎么说到时候咱们一定要将领土战拿下，当做是给孩子的一个礼物。”李景禹跟着开口。
<br/>　　“最近在比赛，就不庆祝了，等到领土战结束后，咱们大家一定要不醉不归。”叶帝也很开心，在那里开口说着。
<br/>　　众人又聊了一会后，看着时间已经不早了，便是纷纷回到了自己的房间中，准备休息。
<br/>　　而回到房间后的叶帝，在和苏沐婉视频了将近半个小时后，才恋恋不舍的挂断了视频，虽然时间已经是快要接近凌晨了，不过叶帝这边却是没有丝毫的睡意。
<br/>　　听到自己有了孩子之后，两世为人的叶帝终于是在这个世界上有了自己的期盼，他终于感觉到自己彻彻底底的融入到了这个世界。
<br/>　　又过了许久，叶帝才感觉到了睡意，渐渐的睡了下去，梦里他梦到苏沐婉为自己生了两个孩子，一儿一女，他们一起生活的其乐融融的，不过不知为何中间却是突然浮现出了林雅楠和柳清月两人的影子。
<br/>　　虽然睡的晚，不过这么长时间已经养好的生物钟还是让他按时起床，跟着众人一起训练。
<br/>　　结束训练、吃过早饭后，一行人便是再次来到了训练室，开始了一上午的训练，裸装、裸符文补兵，这是最考验一个选手耐心的训练，一训练就是一两个小时。
<br/>　　主要目的就是为了让选手保持自己补兵的手感，在游戏中虽然一个人头能够为你带来300块的经济，但是如果你能够在补刀上压制对手，那么其实对方的装备就无法领先你。
<br/>　　就好比对方补了100个兵杀了你5次，但是你这边只需要比对方多个70刀，即使你一次没有击杀过对手，那么其实你们两个的经济还是持平的。
<br/>　　所以这也是为什么你明明看到自己已经两个人头了，但是和对方拼起来却是没有丝毫的优势，没有其他原因，就是你的补刀不如对手，装备无法碾压对方，所以自然还是旗鼓相当的局面。
<br/>　　下午的比赛不出所料的依旧是TM轻松取胜，这个战队毕竟是有着世界十王者选手god在场，而且对手又不是什么知名战队，所以这一点倒是并没有出乎大家的意料。
<br/>　　“好了，皇族那边应该已经开始准备了，今天咱们先和他们来两场训练赛，晚上还有雷霆等着呢。”看着比赛已经结束，叶帝连忙开口说道。
<br/>　　因为皇族的时间很紧，所以他们倒是不想在这上面上浪费时间。
<br/>　　“哎，白衣他们真的是太忙了，就不能让自己轻松一下么。”一旁的香蕉有些感叹的说着。
<br/>　　说完之后，他便是发现叶帝他们用一种看傻子的眼神看着他。
<br/>　　“你四不四傻，他们为什么这么忙你不知道么！”萝卜一脸的无语。
<br/>　　“我就是开个玩笑，活跃一下气氛嘛，不要当真、不要当真。”香蕉尴尬的应着。
<br/>　　很快他们便是来到了训练室，登录游戏。
<br/>　　这边刚一进入游戏，叶帝几人便是收到了楚白衣的邀请。
<br/>　　“你们几个挺快的啊。”看着对面整齐的id，叶帝在下方笑着打字说道。
<br/>　　“累死了，这次你们三支队伍要是拿不下好成绩都对不起我们这么长时间的努力。”皇族比赛室，楚白衣一脸生无可恋的打着字。
<br/>　　一直和这些顶尖战队打，而且还要全神贯注的打，其实他们是非常累的，只不过为了几支战队能够取得更好的成绩，他们却是一直在那里坚持着。
<br/>　　“好累啊，结束比赛后一定要请我们大吃一顿啊。”白鸽跟着打字诉苦。
<br/>　　“放心吧，比赛结束了，你们就算是不想吃，那也不行，你们可是我们夺冠的功臣！”香蕉依旧是自信满满的在那里打字说着。
<br/>　　“没错，你们的冠军有我们一份，可惜了我们不能去台上领奖。”司马贼有些艳羡又有些惋惜的开口说着，相比之下他还是更希望在赛场上做一个见证者，即使最后拿不下冠军。
<br/>　　众人闲聊了一会后，便是开始了今天的训练。
<br/>　　进入游戏后，双方飞快的ban人，如同在赛场上一样，皇族这边教练更是在楚白衣几人的身后指挥着ban选。
<br/>　　这一刻大家众志成城、全力以赴，目标只有冠军。
