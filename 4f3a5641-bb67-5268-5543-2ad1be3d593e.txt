　　云羽天宫的宝物。
<br/>　　云羽天宫的强大，比起如今的七大宗门相加，都要可怕的多。
<br/>　　这其中的种种宝物，绝无任何人，不想得到。
<br/>　　“我云羽天宫的宝藏众多，但是却从来不会给废物，只有真正的天才，才有资格得到我云羽天宫的宝藏。”
<br/>　　老者冷哼一声道。
<br/>　　“在我云羽天宫之中，已经是留有十万枚羽化晶，你们谁可以得到最多的羽化晶，谁就可以得到这云羽天宫的传承，倒时候整个云羽天宫，都会属于他！”
<br/>　　羽化晶。
<br/>　　楚风眠也曾经听说过，这羽化晶，便是云羽天宫特产的一种奇物，足以锻造天级灵器。
<br/>　　只有云羽天宫，才能产出。
<br/>　　不过现在这羽化晶的价值，可不止于此了，谁可以得到最多的羽化晶，谁就可以得到这云羽天宫的传承。
<br/>　　一旦是得到了这云羽天宫的传承，整个云羽天宫，便是属于他。
<br/>　　与此相比，什么宝物，诸天奇物，都算不得什么。
<br/>　　这十万羽化晶，必定会在云羽天宫之中，掀起一场血战，但是从未有人惧怕。
<br/>　　在场众人，谁不是九域天才，谁来到这云羽天宫，不想要得到其中的宝藏。
<br/>　　眼下这云羽天宫最大的宝藏，整个云羽天宫的传承，就在众人面前，现在绝无可能会有人选择放弃。
<br/>　　每一名武者的眼神中都是露出了几分贪婪，占有欲。
<br/>　　云羽天宫的传承，人人都想要得到。
<br/>　　人群中，楚风眠的目光中，也是露出了几分冰冷，这云羽天宫的传承，他是势在必得，绝不可能会让给其他人。
<br/>　　“这云羽天宫的传承，谁敢跟我抢，谁就得死！”
<br/>　　楚风眠心中暗自开口道。
<br/>　　同时在场也是无数道冰冷视线，扫过众人，荒圣君，药海等人，都对着云羽天宫的传承，是势在必得。
<br/>　　在众人一个个议论之际，那老者再度开口道。
<br/>　　“诸位到来，老朽已经是给诸位留下了见面礼，这彩虹阶梯上，一共有着三件宝物，谁都可以得到。”
<br/>　　老者一边说着，一边大手一挥，这彩虹阶梯上的七彩霞光，突然破碎开来。
<br/>　　通往云羽天宫的道路，如今一下子便是变的畅通无比。
<br/>　　一开始众人只能够看见这彩虹阶梯的前半，所以只发现了，在其中十分之一的地方，放置的是黑耀十方石。
<br/>　　而在这彩虹阶梯的中间，跟末尾的位置，又是分别放置了两件宝物。
<br/>　　“天界果！青雀血！”
<br/>　　楚风眠看到这另外两件宝物的一刻，眼神也不由有些惊讶。
<br/>　　云羽天宫，不愧是曾经位列九域巅峰的宗门，真是富有。
<br/>　　那在彩虹阶梯中间部分放置的宝物，便是一枚无色果子，有着拳头大小，看似平凡，实际上却是天地至宝。
<br/>　　这天界果，其实并非天界之物，而是众人觉得，其中的药力，绝非是九域灵药，所以才换名天界之名。
<br/>　　一旦服下，这天界果足以是让武者的身躯，焕然一新，打通灵脉，至少足以打通三十道灵脉。
<br/>　　曾经楚风眠所服用过的九转夺天丹，其中一味重要灵药，正是这天界果。
<br/>　　有着夺天之力的果子。
<br/>　　就算是一只猪，吃下这天界果，都是可以成为妖兽中的至尊。
<br/>　　对无数武者而来，这天界果，有着至高无上的吸引力。
<br/>　　而最后那青雀血，却更是惊人，青雀，是远古异兽之一，青雀之名在九域，鲜有人知。
<br/>　　但是这青雀之祖，青鸾，却是名震九域的恐怖存在。
<br/>　　青鸾，至高无上的天地神兽，诸天的宠儿，在历史之中，任何一个时代的青鸾，都是位于九域之巅的强者。
<br/>　　曾经万年之前楚风眠的好友青鸾，便是在当时，足以跟任何九域强者，一争高下的人物。
<br/>　　当初青鸾，若不是跟星玄剑的上一任主人，那戮血剑魔交战，只怕也不会陨落。
<br/>　　这青雀，便是青鸾的血脉之后，有着青鸾之血，在天地无数血脉中，都足以拍的上极品。
<br/>　　青雀之血，更是有着一种最为恐怖的特性，便是返祖。
<br/>　　每万年，数万年之间，在青雀族群中，便是会诞生一位返祖青雀，化身青鸾。
<br/>　　若是那个武者，得到这青雀之血，便是很有可能也返祖成功，倒时候在拥有的，可就不是青雀血脉了，而是青鸾血脉，天地之间最为恐怖的血脉之一。
<br/>　　这青雀之血，比起天界果，更是要恐怖的多。
<br/>　　尤其是现在这彩虹阶梯上，一共三件宝物。
<br/>　　青雀血，天界果，黑耀十方石。
<br/>　　若是让一尊武者，得到这三件宝物。
<br/>　　以天界果，开辟资质。
<br/>　　以青雀血，化为血脉。
<br/>　　再以黑耀十方石，精粹血脉，很有可能便是一具令青雀血脉返祖，倒时候诞生出一位资质无双的超级天才来。
<br/>　　就算是现在楚风眠身居远古战龙血脉，在加上他的身躯，也是已经经历过了无数宝物的淬炼。
<br/>　　可若是真的有人，可以得到这三件宝物的话，只怕是可能比起现在的楚风眠，还要恐怖的多。
<br/>　　这云羽天宫，竟然是留下了这三件宝物，留在了七彩阶梯上。
<br/>　　实际上便是代表着，谁能够得到这三件宝物，便是谁最有可能，得到这云羽天宫的传承。
<br/>　　“黑耀十方石，青雀血，天界果。”
<br/>　　不止是楚风眠，在场的不少强者，都是注意到了这一点。
<br/>　　那荒圣君的眼神中，都是炙热无比，眼神的目光，已经是在心中，将这三件宝物据为己有。
<br/>　　这三件宝物，让一个废物得到，都可以化身成为九域之中最为妖孽的天才。
<br/>　　更何苦，若是被荒圣君得到的话。
<br/>　　这一个时代，便是以他为尊，无人可挡。
<br/>　　“看来诸位，也都想要得到这三件宝物。”
<br/>　　老者看着众人的眼神，笑着开口道。
<br/>　　“去拼吧，去抢吧，谁能够拼到最后，谁就能够得到这云羽天宫的宝藏。”
<br/>　　“去吧！”
<br/>　　老者说完，便是身躯一动，直接落入到了那彩虹阶梯之中，一跃而上，进入到了云羽天宫之中去。
<br/>　　老者的离开，令在场突然寂静了下来。
<br/>　　几乎是在下一瞬间，无数灵力，猛然暴起。
<br/>　　无数道身影，直接冲这彩虹阶梯，猛然冲了过去。
<br/>　　“嗖！嗖！嗖！”
<br/>　　这些武者，如同蝗虫一般，冲了过去，一个个都如同赶着去投胎一样的急切。
