        明司寒随后在门框上按了指纹，听到“滴”地一声，大门解锁后，又压下门上的手柄，将门打开站到一旁向连星儿迎了迎手。
<br/>
<br/>        连星儿好奇地走进去后，四下看了看。
<br/>
<br/>        这里是个单身公寓，面积不大，但是个复式套房。
<br/>
<br/>        明司寒关上门，脱了鞋，直接走到那边开放式厨房的冰箱前，打开冰箱，从里面拿出了两瓶果汁饮料。
<br/>
<br/>        “走，跟我到屋顶去。”明司寒随之看向连星儿，温和地说道。
<br/>
<br/>        连星儿讷讷地点了点头，跟着明司寒一起走上了旋转楼梯。
<br/>
<br/>        到了楼上，她才发现，这里是个空旷的玻璃房。
<br/>
<br/>        不仅屋顶是玻璃，就连四周的墙壁也全都是玻璃。
<br/>
<br/>        她一抬头就能看到满夜繁星，向周遭看去时，能将城市的灯火阑珊尽收眼底。
<br/>
<br/>        站在这里，就像是站在云端一样。
<br/>
<br/>        连星儿顿时心情大好，当她看到东面的玻璃墙前还有一副天文望远镜时，立即高兴不已地跑了过去，握起镜筒对着夜空看了起来。
<br/>
<br/>        “那个长得像飞碟的是木星吗？”连星儿睁一只眼闭一只眼，一边看着，一边好奇地问道。
<br/>
<br/>        明司寒站在一旁，看着她这可爱的样子，忍俊不禁地回答道：“那是土星。”
<br/>
<br/>        “那哪个才是木星？”
<br/>
<br/>        “我来帮你找。”明司寒说着走到了连星儿的身后，伸出双臂将她圈在了天文望远镜和自己胸膛的中间，然后替她调整天文望眼镜的位置。
<br/>
<br/>        连星儿接下来把月球、木星、火星、金星看了个遍后，又嚷着要看其它星系。
<br/>
<br/>        好在他明司寒这上二十多万的天文望远镜不是白买的，只要是她连星儿想看的星星，他都能替她调好焦距和位置，让她看个够。
<br/>
<br/>        这天文望远镜玩够了，连星儿开始哈欠连连。
<br/>
<br/>        就在明司寒替她调整下一个对焦位置时，她那小脑袋瓜子一歪，直接倒在了他的肩头上睡了过去。
<br/>
<br/>        明司寒会心一笑，抬起手臂给她当枕头，索性搂着她，慢慢地躺在了地毯上。
<br/>
<br/>        两人的身子刚放平，连星儿便本能地侧过身，往他怀中钻了钻，找了一个舒适的位置，嘴中“哼唧”了一下便“呼呼”地熟睡了过去。
<br/>
<br/>        此刻，明司寒的脸上挂着浅浅的微笑。
<br/>
<br/>        眼前是一片美丽的星海，怀里是一个可爱的小星儿。
<br/>
<br/>        翌日。
<br/>
<br/>        连星儿一觉睡到自然醒，睁开眼睛的第一眼，看到的是一片蔚蓝色的天空，还有在空中缓慢缥缈的几缕白云。
<br/>
<br/>        这种感觉，恍若是在空中像鸟儿那般自由地翱翔。
<br/>
<br/>        当她偏过头去时，才发现自己躺在了明司寒的臂弯里。
<br/>
<br/>        明司寒此刻还未睡醒，胸膛平缓地起伏着，鼻息里发出沉匀的呼息声。
<br/>
<br/>        他真的是个长得很好看的男生，侧颜轮廓棱角分明，剑眉飞扬、鼻梁高挺、唇瓣细薄，浑身还透着一股生机勃勃的青春朝气。
<br/>
<br/>        连星儿不知不觉中看出了神。
<br/>
<br/>        直到明司寒的眼皮子在动，她才立马闭上眼睛，假装自己仍旧在睡。
<br/>
<br/>        明司寒睁开惺忪的睡眼后，见这丫头还未醒，嘴角微微上扬，抬起手来轻轻地拨了拨她额前的刘海。
<br/>
<br/>        他的指腹撩过她的发丝后，又爱怜地触摸着她的脸颊。
<br/>
<br/>        又滑又嫩，像煮熟的圆鸡蛋，让他恨不得想要凑上去咬一口就好。
<br/>
<br/>        明司寒也只是心里这么想想，苦笑自己不敢对她实际行动。
<br/>
<br/>        “喂，丫头，别装了，我知道你已经醒了。”
<br/>
<br/>        他刚刚抚摸她的脸颊时，她的身子本能地颤了一下。
<br/>
<br/>        连星儿装不下去了，索性直接坐起身来。
<br/>
<br/>        明司寒也支起身子，捂着肩头，活动着两手臂的各个关节。
<br/>
<br/>        “被你压了一晚上，手都麻了！”
<br/>
<br/>        “这不能怪我！我可没要你的手臂给我当枕头。”连星儿努嘴道。
<br/>
<br/>        明司寒情不自禁地笑道：“对对对，不能怪你。”
<br/>
<br/>        “这还差不多！”连星儿顿时眉开眼笑。
<br/>
<br/>        明司寒随后站起身来，伸了个懒腰，接着说道：“我出去给你买早餐和要换洗的衣服回来，你可以先去洗个澡。衣柜里有我的衣服，你随便挑一件先穿上。”
<br/>
<br/>        “噢，好。”连星儿点了点头。
<br/>
<br/>        明司寒微微一笑，抬着手一边垂着后颈，一边转身往楼下走去。
<br/>
<br/>        连星儿听到关门声后，才起身下楼。
<br/>
<br/>        她走到卧室，打开了明司寒的衣柜，从里面随便挑了一件衬衫出来，便去了浴室。
<br/>
<br/>        这种闷热的天气，她还一天一夜未洗澡，真不知道自己昨晚是怎么睡着的。
<br/>
<br/>        不过，连星儿在拧开花洒淋浴时，才猛然想起，昨晚是她这段时间里睡得最舒心的一晚。
