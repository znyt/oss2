　　第450章
<br/>　　黄昏，日薄西山。
<br/>　　将要落山的夕阳给草木未生的草原洒下一片红色。若是看得仔细，便会发现，这红色不仅仅是草木的余晖，确切来说，是鲜血的颜色。
<br/>　　程大雷双手握着大斧，将一人的脑袋重重劈下，鲜血溅了他一脸。
<br/>　　嘟，收到恐惧值999……
<br/>　　以斧撑地，程大雷目光看过去，脚边是一具具尸体，身周围再无一个立着的人。刚才被程大雷所杀的，是最后一人。
<br/>　　当然，对方不是最后一个敌人，也不是程大雷要杀的最后一人。
<br/>　　从遭遇捉鬼道人到现在，程大雷自己也不知道碰到了多少波敌人，更不知自己的斧下砍了多少脑袋。
<br/>　　在杀虎口附近的草原上，他们一波波冒出来，然后再一波波被程大雷铲除干净，死的不计其数，倒是令程大雷狠狠收割了几波恐惧值。
<br/>　　他走到河边，避开徐神机和银眸，从系统商店里购买了一套皮甲，又简单把身上的血迹洗了洗。
<br/>　　等他回来的时候，徐神机已经从死人身上搜出干粮，看到程大雷回来，他眼神愣了愣。
<br/>　　“大当家，你怎么每次杀完人都有新衣服穿呢？”
<br/>　　“仙人送的，怎么你有问题么？”程大雷已经懒得解释。
<br/>　　“喔，没有，没有。就是能不能下次让仙人顺便捎点吃的。”
<br/>　　“仙人可能没空。”程大雷没好气道。
<br/>　　一路过来，吃得是不愁的，就算不能从死人找不到干粮，但程大雷也可以宰敌人的马吃，老实说，味道还不错。
<br/>　　关键的麻烦是，敌人太多了。就像在房间里发现第一只蟑螂后，意味着你已经该全屋大扫除了。
<br/>　　每一次战斗程大雷都必须竭尽全力，虽然暂时还没有出事，但难保以后不会出事。这严重拖慢了程大雷赶往杀虎口的速度。
<br/>　　现在还不知道杀虎口是怎样的情形，林少羽是活着，还是已经被杀死了，程大雷都无法确定。
<br/>　　“咱们距离杀虎口还有多远？”程大雷问。
<br/>　　“要是顺利的话，今夜就能赶到。”徐神机向北边望了一眼，已经能看到起伏的山脉。
<br/>　　“走啦，走啦。”程大雷上了黑牛：“咱们快些赶路，等把少羽捞出来，就回蛤蟆城，以后有天大的事，我也不出来了。”
<br/>　　徐神机吊着一张苦瓜脸，表情十分痛苦，但最后还是骑上了马。
<br/>　　“大当家……”
<br/>　　“怎么了？”正准备出发的程大雷往。
<br/>　　“我想我们还是等下再走吧，又有客人来了。”
<br/>　　程大雷向远处望去，只见一支骑兵队伍停在不远处，身穿统一的制服，就连表情都差不多相似。
<br/>　　“那就好好招待招待他们吧。”程大雷无可奈何的叹口气：“这波客人是什么来路啊？”
<br/>　　“看着像京州兵，应该是……”徐神机的舌头有些打结：“将军府的老兵组。”
<br/>　　程大雷眉头也皱了起来，收起了轻视的心理。
<br/>　　能打过一次仗，还活下来的人，都是军中的宝贝。见过杀人，流过血，握兵器的时候就不会颤抖，冲锋的时候就不会打摆子，知道自己该如何活下去，如何躲过敌人的冲锋。
<br/>　　尉迟离手中就有这样一支老兵组成的队伍，他们不是打过一场战斗，而是从无数次血战中活了下来。对尉迟离忠心耿耿，就算前方是悬崖，尉迟离让他们冲锋，也不会有任何一个人犹豫一步。
<br/>　　程大雷在蛤蟆城就听过他们的名声，也是他这次最忌惮的敌人。却是没想到，他们在最关键的时候出现了。
<br/>　　“大当家，这次的客人有些扎手啊，咱们该怎么办？”徐神机忧心忡忡道。
<br/>　　“怎么办，凉拌。我们还拖得起么？”程大雷道：“银眸，保护好军师。”
<br/>　　老兵组在有条不紊的推进，百人组的队伍，让程大雷有种面对千军万马的感觉。这就是士气的作用，一支队伍没有士气，不过是一盘散沙，但拥有士气后，沙子却能凝成最坚固的石头。
<br/>　　银眸握紧两根短矛，早已准备好放手一战。徐神机早就有准备，等着银眸将他提上马背。
<br/>　　在程大雷准备好后，老兵组恰恰也完成推进，在距离程大雷一百步时，发动了冲锋。
<br/>　　于此同时，程大雷也展开了冲锋。
<br/>　　一个人对一支队伍的冲锋。
<br/>　　以前，程大雷当然不会如此冒险，但在迈入绝世境界后，程大雷已经很少遇到敌手。秦蛮在顶级的时候，就能对普通人形成百人敌。当然，老兵组的组成都是悍卒，他们当然不是普通人。不过，程大雷觉得以自己绝世的等级，打他们一百个也不成问题。
<br/>　　何况还有银眸这杀星在。
<br/>　　双方撞在一起，墨玉乌月兽狠狠的撞翻了几人，程大雷的大斧已经兜头砍下，一颗血淋淋的脑袋滚落在地上。
<br/>　　程大雷在队伍中横冲直撞，左杀右杀，他领教了老兵组的坚韧，他们当真是悍不畏死，顶着程大雷的斧头攻击，没有一人后退。
<br/>　　程大雷还是很少在帝国军人身上看到这种勇气，上次见到这一幕，还是在戎族身上。
<br/>　　帝国并不缺乏勇士，只不过可惜的是，他们今天是自己的敌人。
<br/>　　这种同归于尽的打法，当真给程大雷带来了巨大的压力。他一边冲杀，忽然发现敌人中有个用大锤的，似乎是老兵组的头目。
<br/>　　程大雷眼前一亮，骑着黑牛冲了过去。这黑牛极其神异，比一般战马力气要大上许多，牛角直接顶穿了一匹马的肚子，程大雷冲到那大锤将面前。
<br/>　　对方留着一个秃头，秃子一般都不好对付，见程大雷过来，直接握着大锤砸了过去。砸的不是程大雷，是黑牛的脑袋。
<br/>　　程大雷一斧砍下他的肩膀，把斧头夺了过来，劈手将大锤夺了过来。入手一沉，至少二百余斤。
<br/>　　程大雷将铁锤交给了身后的银眸。银眸天赋异禀，力大无穷，她丢掉了手中的长矛，双手握住铁锤，整个抡开。
<br/>　　当真是挨着死，擦着伤，一旦躲得慢的，直接被她砸成肉酱。程大雷也赶紧躲远点，避免被她误伤。
<br/>(本章完)
