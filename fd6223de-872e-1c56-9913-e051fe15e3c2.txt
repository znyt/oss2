        落以柔惊愕又心虚地从沙发上弹跳起来。
<br/>
<br/>        “大、大哥！噢！不，是傅总！”落以柔支支吾吾地唤道。
<br/>
<br/>        傅夜沉此时正从中央的卧梯上从容地走下来，他衣冠楚楚，一双冷锐的黑眸，目光的焦点完全没有落在她落以柔身上。
<br/>
<br/>        落以柔却傻愣地仰视着傅夜沉，他的五官像是精雕细琢后的艺术品，高挺的鼻梁、薄而性感的双唇，一双深邃的黑眸仿佛总能穿透她的心。
<br/>
<br/>        待她回过神来时，他已经往大门那边去了。
<br/>
<br/>        落以柔连忙提起自己落在沙发上的手提包，跟在了傅夜沉的身后。
<br/>
<br/>        上车前，落以柔发现傅夜沉开的车，居然是一辆没有牌照的新越野车，心里顿时燃起一种不祥的预感。
<br/>
<br/>        私宅里又不是没房间给她住，为什么他急着把她送走？
<br/>
<br/>        难道是因为怕童筱颖看到她，心里不舒服吗？
<br/>
<br/>        毕竟，傅夜沉是牺牲了他自己的命救了她落以柔，然后死里逃生。
<br/>
<br/>        想到这里，落以柔忽然间心情大好。
<br/>
<br/>        童筱颖在他傅夜沉那里，只不过是床上的发泄工具罢了。
<br/>
<br/>        而她，在他心里，与众不同。
<br/>
<br/>        因为，她落以柔觉得，他傅夜沉可以为了她连命都不要，那么她在他心里一定有比较特殊的地位。
<br/>
<br/>        “我救你，是我的职责所在，没有其他成分。”傅夜沉一边发动了车子，一边冷冷地说道。
<br/>
<br/>        他一句话，便浇灭了刚刚她脑海里那些所有自作多情的恋想。
<br/>
<br/>        “所以，请你好好珍惜你自己的生命，不要浪费在我这种男人身上。”他淡然自若地接着说。
<br/>
<br/>        坐在副驾驶座上的落以柔，偏头看向傅夜沉时，无意间看到了他敞开的风衣下，腰间佩戴的枪支。
<br/>
<br/>        落以柔身心一怔，顿时间，就明白了傅夜沉突然送她离开的真正原因。
<br/>
<br/>        他这一次，还能平安归来吗？
<br/>
<br/>        经历过他的假死之后，她便揣测到，他傅夜沉的身份绝对不是只有傅氏集团总裁这么简单。
<br/>
<br/>        可是，她又能为他做什么呢？
<br/>
<br/>        落以柔心疼他傅夜沉，鼻子一酸，眼泪溢满眼眶，想哭，却只能强颜欢笑。
<br/>
<br/>        她的声音忽然温柔而细沉，十分体贴地说道：“大哥，不管你要去做任何事情，或许即将要做的任何决定，我都会支持你。你放心，我会乖乖听你的话，以后，不会再轻生了。我还会和师父一起，替你守住傅氏集团。我……”爱你。
<br/>
<br/>        “我和郑忠威说了，介绍你去美国的一家大私企里工作。那里的薪水和福利，不会比你待在傅氏集团差。”傅夜沉直接打断了落以柔的话，脑海里却在回忆着刚刚与筱颖的那件事情。
<br/>
<br/>        事后，他满是心疼地抱她回床上躺着，她却在他走之前，拽住他的衣角，对他说：“大哥如果不放了雅珊，我便让这个孩子死在肚子里。”
<br/>
<br/>        “童筱颖，你够了！”他一个旋身，快速出手，直接捂住了她的嘴。
<br/>
<br/>        起初，他就在怀疑她突然跟他要孩子的意图，只是百思不得其解的真相，竟然是她拿孩子来逼他。
<br/>
<br/>        他怎么会爱上这么一个对自己如此残忍的女人？
<br/>
<br/>        “童筱颖，你给我听好了！季雅珊的事，我不会管！你若是敢杀了我们的孩子，我便去给我们的孩子陪葬。我们俩倒是比比看，看谁比谁更残忍！”
<br/>
<br/>        那一刻后，他放开了她，冷笑着离开了房间。
<br/>
<br/>        此刻，童筱颖抱着双膝，傻傻地坐在床头。
<br/>
<br/>        身上残留着他傅夜沉的味道，包括这个房间里，好像都弥漫着他傅夜沉的气息。
<br/>
<br/>        大哥的那句狠话，彻底断了她想要救季雅珊的希望，同时也葬送了她对傅夜沉所有的依赖。
<br/>
<br/>        她不再指望这个男人会给她一个完整的家，也不再指望这个男人会给她带来什么温馨的生活。
<br/>
<br/>        另一边，傅夜沉开车载着落以柔离开私宅后，去了她的公寓楼下。
<br/>
<br/>        他看着腕表上的时间，意味不明地皱了下眉头。
<br/>
<br/>        再去救季雅珊之前，他还得去见十爷。
<br/>
<br/>        落以柔现在已经没再住在那破陋的出租屋里了，而是住进了傅氏集团旗下的员工公寓大厦。
<br/>
<br/>        下车前，落以柔自己解开安全带后，趁着傅夜沉不备，再次扑过身去抱住了傅夜沉。
<br/>
<br/>        傅夜沉未料到落以柔会突然来这招，而落以柔却在他用力反抗之前，先一步十指紧握，将他的肩头牢牢地圈在自己了的双臂间和胸怀前。
<br/>
<br/>        落以柔故意紧紧地靠拢着傅夜沉手臂，想要他感受到来自于她的体温。
