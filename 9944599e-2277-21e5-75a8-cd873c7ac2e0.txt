　　不过楚风眠心中可不敢放松警惕，这山谷中，到底有着什么危险，楚风眠还不清楚。
<br/>　　这山谷，若是云羽天宫的考验，那不可能会是如此简单。
<br/>　　“轰隆！”
<br/>　　突然之间，一声雷鸣声，突然响起。
<br/>　　顷刻之间，就在这山谷周围，一道雷霆，突然从天而降，直接向着楚风眠劈了过来。
<br/>　　这雷霆的突然出现，出乎任何人的意料之外。
<br/>　　尤其是这里，刚刚离开了狂风的袭击，正好是放松的一刻，这雷霆突然降下，绝对是有着极大的危险。
<br/>　　“砰！”
<br/>　　这一道雷霆，劈在楚风眠的身上，顿时楚风眠便是感觉到了一阵麻痹。
<br/>　　也幸好楚风眠没有放松警惕，没有将远古龙皇身散去，不然的话，这一道雷霆很有可能都能让楚风眠重伤。
<br/>　　“好可怕的雷霆，这雷霆的力量，甚至是不亚于许多生死大劫的雷霆！”
<br/>　　楚风眠心中微微有些吃惊。
<br/>　　这云羽天宫的考验，果然不是这样轻易就可以度过的。
<br/>　　这狂风，仅仅只是其中的第一道考验。
<br/>　　如今狂风刚过，雷霆便是袭来，这雷霆的威力，比起狂风的威力还要大得多。
<br/>　　这一道道，仅仅是开始。
<br/>　　随着楚风眠更加深入到这山谷之中去，这雷霆的数量，也变得越来阅多。
<br/>　　尤其是如今在这山谷之中，还无法催动遁光，想要躲避着雷霆，都没有可能，只能够是以身躯来抵挡。
<br/>　　楚风眠冲在最前面，这些无数雷霆，也是首当其冲，率先向着楚风眠轰击。
<br/>　　而跟在楚风眠身后的剑苍穹等人，受到的雷霆却是要少得多。
<br/>　　“哼，这小子冲到最前面，真是自寻死路，这些雷霆，他应付起来绝对是吃力无比，看他还有多少灵力可以应对，一会就是他的死期了！”
<br/>　　剑苍穹看着楚风眠，冷笑连连。
<br/>　　他一直以来都是在注意这楚风眠的一举一动。
<br/>　　如今的楚风眠，速度已经是比起刚刚，要慢了许多，在对付雷霆的时候，也显得吃力了不少。
<br/>　　在剑苍穹看来，这就是如今楚风眠灵力消耗过多的样子。
<br/>　　不过剑苍穹也不着急，他要杀楚风眠，必须要一击必杀，他也不确定楚风眠到底还有没有底牌，谨慎行事。
<br/>　　“哦？居然还不出手？”
<br/>　　楚风眠看着身后，剑苍穹还是一直跟随，没有任何出手的打算，他的脸上都是微微有些吃惊。
<br/>　　这剑苍穹，还真的是沉得住气。
<br/>　　楚风眠现在灵力消耗过多的样子，自然是他自己装出来的，楚风眠拥有神脉，怎么可能会灵力消耗过度。
<br/>　　现在楚风眠故意装出来这样一幅样子，正是要引诱这剑苍穹出手。
<br/>　　现在看来，这剑苍穹还是极为谨慎的，不过这谨慎，终究有个头的。
<br/>　　“轰！”
<br/>　　又是一道雷霆，突然向着楚风眠袭来。
<br/>　　这一道雷霆，呈现出一种金色，其中蕴含的威力，比起之前的任何一道雷霆，都要大得多。
<br/>　　楚风眠的脸色都是猛然一变，急忙调动身上的灵力，在他的面前，化为了一道灵力屏障，前来抵挡。
<br/>　　“咔嚓！”
<br/>　　但是这一道雷霆落下的一刻，楚风眠刚刚凝聚出的屏障，甚至都无法全部抵挡，随着一声响声，这一道屏障，突然破碎。
<br/>　　无数雷霆，顿时落入到了楚风眠身上。
<br/>　　“小子，死吧！”
<br/>　　在楚风眠身后的剑苍穹，看见这一幕，脸色突然变的无比狰狞起来。
<br/>　　这对他来说，这是最好的机会。
<br/>　　“嘶！”
<br/>　　一道剑锋，突然出窍，直接便是向着楚风眠斩杀而来。
<br/>　　如今剑苍穹便是打算趁着这一道雷霆，围攻楚风眠，将楚风眠直接斩杀。
<br/>　　一剑出，便是绝对是杀招，没有丝毫任何一点的留情。
<br/>　　楚风眠的实力剑苍穹可是知道的一清二楚，如今寻找到机会，他出剑，自然是毫不保留。
<br/>　　一抹剑光，瞬息之间，便是向着楚风眠的胸口斩了过去。
<br/>　　这剑光，快到极致，根本是令人来不及任何的反应。
<br/>　　在这山谷中，在这法则如此可怕的压制之下，还足以爆发出如此剑光，这便是代表着剑苍穹的最强一剑。
<br/>　　“终于出手了！”
<br/>　　楚风眠眼光一闪，嘴角突然露出几分笑意。
<br/>　　他的身躯，猛然一动，这就在楚风眠身上的雷霆，顷刻之间，突然全部都都被楚风眠吞噬。
<br/>　　哪里还有刚刚看起来，那般勉强的样子。
<br/>　　同时在楚风眠的手心之中，祖龙至尊剑也是突然出窍，同样是一道剑光，向着剑苍穹斩杀过去。
<br/>　　楚风眠一直是在等待着剑苍穹出手，而并没有主动出手，所追求的，自然就是一招必杀。
<br/>　　想要一招斩杀中域百绝榜的榜首剑苍穹，可并非是一件容易的事情。
<br/>　　在这山谷中出手，若是一击不成的话，剑苍穹只要是退后，楚风眠就必须要分心去追逐他。
<br/>　　如此一来，就需要浪费大量的时间。
<br/>　　所以楚风眠必须要一击必杀，而这剑苍穹，最为分心的一刻，便是他要斩杀楚风眠的一刻。
<br/>　　在这一刻的时候，剑苍穹的防御，便是最弱。

<br/>　　楚风眠手中的剑动了，剑光一抖，这剑苍穹的剑光，便是被彻底粉碎。
<br/>　　这一道剑光，在将剑苍穹的剑光，粉碎之后，毫不留情，直接向着剑苍穹刺了过去。

<br/>　　“怎么可能？”
<br/>　　“剑师兄死了？”
<br/>　　“不好？逃！”
<br/>　　那剩下的三名天剑宗弟子，看到这一幕，皆是脸色骤变。

<br/>　　但是楚风眠的剑光，比起他们的身影，还是要会的多，随着一阵剑光，这三人，全部在剑光之下，化为了一团团血雾。
