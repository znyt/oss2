　　“是什么，到底是什么？”
<br/>　　后土大急，连忙顺着自己的心神追寻而去，想要抓住那一丝感悟。她隐隐感觉到，这一缕灵光的领悟，对自己非常重要。
<br/>　　“呼！”
<br/>　　平心静气，身躯心灵与法则融合。
<br/>　　土之祖巫，后土自然而然的将自己融入到天地法则之中，身躯与大地融合。
<br/>　　“嗡嗡嗡！”
<br/>　　感受着这大地的宽广、辽阔，感受大地的博大，后土陷入一场悟道之中。将身心融合，切身体会大地的伟大，孕育众生，承载万物，默默奉献，不求回报！
<br/>　　“大地浩瀚，承载一切。”后土轻声自语。
<br/>　　“大地……”
<br/>　　后土的眼神有些迷茫了，她的心灵完全与天空大地一切都融合在一起。
<br/>　　天地间万物力量涌动。
<br/>　　那充满灵动的风，那汹涌炙热的火，那柔和莫测的水……
<br/>　　而大地，却是有无尽厚重的大地力量构成，它无边无际，承载着万物。
<br/>　　这一刻，她成了洪荒大地，感受着它的伟大，感受着它的情绪。
<br/>　　它，浩瀚广阔。
<br/>　　它，就仿佛父母的胸膛。
<br/>　　它，又仿佛整个世界的根基，草木生长，花香满园，鸟儿腾飞，溪水潺潺……
<br/>　　这一切都是脚下这片大地的点缀，大地因为有了万物而充满生机，而万物则是因为有了大地才有了承载，而不再是空中楼阁。
<br/>　　后土闭上双眼，将身躯完全交于自己的心灵支配，身体随着心走。她漫步在这宽阔的大地之上，一步一步，完全不知道下一步要到哪里。
<br/>　　点点土黄色的光芒，开始在她的周边浮现环绕。
<br/>　　这无数的光芒，就仿佛在母亲肚子里那种温暖的感觉，它包容一切。
<br/>　　大地苍莽，以厚德载物！
<br/>　　“嗡！”
<br/>　　突然，后土身上那股厚重的气息越来越浓郁了，不知过来多久，她全身土黄色光芒大涨，璀璨无比，在后土的体内，那开辟出来的法则空间内，一股股的法则之水如同河水一般汇入到法则之海中，比之前一滴一滴的状态快了不知多少倍。
<br/>　　不知过了多久，后土全身气势一涨，她身上的光芒更加璀璨了，法则空间中，原先如同一汪湖泊的法则之海如今早已变得如同真正大海般的法则之海，道韵流转，法则彭拜，暗涛汹涌。
<br/>　　当一切都停了下来后，她全身的光芒慢慢敛去，体内空间之中，法则之海喊喊恢复平静。她缓缓睁开眼睛。
<br/>　　那是一双怎样的眼睛，眼中充满温和，充满广阔，充满包容万物的气息，犹如大地一般广阔。
<br/>　　这一刻，天地都要为她失色，日月变得无光，星辰变得暗淡。
<br/>　　这一刻，她美不胜收。
<br/>　　可惜，没人能够有缘看到这一幕。
<br/>　　这一刻，她的祖巫之体变得圆满了，堪比准圣圆满，只需凝聚法则之晶，她便可以突破到巫神境界，堪比洪荒圣人。
<br/>　　“嗤嗤嗤！”
<br/>　　后土全身散发着一种道的韵律，在这种状态下，她感到了大地之上，无数生灵陨落后的魂魄在天地间游荡，无依无靠，化作滔天怨气，覆盖洪荒各处。
<br/>　　这一刻，她心中完全没有突破境界后的喜悦，反而是满脸泪水，心中充满悲伤。
<br/>　　“洪荒死难无数生灵，死后魂魄分散到天地之间，没有安身之所，不能超脱，积累的怨气已经不知有多少，如此下去，迟早有一天会有大灾难降临洪荒。”
<br/>　　想到这里，后土顿时眉头紧皱，面上不由得露出急切之意，担忧的叫道。
<br/>　　“嗡！”
<br/>　　顿时，一股大德慈悲、悲天悯人之意，从后土心头升起。
<br/>　　“不行，我一定要想办法，改变这一切！”
<br/>　　后土突然流泪，泪流满面，大声叫道，宛若发下誓言。
<br/>　　“你果然还是来了！”
<br/>　　就在此时，一道幽幽的感叹之音，传入后土耳中，让心中正悲切不已的后土为之一滞，她抬起头来，看见前方不远处一座高崖之上，一道身穿暗红色道袍的身影正站在那里，他那一头黑发如瀑布般垂了下来，直达腰间。他此刻背对自己，就仿佛背对众生，看透世界苍凉，给人一种孤零零的沧桑之感。
<br/>　　仔细一看，这人又给她一种既不存在过去未来现在，有仿佛三者都存在的诡异感觉。这就让人恐怖了，要知道后土如今已经堪比准圣圆满了，她竟然看不透眼前这人。
<br/>　　虽然这人只有准圣中期的实力。
<br/>　　但不知为何，她心中并没有警惕，眼前这人能够给她一种十分亲切的气息，好似他们早就认识一般，能够让她从心底对他没有防备。
<br/>　　“你是谁？我们认识吗？”
<br/>　　后土看着眼前的男子，轻声问道。
<br/>　　“吾名三生，我们虽然过去不认识，但自此刻开始，我们就认识了，不是吗？”
<br/>　　男子声音柔和，如岁月般无声无息，响在后土的心底，让人不由自主的选择相信。
<br/>　　后土突然沉默了，她是在天机的牵引之下，来到此处的，但眼前这人仿佛早就知道她会来此处一样，她真不知道自己该不该相信眼前这人所说的话。
<br/>　　突然，她轻轻一笑，道：
<br/>　　“不错，自此以后，我们便认识了。我叫后土，不知为何，三生道友总给我一种亲切的感觉，就仿佛……仿佛遇到了自己的家人一般。这是为何？”
<br/>　　男子缓缓转过身来，看向后土。同时，后土也看见了男子的模样。
<br/>　　男子脸色白润，模样俊俏，一双眼眸深邃无比，就仿佛星空一般璀璨，眼中更是透露着苍茫之感，给人一种看透世间生死的感觉。两条眉毛挺翘而起，有一种不可冒犯的凌厉之感。
<br/>　　最让人印象深刻的是，在他的眉心处，有一道三色花瓣似的神纹，红色、黑色与白色，给人一种十分妖异的感觉，同时也使得他的气质更加出尘了。
<br/>　　后土看着眼前的男子，突然间有些说不出话来。男子也同样看着后土，盯了她良久，突然，他轻声笑道：
<br/>　　“巧了，我也一样。说不得我们前世是一家人呢？”
<br/>　　“前世？”
<br/>　　“是啊，前世。”三生道人眼中颇有意味的叹了一句。
<br/>　　后土若有所感，心中仿佛有一道闪电划过，照亮了她的心途，她隐隐有些抓住了未来的路。但当她想要再前进一步之时，却又有些做不到。
