　　炼器殿。
<br/>　　下一座宫殿是炼器殿，是天元商会炼器之地，如今摆放的大部分材料，许多也都是被这剧毒所侵蚀。
<br/>　　剩下的，寥寥无几，不过还是有些价值不菲的珍贵材料。
<br/>　　最终也是被楚风眠全部收取到了空戒之中。
<br/>　　一座座宫殿，都被楚风眠扫荡过来，最终外围的十七座宫殿，都被楚风眠扫荡一空。
<br/>　　不过得到的东西，倒是寥寥无几，最大的收获还是那九龙神苍鼎跟一百多枚古圣丹。
<br/>　　其他的宫殿之中，许多都是天元商会弟子修炼之用的宫殿，都没有多少的收获。
<br/>　　“看来这天元商会的真正宝库，应该就是在主殿之内，看来这主殿，我是必须要去一趟了！”
<br/>　　楚风眠已经是来到了距离主殿最近的一座宫殿之中。
<br/>　　这宫殿之中，一眼望去，已经是看不清眼前的东西了，到处都是紫色的毒雾了。
<br/>　　在这宫殿之内，楚风眠也是打起了十二分精神，将十二万九千六百枚血肉晶体催动到了极致，抵挡着毒雾的侵蚀。
<br/>　　同时楚风眠也是时时刻刻的观察这他的身躯，注意这千万不要被毒雾侵蚀。
<br/>　　以这里毒雾之中的剧毒，一点是侵入到了楚风眠的身躯之中，楚风眠便是真正的万劫不复了，谁也救不了楚风眠。
<br/>　　“走！”
<br/>　　楚风眠观察了一眼主殿，便是毫不犹豫，身形一闪，向着主殿的方向走了过去。
<br/>　　面对迎面而来的毒雾，楚风眠悄然之间催动起吞天秘术来，一股吞天之力，凝聚在楚风眠的身边。
<br/>　　这迎面而来的毒雾，先是被吞天秘术吞噬大半，在以不朽真身来抵挡。
<br/>　　吞天之力在楚风眠的面前，化为了一道屏障，在保护着楚风眠的身躯，经过吞天之力的吞噬，足以化解毒雾七成以上的毒性。
<br/>　　楚风眠步入到主殿之中，一眼望去，这里的剧毒已经是凝聚到了不可思议的地步。
<br/>　　地上尽是紫色的液体，这就是毒雾凝聚的结果，这里的每一滴紫色液体，都足以轻易的毒杀一名九劫古帝。
<br/>　　可以说一名九劫古帝，在步入这主殿的一刻，都会被彻底毒死。
<br/>　　“这主殿里，到底是有什么东西，散发出如此惊人的毒雾？”
<br/>　　楚风眠扫视了周围一眼，目光闪烁，催动起灵识来，探查这主殿之中的种种。
<br/>　　这主殿，是曾经天元商会会长的住所，按理来说，天元商会的宝藏，应该是最有可能出现的地方，就是在这主殿之中。
<br/>　　“这里是？”
<br/>　　楚风眠灵识探察，突然间楚风眠身形一闪，来到了主殿的深处，他观察了一眼这周围的情况。
<br/>　　突然之间楚风眠手指凝聚灵力，向着面前猛然一点。
<br/>　　就在楚风眠的面前，突然间的一道禁制，被这一指的力量顿时崩灭，在其中，一道古老的铜门显现出来。
<br/>　　“果然这宝库，就在这主殿之中！”
<br/>　　看到这现身的古老铜门，楚风眠脸上露出了几分喜色，他手一推，这古老铜门当即打开。
<br/>　　一眼望去，琳琅满目，灵器，圣丹，材料，诸天奇物，几乎是数不胜数，堆积成山。
<br/>　　单单是天级灵器，楚风眠一眼望去过，都足足有着数百件之多，而且无一例外，几乎都是天级灵器之中的高级货色。
<br/>　　“好，这里应该就是天元商会的宝库！”
<br/>　　楚风眠看了一眼，彻底确定了，这里就是天元商会的宝库，楚风眠再度扫了一眼，单单是各色圣丹，在其中足有上千枚。
<br/>　　其中用来提升实力，突破境界之用的圣丹，足有一百多枚，得到这一百多枚圣丹，楚风眠绝对有着九成的把握，可以突破帝尊境。
<br/>　　这些圣丹，都足以造就出上百名古帝了。
<br/>　　就算是楚风眠的突破在难，花费这么多的资源，也足有从容突破了。
<br/>　　而且令楚风眠最为欣喜的是，这里面的所有宝物，都没有收到剧毒的侵蚀，看来是那古老铜门，抵挡住了剧毒的侵蚀。
<br/>　　轰隆！
<br/>　　就在楚风眠步入到这宝库之中，准备收取的一刻。
<br/>　　突然之间一道拳风，从楚风眠的背后突然间的轰击了过来。
<br/>　　“有人？”
<br/>　　楚风眠来到这主殿之中，便是时刻的观察着周围的一切，可是这突然的轰击，还是将楚风眠吓了一跳。
<br/>　　以这主殿之中的剧毒，纵然是半步天人境的强者，都未必是敢踏入到其中，现在居然是有人？
<br/>　　楚风眠也是经历过了无数次生死之战的人物。
<br/>　　电光火石之间楚风眠便是反应了过来。
<br/>　　他急忙退后连连，催动起神树跟不朽真身的力量，化为了一道屏障，同时楚风眠也是毫不逊色，一掌打出，直接打出了始祖第一式的力量。
<br/>　　跟这一拳凌空交手。
<br/>　　碰！
<br/>　　一拳一掌交锋，突然间始祖第一式的力量，被一股巨力当即打破，旋即楚风眠的身躯都是被打飞了出去，一下子撞入到了宝库之中。
<br/>　　“好恐怖的力量，这是什么境界的强者？难道是九劫古帝？甚至是半步天人境？”
<br/>　　楚风眠眼神之中闪过几分惊慌。
<br/>　　以他现在的实力，催动始祖第一式，单单是力量已经是无限接近于九劫古帝的全力一击了。
<br/>　　可现在居然是被这样打飞了出去，这出手的人，力量比起楚风眠至少是要强大十倍。
<br/>　　楚风眠身形连连后退，同时楚风眠目光看了过去，就在他的双眼之中，映入了一个人影。
<br/>　　只是这个人，跟楚风眠想象的却是不一样，衣衫偻烂，低着头，这个人的头发，跟他的肤色，都是呈现出了一种病态的紫色。
<br/>　　与此同时，楚风眠观察到，在这个人出之后，这周围的剧毒，都变的浓郁了无数倍。
<br/>　　“这个人，就是这剧毒的源头？”
<br/>　　无数的剧毒，都是在这人的身上散发出来的，难道他就是毁灭了这天元商会的人？
<br/>　　楚风眠这个念头刚刚是从脑海之中闪过，突然之间这一道身影，再次出手了，身形一闪，冲到了楚风眠的面前，又是一拳对着楚风眠轰击而来。
<br/>　　楚风眠不敢小视，顿时天涯剑匣显现而出，数道剑光猛然打了过去。
