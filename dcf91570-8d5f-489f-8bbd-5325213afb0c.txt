　　第100章
<br/>　　“我信你个大头鬼！”
<br/>　　一股无名火气往天灵盖撞，呼尔勒突然明白过来：这货是在耍自己啊！
<br/>　　戎族生活在荒僻之地，对他们的称呼有蛮，夷，戎，狄种种，咬文爵字，翻书细细查去，这些字表达的都是同一个意思：大家都是猴子，我们从树上下来垦荒种粮时，你们还在森林里和恐龙捉迷藏呢。
<br/>　　在这种时候他竟然敢耍自己，自己看上去那么像智障么，这种部落机密是谁泄露出去的！
<br/>　　不行，一定要杀了他，杀了这个愚蠢的，卑贱的山贼！
<br/>　　“弟兄们！”程大雷大吼，口吐惊雷：“跑呀！”
<br/>　　说着，他不顾一切的拍马，让马跑到最快。
<br/>　　程大雷何尝不想呼尔勒大战三百回合，最后一剑砍掉对方脑袋，用胜利洗刷对方给自己的鄙视。
<br/>　　可是，真不行啊。
<br/>　　别说三百回合，三个回合就得死在对方斧下，而那斧头还是自己的。
<br/>　　刘悲，张肥，徐神机，林少羽，秦蛮选择和程大雷同一个方向。
<br/>　　秦蛮大吼：“大当家先走，我留住他。”
<br/>　　“废什么话，兄弟们一起走，此仇改日再报！”
<br/>　　陆哼所率领的乡勇军也是狼狈逃跑，曾经程大雷以五人挡住千人，不过是仗着地利之便，可如今呼尔勒却只以十余骑，便赶着三五百人如丧家犬般。
<br/>　　戎族可真凶啊！
<br/>　　不知怎地，程大雷好像很拉仇恨，呼尔勒放开别的不管，只追程大雷一人。
<br/>　　程大雷回身抽剑想战一合，可等看到对方赳赳模样，识趣的把剑扎在马屁股上。
<br/>　　“跑啊！”
<br/>　　呼尔勒紧追不舍，一定要杀了这个狗贼，或许是为了不让机密泄露出去。
<br/>　　程大雷气喘吁吁，腹背淋漓透汗，骑在马上头晕眼花，眼前只看到白茫茫一片雪地，连路都看不清。
<br/>　　现在程大雷终于可以理解那些曾经被自己击败的溃兵，一支队伍只要成了溃兵，那么每个人都是洪流中的一滴小水珠。
<br/>　　虽有心杀贼，却无力回天。
<br/>　　真屈辱啊！
<br/>　　被人追得像只狗一样，呼哧呼哧喘着粗气，舌头都吐在外面。
<br/>　　真如果当初稍微抽些时间提升一下自己的战斗力，何尝不能与呼尔勒大战三百回合，也不必有今日之耻。
<br/>　　兵败如山倒，溃兵不如狗。
<br/>　　于此同时，程大雷拥有的恐惧值正在高速增涨。
<br/>　　不，还有机会，自己还没有彻底失败。
<br/>　　关键的那条系统提示：拥有一百万恐惧值便可随机兑换一名顶级武将。
<br/>　　一开始，这笔账并不好算，但如今这帐在清楚不过。
<br/>　　只要自己再拥有一名顶级武将，与秦蛮两个顶级联手，定能击败呼尔勒，或许只拥有一名就够了。
<br/>　　出城时自己的恐惧值已经有七十万，一直在涨。城里不断有人给自己贡献恐惧值，这支乡勇军到现在还以为程大雷和呼尔勒是一伙的，只不过黑吃黑打起来有些奇怪，他们贡献的恐惧值也都落在程大雷身上。
<br/>　　嘟，收到恐惧值688
<br/>　　嘟，收到恐惧值999
<br/>　　嘟，收到恐惧值666
<br/>　　……
<br/>　　恐惧值暴涨，如今已达到可怖的85万，还有机会，还没有输……
<br/>　　在呼尔勒眼中，程大雷逃，急急如丧家之犬，惶惶似漏网之鱼。可是在逃的时候，他回头看了自己一样，那眸底的杀意寒人骨髓。
<br/>　　他为何有这样冰凉的杀意，整个极北之地的寒意和这冰凉比起来都不值一提。
<br/>　　他又凭什么觉得能杀掉自己！
<br/>　　呼尔勒怒，想杀人，与程大雷的距离越来越近。
<br/>　　“秦蛮，替我挡住他。”
<br/>　　程大雷需要时间，如今的恐惧值已达到90万，快够了，很快就够了，只要再给他一些时间。
<br/>　　秦蛮持铁棍上，与呼尔勒战十回合，被呼尔勒冲过。
<br/>　　林少羽上，持一杆夺来的长矛，一个照面被击落马下。
<br/>　　张肥再上，却连呼尔勒的马尾都没接触到。
<br/>　　是否那名传千古的传说之将，亦是从一场场耻辱的失败中走来，饮马天下的余晖下，也能忆起不堪的曾经。
<br/>　　呼尔勒十余骑，追程大雷不休，似乎别人对他们都不要紧，只要今天能杀了程大雷就行。
<br/>　　程大雷身体紧紧贴在马上，那嗖嗖的箭刃破空音，就在他身周围响起。此刻，赤虎等人戏耍程大雷的意思更多些。
<br/>　　如草原中的灰狗，要等黄羊力竭之时，才送上致命一击。
<br/>　　但他们绝没有看到程大雷扣紧的牙齿，以及咬破流血的上唇。
<br/>　　90万……91万……92万……
<br/>　　快了，很快了，只要再给他一点点时间，他就能反败为胜。
<br/>　　而距离呼尔勒追上他，也快了。
<br/>　　99……100……
<br/>　　“点将！”
<br/>　　虚拟点将台上的篝火被点燃，似乎星辰的光芒被引入其中，一个崭新的英雄正在新生，即将踏破时空而来。
<br/>　　脑后风声响，已听到身后急促的马蹄声，呼尔勒的大斧照着程大雷劈来。程大雷滚落雪地，这一斧头砍在马上，呜呼倒地。
<br/>　　他已经跑得很远了，身下是小腿厚的积雪，手摁着雪地，扎人的冷。
<br/>　　呼尔勒笑，骑在马上趾高气扬的看着他，手中的斧头在阳光下是不反光的。
<br/>　　嘟，恭喜获得顶级武将一名……龙……
<br/>　　程大雷徒步向北逃，深一脚浅一脚，速度自然快不了。
<br/>　　刚才太紧张，没看清楚关键的系统提示，如今也没时间检查，只隐约记得有个『龙』字。
<br/>　　龙什么，什么龙……究竟是特么谁！
<br/>　　龙且……？
<br/>　　公孙龙……？
<br/>　　李云龙……？
<br/>　　该不会是……成……龙……吧？
<br/>　　或者是……
<br/>　　赵！子！龙！
<br/>　　程大雷越跑越慢，而呼尔勒却是越追越近，他回头双手撑地对视，呼尔勒骑马距他十步，程大雷是和马对视。
<br/>　　“瘦马啊瘦马，我知道我平日待你不好，但此时此刻，你若有灵，就救我一命。”
<br/>　　瘦马澄清的马眼一无所有，忽地昂首，向天长嘶。呼尔勒控马之术绝佳，但此刻竟被掀翻马背，却是他平生以来第一次。
<br/>　　瘦马奔到程大雷眼前，程大雷一矮身窜了上去，现在他能理解军人和马的感情，这种被毛戴角的畜生却能通人性。
<br/>　　有马助力，向前狂奔，呼尔勒也从手下那里要了匹马急追。
<br/>　　前方有座桥，桥上有个少年，他靠着石桥，程大雷骑马经过的时候多看了他一眼。
<br/>　　少年也奇怪的看着程大雷，等呼尔勒路过的时候，他喊道：“请问这位壮士，你是否见过一位眉目如画，风姿潇洒的美公子？”
<br/>(本章完)
