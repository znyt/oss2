　　也就在这个时候。
<br/>　　卫星手机里传来了一阵嘈杂的信号。
<br/>　　“求救信号？”
<br/>　　林峰愣住了，只听到里面一个男人略带惊慌的声音不断的重复着。
<br/>　　求救！求救！侦查大队遭受到袭击！请求附近幸存者救助！我方坐标位于XXX。
<br/>　　“居然是侦查大队的。”
<br/>　　军部大体上可以分为三个系统。
<br/>　　第一就是去外面探索的侦查大队，第二就是负责战斗的战斗部队，第三就是负责拱卫曙光基地的安保部队。
<br/>　　“好像距离我这里不远，既然如此那就去看看情况，若是可以救就施之援手，若是情况非常糟糕，那么我也无能为力了。”
<br/>　　林峰立即动身前往了侦察大队的坐标。
<br/>　　此时此刻。
<br/>　　足足三十多人的侦察兵被困在一个楼房里，作为侦察兵，他们的火力不是很猛，携带弹药有限，但是他们的作用却不可忽视。
<br/>　　毕竟侦察兵就是人的眼睛，如今的雷达早就已经失效了，所以就需要靠士兵来进行侦查，若是一个人的眼睛没了，肌肉再壮实，身子再高大，那也没有用处。
<br/>　　这一次，这个小队就是负责出去调查情况，回来的路上却被一群变异毒蜂给顶住了。
<br/>　　这群变异毒蜂足足有几百只。
<br/>　　他们小队不得已被困在楼内。
<br/>　　“这一次无论如何也要把调查来的情报送回去。”
<br/>　　一个汉子郑重道。
<br/>　　然后他看着身边的一个十七八岁的少年。
<br/>　　“狗娃子，你是我们这个队伍里最年轻体力最好跑得最快的，你以前学习过跑酷，所以你是最有希望把这些待会基地的。”
<br/>　　“队长，我一定会完成任务的！”
<br/>　　狗娃子认真的点了点头。
<br/>　　“队长，求救信号已经散发出去了，这里距离基地太远，干扰太强，所以直接给基地求救是不可能的了，就是不知道附近有没有幸存者。”
<br/>　　队长摇了摇头，“若是有幸存者，他们来不来也是个问题。”
<br/>　　众人都沉默不语。
<br/>　　话说林峰来到这个地方后顿时就看到了一大群毒蜂在那里飞来飞去，这些蚊子，个个都有人的脑袋那么大。
<br/>　　“我去，这么多？”
<br/>　　林峰扫了眼最起码也是几百只了。
<br/>　　“这怎么搞，这么多毒蜂，虽然都是些二阶的，最强的也不过三阶，但是这玩意屁股上的那针看起了都吓人，被这东西叮一口，最起码也要丢半条命吧。”
<br/>　　林峰立即有了退意，能帮就帮，毕竟军民一家亲，但是帮不了，林峰也不会当铁头娃，阎罗殿上装好汉。
<br/>　　林峰正要离开，忽然想到了什么。
<br/>　　从物品栏里掏出了一张符箓。
<br/>　　这是刚才得到的火龙符箓。
<br/>　　在威力上比火焰符箓要强。
<br/>　　火焰符箓是一星级别的，这个玩意是三星级别的。
<br/>　　“系统，莫非符箓也和灵器一样都有级别之分吗？”
<br/>　　系统：没错，不只是符箓，丹药功法武学等等都有星级之分，你这张火龙符箓是三星符箓，也是D级空投内可以得到的最好的三星道具。
<br/>　　“原来D级空投内最好的就是三星级别的道具，也是，我的追风靴，苏云老师的火凤肚兜的都是D级空投内开到的。”
<br/>　　林峰想到此，直接拿出了这一张符箓，这是个好东西，这群毒蜂虽然多，但是有了这的话那应该没多大问题了。
<br/>　　想到此，林峰就立即派出千年僵尸去挡诱饵吸引马蜂群。
<br/>　　顿时几百只马蜂群朝千年僵尸蜂拥而来，犹如黑色乌云一样，遮云蔽日的。
<br/>　　“队长，那毒蜂群移动了！”
<br/>　　“什么？”
<br/>　　屋内的侦察兵已经做好了最坏的打算，但忽然听到一人喊道，一群人立即朝外面瞧去，果然，那原本盘旋在外面不肯离去的变异毒蜂群全都不见了，转而是朝一个蹦蹦跳跳的家伙追了过去。
<br/>　　“这家伙走路姿势怎么这么怪。”
<br/>　　队长纳闷道。
<br/>　　此时此刻的千年僵尸林峰给他换了个衣服。
<br/>　　海贼王里面海军大将的衣服。背后是两个正义大字。
<br/>　　“没想到真的有人回来救我们。”
<br/>　　“太好了。”
<br/>　　“不要高兴的太早，这可是毒蜂群，希望这个人不是鲁莽行事，不然我们就算死了，还要他来百搭一条性命。”
<br/>　　队长道。
<br/>　　众人都是眼巴巴的看着僵尸那边。眼中带着一抹希冀。
<br/>　　话说千年僵尸把毒蜂吸引过来，这群毒蜂飞的非常快，僵尸难免会被叮几口，但是，千年僵尸，水火不侵，刀枪不入，百毒不侵，什么毒素都不怕，这群毒蜂虽然对着他狂顶，就好像轰炸机一样不断的丢炸弹，但是，千年僵尸压根不怂，甚至身体都不颤抖一下，完全纹丝不动。
<br/>　　林峰掏出火龙符箓极其紧张的看着这冲来的毒蜂群，黑压压一片。
<br/>　　就在这毒蜂群距离林峰不过五米的距离，林峰直接把千年僵尸收入物品栏然后内力催动火龙符箓。
<br/>　　刷。
<br/>　　几乎是一刹那，一条火焰形成的火龙怒吼着从符箓之中飞了出来，一下子就将那群毒蜂全部吞入火海之中。
<br/>　　刷刷刷。
<br/>　　然后就可以看到那一只只毒蜂就好像小火球一样的扑簌簌的从空中掉落下来。
<br/>　　一招下去。
<br/>　　死了一大半。
<br/>　　剩下的一大群变异毒蜂群立即受到了惊吓，纷纷调过头飞走了，很快就没有了身影。
<br/>　　林峰松了口气，收起了火龙符箓。
<br/>　　这时。
<br/>　　几十个侦察兵陆陆续续从他们躲着的大楼内出来看到那群死掉的毒蜂都是震惊的看着林峰。
<br/>　　“居然就你一个人？”
<br/>　　“你怎么做到的。”
<br/>　　一群人都不敢相信自己的眼睛。一个人把这么多毒蜂都给干掉了。
<br/>　　“这一次多谢你了，我是侦查大队的副队长朱民，这一次，我们调查到了重要的线索需要立即回去禀报首长，这一次你功不可没！不知道你叫什么名字！”
<br/>　　一个汉子激动的说道。
<br/>　　“战士会林峰。”
<br/>　　林峰道。
<br/>　　“战士会的？我知道了，本来于情于理我要好好感谢你，但是情况紧急，我需要立即回基地通报，以后有机会我会亲自登门拜访你的！”
<br/>　　朱民道玩大手一挥，一群侦察兵没有任何的拖泥带水就这样走了。
