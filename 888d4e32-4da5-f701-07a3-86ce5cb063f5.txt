　　第190章
<br/>　　帝国十三州一百单八城，疆域是极庞大的，各地情况也有不同。有些地方穷，有些地方富，有些城池坚固，有些城池破败。
<br/>　　而幽州位于极北之地，又经常遭到戎族侵扰，则属于……又穷又破。
<br/>　　幽州辖地一共六城，分别是：黑石、落叶、铁河、赤沙、北镇以及杨龙停坐镇的幽州城。
<br/>　　在黑石城被破后，幽州所有城主的心都揪了起来。面对铁骑横行的戎族，自己能否守住城池，自己是否会重蹈薛半川的命运？
<br/>　　但随之又传来消息，山贼程大雷镇守青牛山，与戎族开战。
<br/>　　事情变得有些难以置信起来，大敌来侵，山河破碎，自有帝国的将领站出来保家卫国，击退敌戎，你一个山贼瞎得瑟什么。
<br/>　　大家就等着戎族攻破蛤蟆寨，砍下程大雷的脑袋挂在旗杆上，希望来生，这个山贼能知道天高地厚。
<br/>　　可接下来的事情不能说难以置信，简直就是匪夷所思。戎族一日未破寨，戎族二日未破寨，戎族十日未破寨……最后，蛤蟆寨足足守了两个月有余，戎族，未破寨。
<br/>　　这件事令所有人大跌眼界，心道是蛤蟆寨太了得，还是戎族太不中用。
<br/>　　也许是前者，也有可能是后者。也许戎族没有那么强大，也许现在的戎族已不是三十年前的戎族……
<br/>　　在程大雷坚守的过程中，大家的自信心再一点点回来，程大雷坚守的时间越长，大家的自信心就越强。
<br/>　　大家太需要一场胜利，帝国全体军人太需要一场胜利。用这场胜利洗刷三十年前的耻辱，告诉天下人，戎族并非不可战胜。
<br/>　　前赴后继，三十年来未曾有人做到的事情，也许一个山贼将会做到。
<br/>　　青州，天一城，码头上汇聚了太多的人，大部分人都是从幽州逃难过来的人。
<br/>　　“快些，快些！把东西都装在船上。”
<br/>　　“方城主，不多住些日子了么，说实话，我还没和方城主聊够呢。”
<br/>　　“不住了，不住了。”方伯山脸上笑嘻嘻，心里买买提。
<br/>　　妈的，再住下去，老子那点家当还不被你搜刮干净。现在程大雷守住了青牛山，自己已可回落叶城，到时候粮草兵员送到青牛山，自己在后方足可安枕无忧。
<br/>　　到时候，自己还是一城之主，失去的家业三两年就能回来。
<br/>　　“那我再送送方城主，你那个小侄子还感谢你送的那柄剑呢。呵呵，他很喜欢，听说方城主还有一柄刀，也是相当不错呀？”
<br/>　　“哎呦，实在不巧，我那把刀丢了。”
<br/>　　“丢了？”
<br/>　　“可不，我还打算改日送到府上，可偏偏就找不到了。”
<br/>　　“哎，那真是太遗憾了。”
<br/>　　正此时，乔野突然飞奔到方伯山面前，低声道：“城主，大事不好，我刚得到的消息……”
<br/>　　听罢，方伯山脸色大变，忽然大声唤道：
<br/>　　“老弟，修走，那把刀……找到啦。”
<br/>　　……
<br/>　　幽州城。
<br/>　　“蛤蟆寨的确守了两个月，戎族大军集结，至今尚未突破，据探子回报，戎族几番攻城，伤亡惨重。”李善言道。
<br/>　　“这个山贼……我还真是小看他了。”
<br/>　　“据说，草原上也遭了旱灾，这些戎族缺衣少粮，或许，他们也只是一群疲兵而已。”
<br/>　　“嗯，有些道理。”杨龙停惜字如金，他坐在帅案后，便是执掌整个幽州生杀大权的幽州王。
<br/>　　“主公，这可是个机会，我们趁其虚弱，大举进攻，说不得有机会将戎族一网打尽。”李善言。
<br/>　　杨龙停沉吟片刻，道：“戎族现在未必虚弱，还是再等等再看看才好，毕竟，那可是戎族呐。目前，就让蛤蟆寨的山贼多消耗他们一些。”
<br/>　　“是。”
<br/>　　“报……”随着一声长呼，一个探子闯进大厅：“王上，大事不好了。”
<br/>　　……
<br/>　　蛤蟆寨，程大雷正让手下喽啰尝试挖山道，然后他发现……做不到。
<br/>　　山石坚硬，即便有火药可以用来爆破，这依旧是不容易做到的事情。想真的在山底，挖一条隧道难于登天。
<br/>　　哎，想简单了。
<br/>　　程大雷看着山壁，仰头向上往，山壁陡峭，高约二三百米。当初选择在山谷内建山寨，就是看中这一点。
<br/>　　“我们的人，有人能爬上去么？”程大雷问。
<br/>　　徐神机看了一眼，道：“不算什么吧，山里人采药，有时候爬得比这还险呢？”
<br/>　　山贼不是戎族，他们就生长在大山中，有些采药为生的，药材长在悬崖峭壁上，就是靠两脚往上攀。
<br/>　　“如果真是这样，派几个好手上去，丢绳索下来好像也不是难事吧。”
<br/>　　有些事情只需要换个角度，本来难如登天，就会变得迎刃而解。
<br/>　　程大雷盯着山壁寻思着，怎样神不知鬼不觉的撤退，不引起戎族的主意。
<br/>　　这时候城头一个喽啰跑过来，道：“大当家，戎族好像退兵了。”
<br/>　　“我擦，终于退兵了。”程大雷长松口气：“我以为他们要和我耗个沧海桑田，海枯石烂呢。”
<br/>　　程大雷和几员首领来到城头，看到戎族的确在拔营，看着看着，程大雷皱起眉头。
<br/>　　他们好像不是向北撤兵，而是要……继续南下。
<br/>　　“大当家，咱们可以撤了？”
<br/>　　“撤，没看到他们还留了人在外面么，这是要堵死我们呐。”
<br/>　　程大雷道了一句，戎族的确没有撤兵的打算，他们分出一部分人守住蛤蟆寨，另外一部分人进兵幽州境。
<br/>　　戎族的战略目的究竟是什么，这是许多人都在思考的问题。他们究竟是想抢一笔就走，还是说想彻底占领幽州，乃至帝国更多领土。
<br/>　　但无论他们的目的是什么，其实程大雷都对他们没什么影响，所以他们和程大雷耗这么久，程大雷也挺难理解的。
<br/>　　现在看来，他们总算是想明白了，准备先打幽州，回过头来再收拾程大雷。
<br/>　　这一去，幽州必定生灵涂炭，伏尸遍野。
<br/>　　可明知道这一幕将会发生，程大雷却什么也做不了，他也不准备做什么。
<br/>　　戎族有戎族的目的，程大雷有程大雷的目的。程大雷的战略目的从不是保家卫国，替天行道什么的，他只是想要活下来，让自己的弟兄更多的活下来。
