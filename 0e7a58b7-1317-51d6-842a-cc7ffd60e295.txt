　　李承乾对于闺女的管教并不是很严，而且小丫头在她娘的言传身教之下，也不是一两句话就可以教明白的，所以尽管他很同情老程，但却并没有说什么，只是拍了拍闺女的脑袋：“爹和你外公还有事情要谈，你去外面玩吧，记得，不要闯祸，不要去折腾那些小猫小狗。”
<br/>　　“知道了爹爹！”琅琊答应了一声，对李承乾做了一个鬼脸，又对着老程摆了摆手：“外公，我出去玩了。”
<br/>　　“去吧，别伤到自己就行。”老程被外孙女可爱的样子萌翻，老脸露出‘慈祥’的笑容。
<br/>　　敢在老程的家里纵马，就算是李承乾都不行，可唯独他的这个外孙女让老程一点办法都没有，甚至为了这事儿他的几个儿子没少提意见，但都被他给无视了，谁让他重女轻男呢，而且程琳那丫头又护犊子呢。
<br/>　　当然，还有一点就是琅琊充分的继承了她娘那一份优良基因，七岁的小家伙如果发起狠来一柄二十六斤半的斩马剑对她来说并不比一把匕首重多少，等闲五、六个程府的家将竟然不是她的对手，很快就会被她揍的满地找牙。
<br/>　　这也是小丫头无法无天的依靠之一，只不过知道的人并不多罢了。
<br/>　　……
<br/>　　等到那小丫头离开，李承乾明显发现老程似乎大大松了一口气，用一种十分幽怨的目光看着他：“陛下，琅琊这小丫头你看什么时候你能带回宫去？”
<br/>　　“为什么？在这里不是挺好么？”李承乾反问道。
<br/>　　“挺好么？刚刚她骑马冲进来你也看到了，现在只要是这丫头一出现几乎立刻就是神鬼辟易，不管是丫鬟还是仆役，全都会被吓的躲起来。”程咬金叹了口气：“俺老程这一生就没服过几个人，除了你爹再就是秦二哥，结果现在又多了一个小丫头片子。”
<br/>　　“呵呵……”李承乾尴尬的笑笑。
<br/>　　能让老程说一声服，可见自己闺女的确是威力不小，可是话也分怎么听，能让老程这种滚刀肉说出一个‘服’字，对小丫头来说未必就是在夸她。
<br/>　　“你还别笑，现在这丫头也就是她娘还能治得了她，否则府中根本无人能治。”老程吐槽般的说道。
<br/>　　“呃……，这个事情咱们一会儿再说，一会儿再说！”李承乾见老程说的可怜便点了点头，而后岔开话题问道：“丈人，这几年在南海那边可有什么心得体会？能否说来听听？”
<br/>　　“这有什么好说的，不过就是每天吃吃海鲜，钓钓鱼，最多再找几个破岛抓几个土著用来‘下酒’。”程咬金很无所谓的说道。
<br/>　　“‘下酒’？什么意思？”李承乾听程咬金说抓几个土著‘下酒’，不由疑惑的问道。
<br/>　　在他看来‘下酒’那就是‘吃了’的意思，想不到这老家伙竟然还有吃人的爱好，真不知道应该如何说他。
<br/>　　程咬金并不知道李承乾误会了他的意思，见他问起便大咧咧的说道：“就是‘下酒’啊，让他们跳舞，耍杂技，反正怎么有意思怎么来呗。”
<br/>　　“呼……”敢情‘下酒’是这个意思啊，李承乾长长出了一口气，想了想又接着问道：“那么丈人觉得南海有没有征服的必要？林邑、真腊、骠国这些地方几乎都在一起，您觉得我们把他们一口吞了如何？”
<br/>　　“想法是好的，但不现实。”程咬金摇了摇头。
<br/>　　说到正事，这老头儿有些时候还是靠点谱的，所以当他说出不现实的时候，李承乾不自觉的皱起了眉头，条件反射般问道：“为什么？”
<br/>　　“水土不服！”程咬金深吸一口气说道：“林邑、直腊、骠国，说来都是蛮荒之地，除了有数的几个地方之外，大多数地方都是人迹罕至的丛林，我们的士兵都是北方人，到了那里根本无法适应那里的环境，一、两个月或许能坚持下来，但时间长了估计非战斗减员就能让你哭出来。”
<br/>　　“有没有其他办法，比如我们可以从南方招兵，岭南的士兵应该能够适应那里的环境吧？”李承乾认真的考虑了片刻，皱着眉头说道。
<br/>　　“一样不行。我跟你说，那边压根就不是人待的地方，毒虫横行，瘴气弥漫，就连他们本地人都很难生存，更不要说我们。”程咬金正色说道，眼下四处无人，只有他与李承乾两个，索性老程也不在装出一副二愣子的样子。
<br/>　　“唉……”李承乾见老程说的坚决，心中也明白似乎自己是有些想的多了，叹子口气不再说话。
<br/>　　“陛下，为什么你会对那里感兴趣？要知道，那里根本就不会有人喜欢，就连他们自己人都不喜欢。”程咬金见李承乾失落的样子，心中有些好奇。
<br/>　　“我不是对那里的土地有兴趣，我是对那里的环境有兴趣。”李承乾耸了耸肩膀，啧了一声说道：“丈人难道不觉得那里的环境很适合种植橡胶么？”
<br/>　　“俺老程可不知道这些，不过你既然这样说了，那就是吧！”程咬金一辈子都在打仗，你问他哪里适合用兵，什么样的地形适合用什么样的军阵他能如数家珍，可是李承乾跟他说橡胶无异于对牛弹琴，老程对橡胶的唯一概念就是那东西大唐没有。
<br/>　　李承乾此时也发现自己是在对牛弹琴，老程这货根本就不是谈这个的材料，叹了口气之后说道：“那就先这样吧，我先走了。”
<br/>　　“走？现在？”程咬金愣了一下，很快便反应过来：“你不带琳丫头走？也不带琅琊那丫头走？”
<br/>　　“我带她们走啥？在您这里住着我还有啥不放心的，没事儿！”李承乾大咧咧的说道。
<br/>　　程咬金头疼那个小丫头，李承乾难道就不头疼了么？好不容易找到一个原意接手的，他又怎么可能把那小丫头带回去。
<br/>　　可怜的老程，以前李承乾还是太子的时候，他还敢上去拉拉扯扯，可是现在某人已经当了皇帝，再拉拉扯扯就有些不像话了，若是被有心人知道，弹劾一下足够他喝一壶的。
<br/>　　所以老家伙只能眼睁睁看着李承乾大摇大摆的离开，欲哭无泪！
