　　叶少阳顿悟了。
<br/>　　相通了这一点，叶少阳脑海中跳出了从小就背过的道德经里的一段：天下皆知美之为美，斯恶已。皆知善之为善，斯不善已。故有无相生，难易相成，长短相形，高下相倾，音声相和，前后相随……
<br/>　　原来老君早在几千年前就堪破了有关善恶之道的真谛。
<br/>　　说起来，法师悟道，跟古代的哲人一样，过程很辛苦，也很微妙，但是想明白之后，总结出来的却是很浅显的道理。而且多半都是别人之前总结过的。
<br/>　　但只有自己所领悟到的，才是自己的道，不然你怎么知道前人说的就是对的？
<br/>　　叶少阳轻轻吐出一口气，睁开了眼睛。
<br/>　　体内那些剩余的戾气，尽数融入在了体内的罡气之中。
<br/>　　叶少阳立刻感觉耳聪目明，看东西的感觉也产生了一些变化，这种变化很微妙，似乎比以前更明亮了，像是整个世界刚经受过一场雨水的洗礼。
<br/>　　这次开悟，提升的不是境界，而是硬实力，叶少阳本来以为最多是炼化了一些戾气，尝试着运气，猝然发现，法术的强度竟然提升了数倍！
<br/>　　叶少阳当场就懵了，又尝试了几次，完全确定了这一点，也许是足够多的累计，让质变引起了量变，之前是灵仙初阶的他，如今一下子提升到了中阶以上，虽然看上去只是提升了一个小段，实力的差距却是巨大，通俗一点来说，自己现在的硬实力，是之前的大约1.5倍。
<br/>　　这是什么概念？
<br/>　　叶少阳感受着自己体内的力量，虽然还没跟人实战过，但他有一种直觉和信心，如今的自己，与空界那些阐教金仙和佛门大能，有一战之力了。
<br/>　　就是这么猝不及防。
<br/>　　积累了那么久，一直憋屈地被他们欺负，而今总算站在了同一个起跑线上。
<br/>　　叶少阳心中隐隐有种感觉，如今的自己，才算真正脱离了人间法术界的限制，真正走上了强者之路，也就是道风所说的三流强者。
<br/>　　虽然，前面的路还很远，那些真正的绝世强者，看上去还是离自己很远，但至少以后跟人打架不用总去叫人了。
<br/>　　叶少阳内心那个舒坦，赶紧把这个好消息告诉了芮冷玉。
<br/>　　芮冷玉听了，比他还要激动。
<br/>　　叶少阳一只手从后面搂着她的肩膀，满怀憧憬地说道：“等我有一天成为一流强者，我就可以带你回人间，那时候，任何人都阻拦不了了。”
<br/>　　芮冷玉伸手捏了下他的鼻子，喜道：“我相信你！”
<br/>　　两人回到花丛里，找了个地方坐下，聊了起来。
<br/>　　时间过得很快，后来在芮冷玉的催促下，叶少阳才跟她一起回到道观里，见到道风，叶少阳立刻说了自己悟道的事，道风没说什么，但神色中也能看出，是为他感到高兴的。
<br/>　　分别之际，道风再三嘱咐，让叶少阳回去之后，一定要弄清那个陈悦的身份，然后把她带到这里来。
<br/>　　芮冷玉一直把叶少阳和瓜瓜送到石碑旁边，叶少阳跟芮冷玉依依惜别，带着瓜瓜出去。
<br/>　　小九和林三生正在喝茶聊天。
<br/>　　叶少阳把道风写的“材料单”交给林三生，讲述了自己开悟的经过，两人也是又惊又喜，小九特意重新泡了一壶茶，四个人又坐下聊了一会，叶少阳说起道风要自己把陈悦带来的事，让林三生帮着揣摩下是什么意思。
<br/>　　林三生想了一会，突然说道：“少阳，我不知道这个陈悦的来历，但是我想到了一个人。”
<br/>　　“谁？”
<br/>　　“后卿。”
<br/>　　叶少阳怔了一下，立刻就明白了他的意思：当初芮冷玉意外救下后卿，那时候，后卿也就像现在的陈悦一样，完全失忆，想不起来自己是谁。
<br/>　　“那不至于吧，陈悦……看上去还可以。”叶少阳慢吞吞说道。
<br/>　　“后卿呢？在他觉醒之前，不也是看上去人畜无害，你们还给他取了个名字叫阿呆。”
<br/>　　叶少阳心中咯噔了一下，的确，当时的“阿呆”，看上去呆萌呆萌的，谁也不会想到那居然是僵尸王，而且觉醒后的他，带领尸族崛起，在青冥界掀起了一场血雨腥风，到如今战争还在继续，而他本人也吸收了将臣之血，将来甚至有机会成为一流强者……
<br/>　　如果当初不救他，或者干脆灭了他，也不会发生后来的这一切了。
<br/>　　瓜瓜有些不爽地冲林三生嚷起来：“我娘不是僵尸，她才不会像后卿那样！”
<br/>　　林三生没理他。
<br/>　　小九拉过瓜瓜，坐在自己腿上，认真地说道：“瓜瓜，那不是你亲娘，就算你们感情很深，也是建立在她不会伤害别人的基础上，万一她将来要杀少阳，你怎么办？”
<br/>　　瓜瓜咬着嘴唇道：“我不想假设这种事，而且，我对老大的感情，不比你们差，我知道怎么做。但如果她不是那样，那任何人也不能伤害她。”
<br/>　　小九满意地点点头。
<br/>　　叶少阳在瓜瓜脑袋上摸了一把说道：“如果她是好人，那就是我们自己人。”
<br/>　　喝完一壶茶，叶少阳也要回去了。小九依依不舍地送他们下山，然后林三生也要走了。
<br/>　　“你现在是大元帅，不知多少人想害你，你也多注意点，尽量别一个人呆着，尤其是赶路什么的。”分别之际，叶少阳想到这个，关照了一句。
<br/>　　林三生当即拍了三下手，几秒钟后，立刻从附近树丛里跳出来八个人，四个身穿甲胄，四个青衣道袍，八个人都面无表情，在林三生身后站成一排。
<br/>　　叶少阳愣了一下，大致扫了一眼，这八人额有灵光，一看就实力不错，虽然相对自己这个层面的来说，八个在一起也不是对手，但只要对手不是特别多，他们都能在第一时间纠缠住对手，让林三生有机会跑路。
<br/>　　保证林三生的安全，这是他们唯一的任务。
<br/>　　叶少阳耸了耸肩。
<br/>　　林三生笑笑，又拍了两下手，八个人一起纵身，消失在树丛里。林三生对叶少阳和瓜瓜拱了拱手，转身离去。
<br/>　　“这么多保镖跟着，他现在好拉风啊。”瓜瓜啧啧赞道。
<br/>(本章完)
