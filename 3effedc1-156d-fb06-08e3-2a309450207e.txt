　　第40章
<br/>　　这一天，怕是青牛山有史以来最屈辱的日子。
<br/>　　在这之后，青牛山的山寨总会想起被陆哼支配的恐惧和屈辱。
<br/>　　车马继续前行，跟在后面的人越来越多，都是各山寨的头目。包括飞虎寨的高飞虎高飞豹兄弟，秃头坡的熊大熊二……之后，没人愿再去回想这一日遭遇的屈辱。
<br/>　　行到杏花岭下，一丛杏树绿叶丛丛，小白狼带一堆女匪立在岭下。
<br/>　　陆哼将马停下，道：“将头抬起。”
<br/>　　小白狼缓缓抬起头，一张欺霜压雪的脸蛋当真是灼人眼睛，在这炎热的天气，只要看到她脸的人都感到一种清凉之意。
<br/>　　“你就是小白狼！”陆哼感慨：“果然美名远播的艳马！”
<br/>　　“请陆将军过山！”
<br/>　　陆哼持着铁棍，棍稍扫过小白狼额头的发丝，缓缓下落，最后杵在小白狼的胸脯上。
<br/>　　小白狼紧咬牙关，猛地昂起头，对陆哼怒目相视。可陆哼就用毫不遮掩的目光看着她，等着她发怒，爆发，然后将这扬名的艳马一棍拍死。
<br/>　　小白狼身后的众女谁不是各自暴怒，虽然落草为贼，自然没有普通女人那种沾衣挂袖即为失节的思想。可是，女人最起码的尊严也是有的。陆哼如此作为，当真是将小白狼的尊严一巴掌狠狠拍在泥淖里。
<br/>　　陆哼笑，用镔铁大棍扎了扎小白狼胸脯，道：“果然好大！”
<br/>　　小白狼再低头：“请陆将军过山！”
<br/>　　陆哼仰头大笑，笑得肆无忌惮，可是，在整个队伍中，如此肆无忌惮的人只有他一个。而他身后的士兵，却是保持着同样的沉默。
<br/>　　这才是真正的精兵，目不斜视，心无旁骛。
<br/>　　陆哼如此作为，激怒的不仅仅是杏花岭，还有青牛山各山寨的山寨。但大家都也明白，各山寨的山贼和陆哼所率领的士兵相比，根本就是一群乌合之众。
<br/>　　如飞虎寨是青牛山最大的一座山寨，人口数百人，算是不错了，但山寨有男有女，有老有幼，真正能上阵打仗的，也不过百十口人而已。
<br/>　　纵然能一拥而上，拼个死活，可陆哼身后的黑石城才真正令人恐惧。
<br/>　　至于让众山寨联手拒敌的事情，几乎是想也不要想，人心……并不是那么容易齐的。
<br/>　　所以，只有忍受屈辱，每个人心里的想法都一样，撑过今天，抹抹脸，就当所有事情都没有发生过。
<br/>　　“山贼而已。”陆哼口中轻蔑的吐出四个字，然后大声道：“抬枪！”
<br/>　　队伍前后的两队骑兵忽然握紧手中的枪，将一物拴在枪头上，高高扬起。
<br/>　　有人看清楚枪上拴着的是什么，忍不住发出一声惊呼，枪上所挂乃是人头。
<br/>　　人头高高扬起，随着队伍的前进在空中一晃一晃，这是朱家寨上下三十几口的人头。有白发苍苍的老人，有黑丝垂落的妇人，有头发还没长全的黄口幼儿……
<br/>　　朱家寨上下被一夜灭尽，人头都被砍断，所有人都在想，人头哪里去了。却原来，竟在今日出现，被高高挂在空中。
<br/>　　山贼之中并不缺乏胆大之人，手上沾血的也并不少，可是在今日见到这样一幅场面，山贼之中已经有人发出干呕之声。
<br/>　　多么屈辱啊！
<br/>　　山贼贱如狗，真的不是一句虚言，黄三元曾说帝国子民分为四等：一等皇族二等贵族三等平民四等奴才，而山贼……只能沦落为不入流。
<br/>　　在高贵的人眼里，山贼甚至不被当人看待，与山间的野兽，巷陌的猪狗同类。如这种不是人的人，杀了就杀了，甚至有一种清理垃圾的愉悦感。其实就算那种匍匐在地上，辛苦的平民对山贼也只是畏惧，而无任何尊重。
<br/>　　山贼，是不被人当做同类的。
<br/>　　队伍向前进，行到蛤蟆岭下。蛤蟆寨的位置算是青牛山入口，从落叶城到青牛山这是第一站，从黑石城到落叶城，这是最后一站。
<br/>　　蛤蟆寨的情形和其他山寨有些不同，山寨大门紧闭，寨门前是一片空旷的山坡，而山坡下却一个人也没有。
<br/>　　队伍到这里停下，陆哼抬起头，微微蹙眉。
<br/>　　“这里就是蛤蟆寨？为何没人出来？”
<br/>　　几乎所有人都抬起头来，目视着紧闭的寨门，百人组成的队伍忽然腾起一片杀伐之气。
<br/>　　高飞虎心中咯噔一下，不想看到的事情终于发生了。程大雷怕是没看到今天陆哼的做派，不然绝不会这么做。
<br/>　　生气嘛，大家都生气，屈辱嘛，大家都屈辱。可谁不是怀着愤怒，忍者屈辱在陆哼面前低下头来。大家都能低头，而你为什么不能低头。
<br/>　　紧闭寨门，闭门不出，这就是你表达愤怒的方式，可这样做又有什么意义。
<br/>　　陆哼将马头冲向蛤蟆寨方向，抬起手伸展五指。
<br/>　　“数五之内，寨门不开，与我破门，杀人，灭寨！”
<br/>　　随着陆哼的声音，所有视线追随他的手掌，刀出鞘，枪握紧，只要陆哼一声令下，上百人会瞬间变成杀人恶兽。
<br/>　　高飞虎心头跳动两下，和于虬然对视一眼，难道今天真的要见点血嘛。
<br/>　　吱呀吱呀……
<br/>　　随着木头摩擦的声响，沉重的寨门缓缓打开，随着寨门打开，一人一马出现在大家视线中。
<br/>　　他骑在瘦马上，肩上扛着宣花大斧，一副没睡醒的样子，气色不是很好。
<br/>　　高飞虎心头松口气，终于到最后关头，程大雷最后还是屈服。这样才对嘛，不要让事情搞得不好收拾，让陆哼从这里经过，就当今天的事情没有发生过。
<br/>　　陆哼竖起的手掌落下，点点头，似乎以此表现对程大雷的满意。
<br/>　　“我不想看到你带兵器。”陆哼道。
<br/>　　程大雷今日第一次见到陆哼，他年纪并不大，依旧英气勃勃，肤色焦黄，身材并不高，但身量很大。这并不是一个追求健美肌肉的时代，如陆哼这种身材代表了力量，强壮。
<br/>　　姓名：陆哼（小有名气的优秀武将）
<br/>　　年龄：29
<br/>　　技能：三招棍法
<br/>　　隐藏属性：无
<br/>　　程大雷认认真真打量陆哼一遍，然后打了个喷嚏，他风寒还没彻底好，脸上的气色也是源于此，然后他吐出两个字，声音却也并不太大。
<br/>　　“打劫。”
<br/>　　
<br/>(本章完)
