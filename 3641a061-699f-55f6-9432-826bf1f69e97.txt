        “晚上不用等我吃饭，有应酬。不过，我会在九点前回来。”傅夜沉看着童筱颖，随之又亲了一下她那微烫的脸颊。
<br/>
<br/>        童筱颖接着叮嘱道：“不许喝酒！”
<br/>
<br/>        “嗯，放心，我不喝就是。”傅夜沉微微一笑，“那我去公司了。”
<br/>
<br/>        童筱颖微笑着点了点头：“路上小心，我等你回来。”
<br/>
<br/>        连星儿是看着爹地妈咪秀完恩爱后，才偷偷笑着跑回自己的房间的。
<br/>
<br/>        其实，她家爹地妈咪并不是没有吵过架。
<br/>
<br/>        他俩吵起架来，整个家就像被南极给包围了一样。
<br/>
<br/>        人家爹地妈咪吵架，是“热闹非凡”，她家爹地妈咪吵架，是“开冷战”。
<br/>
<br/>        她家妈咪有一种本领叫，爹地明明在家，她的妈咪也能把爹地当空气，还能带动着她和哥哥一起对爹地视而不见。
<br/>
<br/>        不过了，不管谁对谁错，先低头示好的人总是她家那没骨气的爹地，总会变着法子把妈咪哄开心。
<br/>
<br/>        想到这里，连星儿打开了床头柜的抽屉，从里面拿出了一个精致的粉色铁皮盒子。
<br/>
<br/>        打开盒子后，里面放着一颗白色的小纽扣。
<br/>
<br/>        这是她小时候和小朋友一起在公园沙池里玩，突然被一条恶狗追到跌倒时，一个路过的小哥哥救了她，替她赶走了恶狗，小哥哥将她从地上抱起来时，她不小心从小哥哥的衬衣上拽下来的。
<br/>
<br/>        这么多年过去了，她已经不记得小哥哥长什么样子了，但是她却一直收留着这颗小纽扣，而且还记得小哥哥当初抱着她时，她看到了小哥哥的脖子正中央有一颗痣。
<br/>
<br/>        其实，她还挺想找到这个脖子正中央有颗痣的小哥哥。
<br/>
<br/>        连星儿捏起纽扣，然后拿起油性笔，在上面画了一个笑脸。
<br/>
<br/>        如果有缘再见的话，她一定要好好感谢这个小哥哥。
<br/>
<br/>        想到这里，连星儿又取下了脖子上这条爹地送给自己的铂金项链，将这颗小纽扣穿到了项链上。
<br/>
<br/>        “小纽扣啊小纽扣！让我和你的主人相遇吧！”
<br/>
<br/>        连星儿甜甜地笑着自言自语道，然后倒床便睡。
<br/>
<br/>        呼，还是家里她的大床最舒服。
<br/>
<br/>        私宅外后花园的八角亭里，童筱颖坐在石桌前，正在做建筑模型。
<br/>
<br/>        傅夜沉给了她一个温馨的家，也让她过上了她最想过的日子。
<br/>
<br/>        而且，季雅珊和季思妍这对姐妹偶尔也会上她这儿来坐一坐，三个女人一起喝喝下午茶吃吃茶点聊聊天。
<br/>
<br/>        她不想混入上流圈去结识那些豪门贵太，也没兴趣去攀比身价，她只喜欢现在的这种安逸，能做自己喜欢的事情的生活。
<br/>
<br/>        比如建筑设计还有室内设计，再比如这些建筑模型，她还拿过个人建筑设计赛的各种大奖，也有成立了自己的工作室，请了人替自己管理。
<br/>
<br/>        就算哪天傅夜沉对她没感情了，要跟她离婚了，她也能在经济上独立。
<br/>
<br/>        不过，童筱颖倒是真没想过傅夜沉会跟她离婚。
<br/>
<br/>        毕竟，她现在是有老爸和哥哥罩的人，傅夜沉才不会跟她离婚。
<br/>
<br/>        以前她还得为家里那两个淘气的小朋友伤脑筋，现在儿子在部队当兵，女儿也高中毕业了，所以，她也不用再为儿女操什么心了。
<br/>
<br/>        不过话说回来，她的那个鬼精灵小棉袄，已经到了谈男朋友的年龄了，她还真是挺期待，星儿那丫头将来会带一个什么样的小伙子回来。
<br/>
<br/>        连星儿这一觉睡到大中午才醒来，醒来的时候，洛羽的后援会粉丝群里早已炸开了锅。
<br/>
<br/>        趴在床上，拿着手机，连星儿顿时睡意全无。
<br/>
<br/>        今天洛羽和明希哲的粉丝掐架上了热搜。
<br/>
<br/>        洛羽争回电影《联盟》的男主角之位，被批是后台硬，明希哲的粉丝不服，认为一个十八线外刚出道的小明星，凭什么跟他们的大明星明希哲争角色？
<br/>
<br/>        对于粉丝之间的这种掐架，连星儿虽然站洛羽这边，但是却没兴趣去参与。
<br/>
<br/>        洛羽才没有后台，更何况，洛羽至今都不知道有她这么个小粉丝存在。
<br/>
<br/>        倒是她连星儿，有一个“万能”的爹地。
<br/>
<br/>        连星儿觉得，这种热搜看似对洛羽不利，却是一种对他最大的宣传力。
<br/>
<br/>        至少，能让洛羽一时间有了一定的知名度。
<br/>
<br/>        此时，后援会粉丝群的群主又发全体消息了。
<br/>
<br/>        明日上午十点，云城机场，给洛羽接机，大家有空的记得要去。明日电影《联盟》在云城电影城开机，历时一个月，想给洛羽探班的亲们，请在我这儿报名，截止时间为三日后。——群主。
