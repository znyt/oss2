　　荒古神拳！
<br/>　　荒古门之中最为强横的武技之一，荒古门的镇宗武技，虽然荒古门弟子之中懂得这一招的人许多。
<br/>　　但是对荒古之道的了解越深，这荒古神拳的威力，也就越发的强横。
<br/>　　这鸿轩的荒古之道，便是领悟到了极深的层次，一拳轰击而出，引动荒古之力降临，一同向着楚风眠轰击而去。
<br/>　　“这一拳的力量好强，这就是九域武技？”
<br/>　　青梦跟燕皇看到这一拳的降临，都是面色严峻，尤其是燕皇，他的境界其实比起这鸿轩，还要更高几分。
<br/>　　可是看到这一拳笼罩下来的一刻，燕皇的心中都是露出了几分危险的感觉。
<br/>　　哪怕是他，想要抵挡这一拳，都不是很容易，九域哪怕是没落，其中的武技，也要超越无数星域。
<br/>　　“荒古神芒都没有，也敢来我的面前放肆？”
<br/>　　楚风眠看到这一道荒古神拳轰击而下，眼皮子都没有抬起，刹那之间一道剑锋，突然闪烁而出。
<br/>　　嘶拉！
<br/>　　这一道剑锋之快，令人都无法看清。
<br/>　　众人只感觉到，一道剑锋闪过，在猛然看过去，那荒古神拳，已经是被无情的毁灭。
<br/>　　“鸿轩的荒古之拳，就这样被破了。”
<br/>　　许多荒古门弟子，还在惊叹之余，突然有人发出一声大喊，无数人猛然随着喊声，看向鸿轩的方向，突然呆住了。
<br/>　　在鸿轩的胸口之上，一道剑痕，清晰可见，随着这一道剑痕，那鸿轩的身躯，都猛然裂开，当即爆炸开来。
<br/>　　这鸿轩的肉身，被楚风眠的一剑之下，彻底毁灭。
<br/>　　“我的肉身！你敢毁我肉身！”
<br/>　　从这鸿轩的身躯之中，一团精血飞了出来，看到那毁灭的肉身，疯狂咆哮道。
<br/>　　他才刚刚踏入九劫古帝的境界，一旦是被毁灭肉身，都有着境界跌落的可能。
<br/>　　一剑之下，鸿轩的肉身直接毁灭。
<br/>　　一时之间，刚刚那些还一副看好戏样子的荒古门弟子都是愣住了，许久没有能回国神来。
<br/>　　鸿轩可是一位九劫古帝啊，哪怕鸿轩只是一位刚刚突破的九劫古帝，也是一位真正的九劫古帝啊。
<br/>　　面对其他的九劫古帝，也许鸿轩还会吃亏一些，看楚风眠只是八劫古帝的境界啊。
<br/>　　要论起境界，这鸿轩的境界，要比起楚风眠还要强横，可现在这鸿轩，就这样被斩了？
<br/>　　被楚风眠随手一剑，直接毁灭了肉身。
<br/>　　尤其是刚刚楚风眠斩出的这一剑，速度更是快到了一种不可思议的地步，就算是他们之中的许多强者，都看不清刚刚楚风眠的剑光。
<br/>　　而另一旁，那东宫绝，星百里的目光同样震惊，可是他们的目光都是落在了楚风眠手心之中，一把滴血的巨剑之上。
<br/>　　这一把巨剑，正是巨阙神剑，剑身之上，沾满这鸿轩鲜血。
<br/>　　“巨阙神剑，果然是巨阙神剑！”
<br/>　　“巨阙神剑，是曾经剑道之主的佩剑，此人居然能用，此人跟剑道之主，必然有着极大的关系，只怕是剑道之主的弟子之一，难道是剑道之主留下了传承，被此人得到？”
<br/>　　东宫绝的目光之中，露出了几分炙热之色。
<br/>　　七大宗门想要围杀剑道之主，一是为了剿灭当时的剑道门，独霸九域。
<br/>　　第二却是也想要得到剑道之主一身无敌的剑术，可当年七大宗门攻入天绝峰之后，没有任何一位剑道之主的弟子投敌。
<br/>　　哪怕是当时手无缚鸡之力的楚风眠，都是选择跳崖天绝峰，尸骨无存。
<br/>　　这剑道之主的一身无敌剑术，也就此失传，现在楚风眠的身上，居然有着如此惊天的剑术，他们必定是要得到。
<br/>　　“我必用此剑，屠了七大宗门。”
<br/>　　楚风眠微微挥剑，在这巨阙神剑之上的鲜血，都被甩了下去。
<br/>　　巨阙神剑，是曾经剑道之主的佩剑，楚风眠必定要用七大宗门的血，来给这巨阙神剑祭剑。
<br/>　　今日这鸿轩，就是第一个。
<br/>　　但是楚风眠还没有打算，这样一剑杀了鸿轩的打算，这鸿轩既然是敢触犯楚风眠的逆鳞。
<br/>　　那一剑杀了此人，也太便宜他了。
<br/>　　“给我过来！”
<br/>　　巨阙神剑从楚风眠的手心消失，重新飞到了天涯剑匣之中，楚风眠右手一动，一把对着那空中鸿轩再一次凝聚出来的肉身抓了过去。
<br/>　　同时另一边，在楚风眠的左手之上，一团白色的火焰，浮现而出。
<br/>　　这白色火焰，并非是寻找火焰，而是真正的魔火，足以灼烧灵魂的魔火。
<br/>　　“魔火？”
<br/>　　看到这楚风眠手心之中浮现出的白色火焰的一刻，在场的众人，心中都是有些不寒而栗起来。
<br/>　　魔火的出现，让他们想到了一种古老的刑罚，抽魂炼魄！
<br/>　　九域之中，最为凶残，最为厉害的刑法，便是抽魂炼魄，以魔火灼烧灵魂，灵肉合一，炼制成器。
<br/>　　一旦是将人炼制成器，其中的灵魂便是会永远的承受魔火的炙烤，是天下间最为狠毒的招数。
<br/>　　被魔火炙烤，可是来自灵魂的疼痛，无法忍受，哪怕是一瞬之间，就比起死亡千万次还要痛苦。
<br/>　　更何况是一直被魔火炙烤，一旦是被抽魂炼魄，可就是真的求生不得求死不能了。
<br/>　　“不！”
<br/>　　那鸿轩也是看出了楚风眠的举动，眼神之中露出深深的恐惧之色，他没有承受过魔火的灼烧，可是他曾经也在对付他人的时候，用过这一招。
<br/>　　那些人惨叫的样子，还在他的面前，历历在目，他可不想要被抽魂炼魄。
<br/>　　在楚风眠右手抓过去的一刻，这鸿轩重新凝聚而出的身躯，连连后退，想要逃出。
<br/>　　可是这鸿轩的身形，如何逃脱，都无法逃出楚风眠的手掌心，只见一股灵力催动之下，这鸿轩的身躯，非但是无法逃脱，反而是向着楚风眠的手心之中飞去。
<br/>　　“休得逞凶！”
<br/>　　一道身影猛然从人群之中飞了出来，这又是一名荒古门之中的九劫古帝，突然一拳向着楚风眠轰击而来。
<br/>　　在这荒古门之中，也有着跟鸿轩关系交好的人，如今看到楚风眠要将鸿轩抽魂炼魄，突然出手，来阻止楚风眠。
