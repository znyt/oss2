　　这次碰到公羊世家的人，也只能算是楚风眠的意外，无心之举救了他们，却被恩将仇报。
<br/>　　也幸好楚风眠的实力足够，才能将这些公羊世家的弟子反杀，不然的话，若是实力稍弱一些的话，现在的楚风眠就已经是一个死人了。
<br/>　　北域之中这种武道家族，无数势力，也许看起来正义凌然，但是背地里却不知道是一种什么样的人。
<br/>　　就算是七大宗门，光鲜亮丽，但是当初为了得到一本剑道古籍，就去围杀剑道之主。
<br/>　　“杀了这些人，真是便宜了他们了。”
<br/>　　寒月漓在一旁，暗恨一声道。
<br/>　　这种恩将仇报的小人，寒月漓也是极为痛恨，这样杀了他们，也算是便宜了他们。
<br/>　　“无妨，我们没时间跟这些小人纠缠，先赶去武王城吧，走。”
<br/>　　楚风眠站起身来，一跃冲天，化为了一道遁光，直接向着武王城的方向飞去。
<br/>　　“公羊世家，也算是北域有名的武道世家了，他们的空戒中，应该有着不少的财富。”
<br/>　　楚风眠一边遁光飞行，一边将空戒拿出来，将其中的无数财富，全部拿走，至于许多灵器，都被楚风眠融入到了剑阵之中去，为剑阵增添一份力量。
<br/>　　两道遁光，在空中划过，一直是来到了这北域之北。
<br/>　　临近海边的一尊巨大无比的城池，武王城。
<br/>　　接近到武王城，楚风眠便是观察到周围，有着不少像他一样的遁光，这些人，便也都是前来参加这次武王城武斗大会的人。
<br/>　　楚风眠在注意到他们的一刻，这些人也都在打量着楚风眠二人，不过大部分人的注意力，都是放在了寒月漓的身上。
<br/>　　生死境，先天神体，这已经是够引人注目的了，在这武斗大会上，任何人，都是敌人。
<br/>　　许多人看向寒月漓的眼神，都是如临大敌。
<br/>　　不过寒月漓倒是不在乎，她拥有玄寒之体，早已经是习惯了这种眼神。
<br/>　　在她刚刚前去北邙圣域的时刻，便是有这不少的种子弟子，嫉妒寒月漓的天资，想要教训教训她，却都被寒月漓大变，这才立下了她在北邙学院的威严。
<br/>　　面对敌意，只要将对方击败便是。
<br/>　　楚风眠跟寒月漓，两个人，大摇大摆之际，便是直接落入到了城门前。
<br/>　　这武王城，一共三个城门，东门，西门，便是侧门，只有这南门，才是真正的大门。
<br/>　　来参加这武斗大会的武者，哪一位不是天之骄子，自然没有人走侧门的道理，全部都来到了这南门前。
<br/>　　一时间这前门前，便是车水马龙，无数武者，四面八方都向着武王城涌了过来。
<br/>　　这些人中，许多的武者，都不是前来参加武斗大会的，只是来观看这三年一度的盛况。
<br/>　　历年历届的武斗大会中，都有着不少黑马天才崛起，其中的不少人，都是如今成为了各大势力的高层，成为了北域的大人物。
<br/>　　也有不少人现在想要看看，这一届中，到底是诞生了什么人物。
<br/>　　“我们这次是来参加武斗大会的，不需要在这大门前排队，直接飞进去就是。”
<br/>　　寒月漓开口道。
<br/>　　他们来参加武斗大会，自然不需要排队进入，这南门前的队伍，只怕是排上一天一夜，都不一定足以进得去。
<br/>　　楚风眠跟寒月漓的遁光，来到这武王城前，没有落下，直接便是向着武王城中飞去。
<br/>　　“这两个人是谁？怎么如此嚣张？”
<br/>　　下方的一尊武者，看到楚风眠跟寒月漓的遁光，直接是飞了进去，有些不满的说道。
<br/>　　“居然不排队，直接进入其中？”
<br/>　　“白痴，那两人显然是这次来参加武斗大会的，不然的话，谁敢擅闯这武王城。”
<br/>　　在他身旁的另一位武者，有些羡慕的开口道。
<br/>　　“什么时候，我也有资格参加这武斗大会就好了。”
<br/>　　“哼，这武斗大会，至少也要接近到北域百绝榜的天才，才能参加，就凭你，一辈子也不可能。”
<br/>　　令一尊武者，冷冷的开口道。
<br/>　　这样的议论声不止，无数武者都在议论着，这一届的武斗大会，到底是谁会夺冠，谁又会突然杀出来，成为一匹黑马。
<br/>　　一入武王城，楚风眠便是一眼看见，其中足足是有着十几万武者，这些武者，几乎都是北域中所有天才强者的聚集了。
<br/>　　这大部分的人，都只是来看热闹的，都没有资格参加。
<br/>　　真正有资格参加的，每年也就千人左右。
<br/>　　不过想要从这一千个人中，想要得到前十的名次，也不是一件容易的事情，至少也要北域百绝榜上，甚至是百绝榜前排的那些人物，才有可能，做到这一点。
<br/>　　当然像是历年历届之间，也的确是会有着特殊的黑马出现，上一届中，便是有着一位名为叫做毒手魔君的地魔门弟子，从默默无闻，甚至都没有上北域百绝榜的名次，到一跃拿到的第三的排位。
<br/>　　随后更是进入归墟天，有着极大的奇遇，如今已经是达到了生死境高阶的地步，一身实力，足以就连生死境巅峰的人物，都要击败，已经是成为了地魔门的大人物。
<br/>　　扬名立万，跟归墟天的宝藏，这两点，刺激着北域中的无数天才，都来参加这武斗大会。
<br/>　　就连许多隐秘势力的天才弟子，都来参加。
<br/>　　武斗大会，称得上是北域三年间，最为热闹的盛况。
<br/>　　“来者止步！”
<br/>　　就在楚风眠跟寒月漓二人，刚刚飞入到武王城的一刻，两道身影，却是直接飞了过来，看向楚风眠开口道。
<br/>　　“二位是来参加这一次武斗大会的，可否有着证物？”
<br/>　　“嗯。”
<br/>　　楚风眠也不多费口舌，在他的空戒中，便是出现了两个金属牌，上面刻画这武王二字。
<br/>　　“原来两位是北邙学院的阁下，请进。”
<br/>　　这两名武者，看了一眼，便是恭敬的说道，直接放行。
<br/>　　楚风眠跟寒月漓，便是直接进入到了这其中。
<br/>　　“哦？这一次倒是有着不少人啊。”
<br/>　　楚风眠扫视了一圈，便是在其中，一眼发现了不少的身影，像是曾经在楚风眠北邙圣域中见到过的。
<br/>　　王党的党首陆画，就在其中，还有这许多北邙圣域的核心弟子，这一次都来参加了。
