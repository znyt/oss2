　　第558章
<br/>　　门一下被撞开，一队人涌入房间。人人身穿短打衣衫，表情狰狞严肃，双臂粗壮有力。
<br/>　　刘发财的茶还没送到口中，一下愣住了，结结巴巴道：“夫子，这是搞什么？”
<br/>　　没人理睬他，这些人看服饰便知道，是书院豢养的打手。他们不与刘发财废话，直接涌上来动手将刘发财擒住。
<br/>　　“嗳。”夫子挥挥手：“刘公子是书院的客人，你们不可太粗鲁。”
<br/>　　刘发财感激涕零：“夫子，你是个明白人呐。可为什么要这样……”
<br/>　　夫子也没再说什么，挥挥手让手下将他押下去，只听得刘发财一声声大叫从外面传来。
<br/>　　“我是冤枉的啊，我是冤枉的……”
<br/>　　他一路被抬着来到书院的地牢。刘发财是第一次知道，书院却还是有地牢的。平日惩罚一些不安分的学生，当然，也有书院的对头。
<br/>　　刘发财第一次从心里认同程大雷的话：读书人，果然都没有好心眼啊。
<br/>　　当然，书院毕竟是书院，就算是地牢，收拾得也是蛮干净。牢房内铺着稻草，外面挂着长明灯。不过，书院的校工平时肯定不用功读书，没有受过圣贤熏陶。
<br/>　　夫子都说了要对刘发财温柔些，他们也没听进去，依旧粗鲁得将刘发财丢进牢房。虽然有稻草垫着，仍摔得刘发财浑身酸疼，骨头都像裂开一般。
<br/>　　“哎呦，哎呦……”刘发财倒在地上呼疼不止。
<br/>　　“那个……你没事吧？”
<br/>　　一个声音在刘发财耳边响起，他抬起头来，恰与楚云生的目光碰撞。
<br/>　　“你说我能没事嘛。”刘发财揉着胳膊坐起来，道：“你是怎么会事，为什么被关进来？”
<br/>　　楚云生抱膝坐在墙角，道：“我也不知道为什么。”
<br/>　　刘发财没好气的瞪了他一眼，因为今天的事情，连带着天下读书人在他眼里都是可恨的。
<br/>　　这楚云生不知道什么情况，的确是稀里糊涂被丢进牢中的。但刘发财大概已猜出些什么。
<br/>　　还能为什么，不就是程大雷的身份暴露了。他一去不回头，自己和楚云生二人却被丢进牢中。
<br/>　　如今就算骂程大雷十八代祖宗，也是没半点用处。凡事还得靠自己，想办法如何脱身。
<br/>　　“这件事似乎和欧公子有关系？”楚云生忽然道。
<br/>　　“嗯？”刘发财眉头皱了起来，忽然笑笑道：“你觉得会是什么关系？”
<br/>　　“我怎么猜得到，总是觉得该和他有关系的。”
<br/>　　“哈。”刘发财笑道：“不过和谁有关系，反正和我没关系，我是冤枉的啊。”
<br/>　　说到这里，刘发财忍不住对书院破口大骂，上到夫子，下到书院的仆人童子，恨不得说，书院门口两只石狮子都不是干净的。言语粗鄙，却又令人浮想联翩。
<br/>　　所谓非礼勿言，非礼勿听，这些话传到楚云生耳里，就有些不中听了。
<br/>　　“好了，我相信书院是个有公道的地方，他一定会还我们清白的。”
<br/>　　“书院是个有公道的地方，哈，我当真没听错吧。书院有个屁的公道。”刘发财来了精神：“你去琅琊城问问，看书院究竟有没有公道。多少读书人，没成名的时候花魁养着，一朝成名了，就把过去的姘头忘在脑后，嫌弃人家出身不好。就连夫子当年都……”
<br/>　　“夫子当年怎么了？”楚云生来了好奇心。
<br/>　　刘发财开始口若悬河起来，说得故事半真半假，当然还是胡编乱造的成分居多。他本身就是个骗子，妖言惑众的隐藏属性也不是白给的，说到最后，不仅楚云生将信将疑，就连刘发财自己都快信了。
<br/>　　隔墙有耳，现在牢外就有人侧耳听着。他们二人是主管书院纪律的张明理与李亚子，算是书院的暴力部门。
<br/>　　二人躲在外面，是希望从刘发财与楚云生的谈话中探查真相。确实是没想到刘发财竟然说出这样一番话。
<br/>　　“夫子当年真的……”张明理张大了嘴巴。
<br/>　　李亚子脸色极其难看，冷喝一声道：“好了，把他们二人分开，别让他们在一起胡说八道。”
<br/>　　刘发财正说到得意处，感觉自己的口才从未像今日一般好。楚云生虽然不是很愿意听这个，但此刻也是听得津津有味。
<br/>　　这时候，牢房外出现两个人，身高模样各不相同，脸色却是一般的铁青。
<br/>　　张明理将牢房的门打开，冲楚云生道：“楚公子受委屈了，请。”
<br/>　　刘发财睁大眼睛，看着楚云生被放出去，张明理引着他向外走去。
<br/>　　“凭啥呀，凭啥放他走不放我啊。”刘发财大声吵嚷：“我不服，我要上诉。”
<br/>　　“莫着急。”李亚子挡在他面前：“对你，我们也另有安排。”
<br/>　　楚云生当然和刘发财不同，楚云生的来历是清白的。他在琅琊城住了三年，做过什么事，说过什么话，很容易就能查到。就算书院内部，大家也听过这个执着要考入书院的瘸子。
<br/>　　而刘发财……他究竟姓甚名谁，来自何方，要去哪里，书院便一无所知了。而且他那副相貌，想把他当做好人，说服力的确差了些。
<br/>　　张明理引着楚云生出了大牢，来到茶厅，让童子准备茶汤。
<br/>　　“楚公子莫要生气，在下向公子赔罪了。”
<br/>　　楚云生颦着眉头，此刻天还未亮，但书院却给他一种肃杀的味道。究竟是发生什么事了，令书院这庞然大物拿出了战斗姿态。
<br/>　　“欧公子是不是有什么问题？”楚云生问。
<br/>　　“哈，公子果然聪慧。”张明理笑笑道：“对那位欧公子，阁下了解多少？”
<br/>　　“萍水相逢，只不过承了他一些恩德。”楚云生道：“欧公子究竟出什么事了，能否烦请阁下将详情告知？”
<br/>　　“滴水之恩，当涌泉相报。”张明理赞道：“楚公子不愧读书人的风骨。不过，我若告诉阁下，你那位恩人，其实是杀人如麻的魔头呢？”
<br/>　　“他是……”楚云生一怔，脑海里冒出一个大胆的猜测：“程大雷。”
<br/>　　“公子了然。”
<br/>　　楚云生一下呆坐在椅子上，呐呐良久无语。
<br/>(本章完)
