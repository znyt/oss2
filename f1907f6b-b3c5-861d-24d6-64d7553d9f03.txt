夏夕绾其实可以理解的，陆寒霆出身高贵，受着最高等的教育，夏妍妍救过他，对他有救命之恩是不争的事实，他无法在夏妍妍的背后说起当时一段恩，这是男人最起码的修养。
<br/>
<br/>        夏夕绾没有再追问，其实她问出这个问题时就后悔了，因为无论他说什么，她都会不开心。
<br/>
<br/>        聪明的男人就应该像他这样一笔带过。
<br/>
<br/>        况且，除了救他的事情，他对她并没有任何的隐瞒，就连送夏妍妍去F国圣黎院的事情都告诉了她。
<br/>
<br/>        可是…
<br/>
<br/>        女孩子生气的时候都喜欢无理取闹钻死胡同，夏夕绾也是的，现在她满脑子都是，夏妍妍不过就是救了他，既然什么都坦白了，为什么就当时怎么救他的不告诉她？
<br/>
<br/>        男人这样遮遮掩掩回避的时候，就是有猫腻。
<br/>
<br/>        夏夕绾实在无法对这件事保持心静如水的态度，这时劳斯莱斯幻影已经停在了机场门口了。
<br/>
<br/>        夏夕绾一刻一分一秒都不想多呆，她打开了安全带，伸手去拉车门，想要立刻下车。
<br/>
<br/>        这时陆寒霆一把拽住了她的纤臂，高大英挺的身躯欺压了过去，大手捧起了她的小脸。
<br/>
<br/>        “你干什么，放开我，翎翎回来了，我要进去接机！”夏夕绾用力的推搡他。
<br/>
<br/>        陆寒霆将她牢牢的禁锢住，“绾绾，我们的话还没有说完呢，夏妍妍的事情我已经跟你坦白，我和她之间并没有私，你不要跟我生气了好不好？”
<br/>
<br/>        “不好不好，陆寒霆，你应该知道我这个人眼里容不得沙子，夏妍妍第三件事一日不提，你就要跟她牵扯到那一日，等你先将自己这些事都处理干净了吧！”
<br/>
<br/>        陆寒霆见她这一次是真的生气了，也开始抗拒他的触碰和亲昵了，他蹙了一下英气的眉心，“陆太太，你真是没良心的小东西，如果我是你，在知道夏妍妍喜欢我的前提下，难道你不是应该更紧的攥牢我的心，不让别人得逞么？”
<br/>
<br/>        “这一次我偏要反套路，如果你真的跟夏妍妍纠缠不清，到了滚床单的那一天，我一定会给你们递套子！”
<br/>
<br/>        她说什么？
<br/>
<br/>        陆寒霆也有点生气了，他将她拽进自己的怀里，低眸就往她的红唇上亲去。
<br/>
<br/>        只想堵住她的嘴，不让她再说话。
<br/>
<br/>        刚才在客厅里，她就没有让他亲。
<br/>
<br/>        男人身上那股干净清冽的气息萦绕而来，还带着一股凌冽侵略性的气场，夏夕绾不停的躲，就是不想给他亲。
<br/>
<br/>        很快她就低头在他结实的小臂上狠狠的咬了一口。
<br/>
<br/>        嘶。
<br/>
<br/>        陆寒霆吃痛，一下子将她给松开了。
<br/>
<br/>        夏夕绾迅速拉开了副驾驶的车门，头也没回的跑了出去。
<br/>
<br/>        陆寒霆将挺括的后背倒进驾驶座椅里，抬手遮了一下眼角的猩红，她给他亲的时候可以要人命，不给他亲的时候更要命。
<br/>
<br/>        结实的小臂上已经多了一圈小小而秀气的牙印，这一次她咬的很用力，一点都没有心疼他。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾进了机场大厅，几乎一眼就看到了人群里的叶翎。
