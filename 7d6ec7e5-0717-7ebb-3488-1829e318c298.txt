　　“请大家排好队，保持好秩序。”一个矮胖子大叔，带着一个滑稽的小红帽，拿着大喇叭不断指挥。
<br/>　　二十多个人从大巴车上下来，一边好奇的打量四周，一边按照导游的安排站好队。
<br/>　　“再次重申一遍，这里非常危险，请务必管好自己的好奇心。”通红的酒糟鼻越发红润。
<br/>　　带着一丝丝神秘恐怖意味，胖导游森然道：“这里是枯枯戮山，是最恐怖的暗杀家族揍——敌客家族居住地。”
<br/>　　“喔！”忍不住打了个哆嗦。揍敌客的名声太大了。
<br/>　　胖导游满脸严肃，心里却偷着乐。他最喜欢看一众游客被吓呆的表情。
<br/>　　他不止一次带人来此参观。知道揍敌客家的规矩，只要不擅自闯入揍敌客家族地盘。绝对平安无事。
<br/>　　扫了一圈游客，果然，每个人眼中都有着恐惧。刚才还散漫的动作，不约而同的克制谨慎起来。
<br/>　　嗯。还不错。咦？这小子是谁？怎么那么镇定。
<br/>　　胖导游盯着那个青年看，高高瘦瘦，皮肤白皙，眼睛狭长。猛一看，竟有一种邪意的美感。
<br/>　　高？瘦？美？矮胖导游低头看了看自己竖一米五，横一米五的身材……扎心了！小白脸！
<br/>　　他觉得很有必要好好恐吓一下这个小白脸。
<br/>　　“不要怪我没提醒大家，这里的环境危险至极。每年不知道有多少猎人闯入枯枯戮山，结果全都死无全尸。”胖导游语气越发诡秘：“有一次，我带的一个游客，不听劝告，非要走到门口去看，结果……”
<br/>　　“结果怎么了？”
<br/>　　“发生了什么事情？”
<br/>　　看着越发紧张的众人，胖导游得到了极大的满足。
<br/>　　“结果，他毫无征兆的就被杀死了！脑袋都碎了！”
<br/>　　“呼！”
<br/>　　“太吓人了！”
<br/>　　胖导游脸色肃然，心里却乐开了花。被吓到了吧，被吓到了吧！
<br/>　　谁不知道，只要不闯进揍敌客家的院子，就绝对没危险。至于站在大门口看的事。来这里旅游，不就是为了看一看那个夸张大门的吗？哪里有什么暗杀事件。
<br/>　　咦？这小白脸怎么还不害怕？
<br/>　　看着那夸张至极的大门，苏重眼中闪过一丝缅怀，心里突兀有种轻松感觉。
<br/>　　虽然因为穿越附身，对揍敌客的感情并不深。但毕竟在这里生活了不短的时间。而且，正是这里，在最初时给了他庇护。让他得以快速成长。
<br/>　　“喂？不要走的太近，远远的看一看就行了。”胖导游发现，那个小白脸竟然走向大门。心里顿时一惊。
<br/>　　组建旅游团那么就以来，虽然一直没出事。但如果靠的太近，万一人家心情不好呢。
<br/>　　那可是揍敌客，杀人不眨眼的！
<br/>　　“你不要命了，不要靠那么近，快回来！”胖导游大惊失色。
<br/>　　那小白脸竟然还往前凑，找死吗？他刚想跑过去拉回来。可不远处的小木屋内，走出了一个人。
<br/>　　胖导游立刻停下脚步，一句话不敢说。那人只不过是个看大门的，可他么人家是给揍敌客看大门的啊！
<br/>　　完了完了！这小子死定了！
<br/>　　我也死定了！出了事，我这旅行团完全开不下去啦！该死的小白脸！可千万不能死啊！
<br/>　　抬头看着数十米高，布满诡秘花纹的大门。苏重嘴角一翘，不由露出微笑。
<br/>　　当初为了给自己突兀改变找理由，苏重努力表现精神病的潜质。到处宣扬，生活就是一场游戏，自己就是在不断通关。
<br/>　　按照当初的设定，这扇大门，还是个贯穿始终的大boss呢。
<br/>　　苏重抬起右手，猛然砸向高大钢门。
<br/>　　轰！
<br/>　　魔龙虚影骤然闪现，火车头一般的恐怖头颅向前探出。血盆大口张开，猛然一声嘶吼。
<br/>　　魔龙·吐息！
<br/>　　嗡！
<br/>　　大地剧烈晃动，好似超级地震。方圆千米内，鸟类全被惊吓的四处乱飞。
<br/>　　嘎吱！砰！
<br/>　　矗立在大地上，深沉而坚定的神秘大门，竟然被苏重一拳给砸开了！
<br/>　　“恭喜糜稽少爷实力大进！”皆卜戎满头大汗，小眼睛里满是震惊。
<br/>　　糜稽少爷竟然能把整个大门给推开啦？！那可足有256吨！
<br/>　　胖导游快要都都快要吓尿了！糜稽……少爷？！小白脸就是揍敌客？！刚才自己在吓唬的，就是揍敌客？他只觉双眼发黑，完蛋了！
<br/>　　随着魔龙法相凝聚，魔龙六段成就。苏重的身体强化到了极致。
<br/>　　即便是尼特罗的攻击，也无法轻易攻破苏重身体防御。身体力量，同步增强。
<br/>　　以前看来极其难以打开的大门，现在已经不能阻拦苏重脚步。
<br/>　　轰！
<br/>　　苏重刚走进大门，一个硕大狗头突然出现。猩红眼睛里全是冰冷漠然，嘴角裂开，闪烁着寒光的牙齿已经露了出来。
<br/>　　“三毛？”苏重眼珠转动，狭长眼睛一扫而过。不认识我了吗？魔龙气息一放即收。
<br/>　　呜呜呜！
<br/>　　刚才还气势汹汹，一副要下口的三毛，顿时就趴在了地上，再也不敢呲牙。
<br/>　　三毛很凶悍，但魔龙意志更霸道。
<br/>　　“糜稽少爷，欢迎回家。”不知何时，梧桐悄然出现在苏重身前。微微弯腰，恭敬行礼。
<br/>　　“好久不见了梧桐。”距离他上次离开揍敌客，已经过去了一年多的时间。
<br/>　　“少爷这一次，打算呆多久？”梧桐问道。
<br/>　　“应该会呆一段时间。”和尼特罗一战，让苏重充分了解了自身实力。
<br/>　　尼特罗无法快速击败他，可苏重也无法击败尼特罗。这还不是全盛时期的尼特罗。
<br/>　　苏重距离最强的百式观音，还有一段距离。魔龙法相的凝聚，毕竟取巧。当年尼特罗修炼十多年，才练成百式观音。魔龙还有这可以提升的余地。
<br/>　　而且，魔龙法相是意志的体现。并不是简单的能量凝聚，还有精神意志的打磨。
<br/>　　“魔龙七段终究太过极端，需要时间来进行打磨。”这一次，他打算留在枯枯戮山好好闭关。
<br/>　　到如今，苏重的实力已经彻底步入整个世界的顶尖阶层。即便有那些个诡异能力，苏重也已经有了底气。
<br/>　　揍敌客即便想要束缚苏重，也已经不可能。即便马哈揍敌客高深莫测，苏重也不怕。打不过，他还能跑。有了魔龙傀儡，苏重的机动能力更强了。
<br/>　　再说，他和揍敌客并不是对立关系。只不过是老一辈的人，想要安排小辈的人生而已。
<br/>　　对于魔龙七段的修炼，苏重有了些想法。但还不够具体。当务之急，是把魔龙法相淬炼的更加强悍精纯。
