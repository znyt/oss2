　　罗昊拿出挂在自己脖子上那枚玫瑰花枝造型的戒指，放在唇边轻轻一吻，随后把戒指又塞回衣服中，眼睛中跳动着充满战意的火焰。
<br/>　　龙隐部队的兵，只有站死，没有跪生。
<br/>　　罗昊把G36C自动步枪快慢机拨到半自动模式，他们现在弹药紧缺，在火力上占据不了优势，只能使用精确点射来进行火力压制。而对于训练有素，射击训练子弹不是按箱计算而是按吨来计算的龙隐部队特战队员来说，即便是半自动步枪，也能打出全自动步枪的效果。
<br/>　　包围着二层小楼的反政府武装人员的耐心，显然是被消耗的差不多了，开始蠢蠢欲动起来。
<br/>　　两个头上包着阿拉伯方巾，只露出一对眼睛的反政府武装人员，端着具有标志性的AK自动步枪，在身后两名扛着RPG-7火箭弹的反政府武装人员掩护下，朝着楼房靠近过来。
<br/>　　“注意。”
<br/>　　罗昊平举G36C，对身边其他三个人，说道：“狐狸和无常干掉火箭筒手，我和冰刃解决那两个拿AK的。注意狙击手，不要恋战，打完后，立刻带着噬蜂从后门撤离。我们现在弹药紧缺，不能继续固守在这里，只能在这些街道、巷子和楼房中跟他们打巷战和游击战。”
<br/>　　“明白！”
<br/>　　慕容飞和李毅峰两个人分别瞄准了那两个扛着火箭筒的反政府武装人员，即便是没有了狙击步枪，他们也依旧是神枪手。
<br/>　　噗！噗！
<br/>　　噗！噗！
<br/>　　慕容飞和李毅峰率先开火将两个火箭筒手爆头，罗昊和齐凡两个人紧接着扣动了扳机，将两个端着AK自动步枪的反政府武装人员爆头。
<br/>　　四声沉闷的声响过后，四名反政府武装人员倒在了地上，从他们头部涌出的鲜血，很快就在他们尸体下形成一滩血泊。
<br/>　　砰！
<br/>　　隐藏在小楼外的狙击手，立即朝着窗户的方向扣动了扳机，一颗俄制7.62mm子弹击中了水泥窗框，将一大块水泥崩碎下来，子弹擦着李毅峰的脸颊飞过，在屋子地板上，留下一个浅浅的弹坑。
<br/>　　李毅峰贴身靠着窗户边的墙壁，暗暗心惊。刚才自己如果反应慢半拍，子弹击中的就不是水泥窗框而是自己的脑袋。虽然没有跟那个藏在暗处的狙击手交过手，但是李毅峰能感觉到这是一个高手，稍有不慎，就会丧命。
<br/>　　这些反政府武装本来只是一些乌合之众，但是跟西伦科特有联系后，情况就发生了变化。西伦科特手里不仅有枪，而且还有一支“影子部队”，这支“影子部队”绝对算的上是一支精锐，不管是单兵作战还是多人战术配合，他们都足以甩开当地的反政府武装几条马路，当他们的教官完全是绰绰有余。
<br/>　　有了“影子部队”的训练，这些反政府武装人员的战斗力也是有了很大的提高，不仅是单兵作战能力得到了提升，彼此之间也是有了战术配合。
<br/>　　就如刚才那样，在有火箭筒手的掩护下，还有狙击手的配合，这样的战术配合，绝对不是普通反政府武装人员能有的表现。
<br/>　　“狐狸，带上噬蜂，其余人交替掩护，撤！”
<br/>　　罗昊趁着楼房外因为四名反政府武装人员被击毙造成的短暂震慑，让人抓紧时间撤离。
<br/>　　不管是罗昊，还是身边的李毅峰、慕容飞、齐凡和徐旭东，他们都不再是孤身一人，他们都有着自己的家庭，有着牵挂。
<br/>　　小楼外的反政府武装势力很快就反应过来，大声叫喊着朝着小楼的方向冲来，AK自动步枪、RPK机枪的扫射声就如同炒豆子般响起，子弹打在楼房墙壁，墙壁水泥层扑簌簌的往下掉落；那些挂在破碎窗户上的玻璃碎片，被全部击落下来，子弹穿过窗户飞入楼房内，令得楼房内的几个人不得不侧身躲避飞入楼房的子弹，严重影响到了撤离的速度。
<br/>　　“狐狸，带着东子撤出去，其他人掩护！”
<br/>　　罗昊手持G36C自动步枪，用精准的半自动射击不断的对楼房外的反政府武装人员进行着射击，但是对方强大的火力，还是让得楼房内的罗昊、齐凡、慕容飞三个人感觉到很大的压力。
<br/>　　齐凡的子弹最早耗尽，把打空子弹的自动步枪甩在身后，从快拔枪套中拔出手枪，对着外面朝着楼房涌来的反政府武装人员继续射击。
<br/>　　这次龙隐部队战术中队所携带的是P226SCT型手枪，这是一把使用Nitron表面处理的P226衍生型战术手枪，它的套筒前方有多条锯齿状突起防滑纹、战术配件导轨，准星方面考虑到夜间作战的关系，使用了SIGLITE夜间照门、TRUGLO氚光光纤准星，子弹口径除了原有的9mm口径外，还多了.40S&amp;W口径。
<br/>　　龙隐部队战术中队这次所使用的就是.40S&amp;W口径的子弹，毕竟这种子弹在弹道表现上比9mm口径子弹更加优秀。尤其是在使用了空尖弹后，虽然牺牲了子弹的侵彻力，但是在停止力量有了更加出众的表现。
<br/>　　但凡是被齐凡点名了的反政府武装人员，不是少了左半个脑袋，就是被打飞了右半个脑袋，红的、白的、红白混合在一起的，喷溅在街道上，令得原本在太阳暴晒下干燥滚烫的街道，变得血腥无比。
<br/>　　“RPG！！”
<br/>　　罗昊看到街道上一个反政府武装人员捡起身边一名同伴尸体边RPG-7火箭筒，半跪在地上，竖起火箭筒上的机械瞄准具，立刻大喊着从窗户边跑开，扑倒在地上。
<br/>　　轰！
<br/>　　火箭弹击中了楼房的铁门，将门炸开。铁门砸落在地面上，扬起一阵尘土。
<br/>　　罗昊把自己手中的G36C自动步枪扔给齐凡，对齐凡说道：“掩护他们撤出去！”
<br/>　　“你怎么办？！”齐凡接住罗昊扔来的G36C自动步枪，有些着急的对罗昊问道，语气中充满了担忧。
<br/>　　“放心，我不会想不开就这样出去跟他们玩命的。”
<br/>　　罗昊拔出龙牙军刀反握在手中，贴着门口墙壁藏起来，对着齐凡使了个眼色，催促着齐凡、慕容飞掩护李毅峰和徐旭东离开。
