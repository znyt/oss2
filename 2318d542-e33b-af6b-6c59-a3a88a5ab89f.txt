　　“有救兵来了！”
<br/>　　鲲鹏怔了一下，道：“我们这空间，只进不出，救兵来了又能怎样，等我师父神功大成——”
<br/>　　影魅打断他，说道：“来的是钟馗！”
<br/>　　钟馗！！！
<br/>　　鲲鹏也是一下懵了。
<br/>　　虽然他们一向不太看得起阴司，但阴司的那几个大佬，就算在他们眼中，分量也是极重的，更不要说钟馗这样的大佬。
<br/>　　经过这一番对话，影魅自己也冷静下来，说道：“他们落在几十外，倒也还好，就算搜查，也不会这么快来到，刚是我突兀了……继续布阵，逼迫叶少阳念咒。”
<br/>　　几个人立刻又加紧念咒。
<br/>　　鲲鹏开口，对叶少阳施压，“叶少阳，实不相瞒，你的救兵到了，不过却落在几百里外，等他们找到这里，一切早就结束了，你想要等他们来搭救，简直做梦！”
<br/>　　“你当我跟你一样傻逼吧，那黑影是影魅的弟子吧，速度是挺快的，但就这一会工夫，能穿越几百里，然后返回？”
<br/>　　谎言被拆穿，鲲鹏有点尴尬，冷笑道：“纵然如此，你依然必须无疑，你不跟着念经，那便看着他们一个个死掉，嗯？你听见我说的？”
<br/>　　发现叶少阳正抬头望天，忍不住抬头看了一眼，顿时大惊失色。
<br/>　　天空中飘着一面旗子，黑色的，拖着一根长长的旗杆，在空中迎风招展，盘旋着飞过来。
<br/>　　血海万魔幡！
<br/>　　“是道风来了！”鲲鹏大叫。
<br/>　　玉玑子听见道风两个字，差点当场就尿了，双手不听使唤地哆嗦起来。他怕道风，是真的怕。
<br/>　　“道风已死，这不是血海万魔幡。”影魅说道。
<br/>　　黑旗在空中飞行，旗面上似乎有一道金色的图案，大伙一起观望，正好这时候黑旗飞到他们头顶上空，在风中展开，上面的图案也被大伙看清：是一个骷髅头。
<br/>　　“钟馗的帅旗！”玉玑子率先叫起来。
<br/>　　“卧槽！”
<br/>　　小马一巴掌拍在叶少阳肩上，大喜道：“钟天师来了，咱们有救了啊！”
<br/>　　虽然跟钟馗不是特别熟，虽然也很吃惊他为什么会来，但叶少阳一伙人都相信，钟馗不可能是影魅他们一头的。
<br/>　　“别高兴太早，他不知道我们在这。”
<br/>　　黑旗在空中盘旋着，不断朝一个方向飞去，显然是在寻找，当然不会是用眼睛看，而是感知周围的生灵气息——对得道的邪物来说，感知力比眼睛要好用得多。
<br/>　　“屏息凝神，让帅旗过去！”影魅下了命令。
<br/>　　几个人停止了念咒，盘古僧也停止了念经，叶少阳他们在法阵之内倒是好过了，不少生灵望着天空中的帅旗大喊大叫起来。但是没有，结界的存在，隔绝了内外的气息，他们不管做什么，都不可能被帅旗感知到。
<br/>　　帅旗就要飞过去了。
<br/>　　“操啊！”
<br/>　　小马握紧双拳，大骂起来。
<br/>　　那些生灵都怔怔地站在莲叶上，眼中绝望的神色越发浓烈起来，如果没有这件事的发生，他们或许也都认命了。帅旗的出现，带给了他们生的希望，但现在，希望又飞走了。这种落差，让每个人的精神都要崩溃了。
<br/>　　叶少阳也没有办法。
<br/>　　诛仙剑阵，传闻能困住神佛，自己就算是上仙，对其也无能为力，否则也不会在里头呆到现在了。
<br/>　　玉玑子等人脸色开始好看，洋洋得意起来。
<br/>　　只要钟馗走了，再转过来，不知道什么时候了，到时候收拾了叶少阳，再慢慢对付他也不迟。
<br/>　　“叶少阳，连救兵都走了，你还有什么好说的。”影魅用得意的口吻对他说道，他没有五官，但听这语气就知道他在冷笑。
<br/>　　叶少阳没说什么，一向擅长在战斗中寻找机会的他，这次真的无能为力了，因为被阵法所困，从头到尾都被压着，很憋屈也很窝囊，但是一点办法没有。
<br/>　　突然，一道人影从影魅的身后直直地冲向了天空，影魅也察觉到气息异动，转头看去。是一只湿生邪灵。
<br/>　　之前跟着念了经，被放出去的，几个邪物一起站在影魅他们身后，老老实实地，丝毫没有存在感，本来他们存在的意思，就是刺激叶少阳、然后再打压他，他们跟着念经之后，任务就完成了。
<br/>　　不管是影魅还是鲲鹏，都没有把他们放在眼里，料想他们这种叛徒，又没什么实力，只能任人宰割，结果，偏偏是这些人中，居然出了问题。
<br/>　　“叶天师，我负了你，天诛地灭！”
<br/>　　这邪灵飞起的时候，丢下了这句话。
<br/>　　“你没负我！”叶少阳握紧了双拳，他已经这邪灵在干什么了——他要对帅旗示警，把钟馗引过来。大喊大叫是没用的，唯一的办法，就是引起灵力波动。
<br/>　　影魅一抬手，一道黑气射出，打在那邪灵身上，当场魂飞魄散，只剩下一堆精魄。然而……影魅能杀他，却不能阻止他体内魂魄碎裂时产生的魂力波动，波纹一般地在四周蔓延开来。
<br/>　　“好汉子！”小马也咬牙叫起来。
<br/>　　“不错！”吴嘉伟也动容不已。
<br/>　　这邪灵在飞起的那一刻，就已经知道了自己的下场，他一定是很懊悔之前的背叛，但是在这生死攸关的一瞬间，他还是做出了救赎的举动。
<br/>　　最后的闪光。
<br/>　　叶少阳心中叹息，却想起来，自己连那邪灵的名字都不知道。
<br/>　　所有人的目光，都移到了飞走的帅旗上面。
<br/>　　一秒，五秒，十秒……
<br/>　　帅旗陡然回头，朝这边飞了过来。
<br/>　　哈！
<br/>　　法阵中的所有人，都感到了强烈的激动，那邪灵没有白死，他的死，换来了他想要的结果。
<br/>　　谁说小人物做不了大事情？
<br/>　　影魅气的七窍生烟，但很快冷静下来，命令鲲鹏：“收剑！”
<br/>　　鲲鹏一怔，说道：
<br/>　　“师伯，不必这样吧，就算是钟馗，未必就能怎么样吧，而且我们可以用叶少阳来要挟……”
<br/>　　“要挟什么，一计不成，机会还多，若是诛仙四剑被夺走，那才是得不偿失！”
<br/>　　鲲鹏一想也是，本来今天这个局也不是终极之战，虽然太可惜了——离得手只差一步，但毕竟他们也没什么损失，当下一转身，手腕一翻，手掌中多了一片羽毛，朝面前那些个直愣愣站着的生灵飞过去——之前那邪灵的所为，让他迁怒到了这些人身上。
