　　那姜让，死了也就死了，楚风眠也懒得在乎。
<br/>　　但是这鸿轩，触犯了楚风眠的逆鳞，楚风眠要让他知道，死亡，对他来说，都是一种奢求。
<br/>　　“凝炼！”
<br/>　　轰隆！
<br/>　　楚风眠手心之中的那一团魔火，更是燃烧的无比剧烈起来。
<br/>　　在魔火之中，鸿轩感受到的痛苦，几乎是剧烈了数千倍，数万倍。
<br/>　　如果他还有着肉身，那几乎足以是让他疼死一万次了，可现在的鸿轩，只剩下了魂魄。
<br/>　　在这魔火之中，非但是不会晕厥，反而是感受的无比清晰。
<br/>　　这就是魔火的厉害之处，不会杀了鸿轩，却是要鸿轩，感受到无穷无尽的痛苦。
<br/>　　在场的其他人，都是足以清晰的听到鸿轩的惨叫声，一个个面色不寒而栗，纷纷退后，看向楚风眠的神情，如同一个恶魔。
<br/>　　抽魂炼魄的事，也许他们之中许多人都做过，但是这样，在这众目睽睽之下，当着无数荒古门强者的面，将鸿轩就这样抽魂炼魄，这简直是疯子一样的行为。
<br/>　　“小子，你做的太过了吧！杀人不过头点地，你这样做，跟魔头无异！”
<br/>　　一名天枢殿弟子，突然走了出来，看向楚风眠冷声说道。
<br/>　　“滚！”
<br/>　　楚风眠听到这天枢殿弟子的话，根本不跟他废话，怒喝一声，从楚风眠的头顶之上，战龙之爪再度浮现而出，向着那天枢殿弟子轰击而去。
<br/>　　对于七大宗门的弟子，楚风眠从来不会浪费口舌。
<br/>　　看着战龙之爪轰击而去，方才出声的那一名天枢殿弟子，都是脸色剧变，没想到如今楚风眠会直接出手。
<br/>　　这战龙之爪的威力，他刚刚可是亲眼目睹过的，那姜让都是被一招轰杀。
<br/>　　他的实力，可不比姜让强横到哪里。
<br/>　　“百里长老。”
<br/>　　这天枢殿弟子只好看向旁边的星百里，看到战龙之爪落下，震袖一挥，一股星辰之力，从星百里的身上爆发而出，跟这战龙之爪，凌空碰撞。
<br/>　　无论如何，他也不可能坐视不理，看着这天枢殿弟子，被楚风眠所杀。
<br/>　　“小子，你做的太过了吧，难道就凭你一个人，真的打算跟我们七大宗门为敌吗？”
<br/>　　星百里一步踏出，看向楚风眠，冷冷的说道。
<br/>　　“本来你只要好好交代你到底是从哪里，得到了剑道门传承，随后加入我们七大宗门之一，我们就可以不杀你的，你今日的所作所为，是要将你直接推入火坑之中。”
<br/>　　“不杀我？七大宗门的人，什么时候变的如此仁慈了？”
<br/>　　听到星百里的话，楚风眠嘴角露出几分嘲讽笑容，一字一字的说道。
<br/>　　“万年之前，围攻天绝峰的时候，天绝峰上所有剑道门弟子，所有奴仆，甚至方圆百里的生灵，可都被尽数灭绝了。”
<br/>　　“你们不过是想要得到剑道门的剑术罢了，说的如此冠冕堂皇干嘛。”
<br/>　　楚风眠目光看向了星百里，眼神之中露出几分复杂道。
<br/>　　“你们七大宗门不杀我，我也要灭了七大宗门，星百里，你的星家的人，我不想杀你，自己滚！”
<br/>　　楚风眠话中的最后三个字，几乎是吼出来的。
<br/>　　一想到万年之前的事，楚风眠心中的杀意，都忍耐不住了。
<br/>　　“万年之前的事，你怎么可能记得那么清楚，你到底是谁？”
<br/>　　星百里听到楚风眠的话，越来越吃惊。
<br/>　　当年七大宗门，为了毁灭剑道门，不留下任何的余孽，不止是将剑道门上上下下，无数弟子，所有奴仆都给斩杀。
<br/>　　甚至是天绝峰周围，方圆百里之内的任何生灵，无数武者，都被七大宗门，一同灭绝。
<br/>　　事到如今，天绝峰周围，都还依然弥漫着浓重的血腥味，万年之前，都散不去，无数冤魂在其中，几乎成为了一处绝地。
<br/>　　这件事，在七大宗门之中，现在都有些一些老古董知道了。
<br/>　　可现在楚风眠居然是知道，哪怕是楚风眠，得到了剑道门的传承，都不可能知道这种隐秘的。
<br/>　　除非是曾经剑道门之中，有人逃了出去，才有可能，可口口相传，又怎么可能像是楚风眠知晓的如此清除。
<br/>　　就在星百里脑海中思考的时候，一旁的那东宫绝已经是气得发狂了，他现在被燕皇跟青梦缠住，根本无法救下鸿轩，只能够是眼睁睁看着鸿轩被楚风眠抽魂炼。
<br/>　　看着鸿轩发出的一声声惨叫，东宫绝都有些忍耐不住了，对着星百里大声的吼道。
<br/>　　“星百里，不用跟这小子在浪费口舌，出手抓住他，将这小子抽魂炼魄了，他的一切秘密都会说出来的！”
<br/>　　“别忘了，七大宗门之间的约定！”
<br/>　　看着星百里似乎还有些犹豫，东宫绝又是大吼一声道。
<br/>　　七大宗门之间的约定。
<br/>　　在听到这句话后，星百里眼神之中的犹豫之色，一扫而空，猛然出手，震袖一挥，星辰之力在他的手心之中凝聚。
<br/>　　一股股的星辰之力，在星百里的手心之中，当即化为了一条星河，蕴含恐怖的星辰之力，向着楚风眠轰然落下。
<br/>　　刹那之间，楚风眠的身形，就陷入到了这星河之中，星辰之力，压迫这楚风眠的身躯，要将楚风眠给镇压。
<br/>　　“不滚，那就不要怪我了。”
<br/>　　楚风眠看着星百里出手，也是叹息一声，七大宗门之中，对楚风眠来说，稍微有些忌惮，让楚风眠不愿出手的，就是天枢殿了。
<br/>　　可现在，这星百里既然出手了，那楚风眠可就不会犹豫了。
<br/>　　哪怕是曾经有着情分，可现在的天枢殿，对楚风眠来说，也是死敌。
<br/>　　“凝。”
<br/>　　看着手心之中还被魔火灼烧的鸿轩，楚风眠大手猛然用力一捏，这鸿轩的魂魄，便是跟魔火，永远的凝聚在了一起，化为了一枚黑色的宝珠，被楚风眠如同垃圾一样的，扔在了地上。
<br/>　　在这宝珠之中，鸿轩的灵魂，便是被魔火永远的灼烧，谁也救不出来。
<br/>　　同时从楚风眠背后的天涯剑匣之中，两把灵剑，同时出鞘，祖龙至尊剑，巨阙神剑，同时被楚风眠握在了手心之中。
<br/>　　惊天的剑意，被楚风眠的背后爆发而出。
