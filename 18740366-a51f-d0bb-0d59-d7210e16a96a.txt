        “你们快下来，我请大家吃好吃的！”
<br/>
<br/>        连星儿说着，就把包包的拉链拉开，将里面的零食全部倒了出来。

<br/>        这一大堆的零食加起来，起码近一万块钱了，因为就那盒巧克力，就价值八千八百块。

<br/>        顾千凝识货，是因为她自己很爱吃巧克力，所以知晓全球各大品牌巧克力的价格。

<br/>
<br/>        “星儿，我可以开吃了吗？”路娅玟问道。

<br/>        连星儿笑道：“当然可以，别客气！”

<br/>        “星儿，巧克力吃完后，盒子可不可以送给我？”顾千凝接着问道。

<br/>        她们竟然在一个晚上，吃掉了一万多块钱！
<br/>

<br/>
<br/>        等到连星儿刷完牙从洗漱间里出来时，桌上的零食已经被整整齐齐地放好，那些垃圾也被收拾得干干净净。

<br/>        顾千凝和路娅玟纷纷点头。

<br/>        连星儿二话没说，就直接答应了。
