　　“巫！巫！巫！……！”
<br/>　　“妖！妖！妖……！”
<br/>　　惊天的厮杀声震动苍穹，磅礴的气势直冲九天，遮天的杀气撼动乾坤。巫、妖两族的战斗还在持续，浩瀚的波动直冲穹天，九霄之上，乌云翻滚，苍穹为之变色。
<br/>　　两族流出的鲜血，如今早已染红天际，两族成员的尸骨，如今也早已铺遍大地，堆积成山。事到如今，这场战争早已不是一两个人能够停止下来的了。战斗的发展，超乎想象。战场之上，瞬息之变，让人防不胜防，一旦大战起来，谁能阻止？
<br/>　　两族成员，都死伤无数，昔日的朋友、道友、战友等，今日早已身首异处，灭族之仇，生死之恨，不共戴天，就算他们想停下来，但也停不下来。
<br/>　　两族的实力势均力敌，底层的战斗，大多都有损伤，决定这场战斗的胜负的关键因素还是在于最顶级的战斗。也就是妖族的一帝三皇一妖师，十大妖神与巫族的十二祖巫，还有一些顶级的大巫之间的争斗。
<br/>　　战斗到了如今，巫族的大巫死了好多，妖族的妖神也死了两个，至于妖神以下者，那就更多了。
<br/>　　苍天在流泪，可是为何他流下的泪水是鲜红色的呢？
<br/>　　河水在倒流，可是为何那流淌的河水是血红色的呢？
<br/>　　高山在堆积，可是为何那险峻的高山上堆满了尸体与残甲？
<br/>　　那一幕幕，一堆堆、一片片，无时无刻不在深深震撼这洪荒大神通者，他们此时已被巫、妖两族之间的战斗深深震撼了。
<br/>　　“巫族蛮子，给本皇死来！”东皇太一一身伟岸，紫金色龙袍飞舞九霄，踏着霸道步法，庞然的气势席卷诸天，如同一尊帝王巡视天下，仿佛整个寰宇苍生都要臣服在他脚下，这种气势，如汪洋大海一般，深不可测！
<br/>　　他轻轻抬起手中混元剑，明亮的黑眸中透露着滔天杀意，有着无量煞气在他身上升腾而起，冲刷九天十地。
<br/>　　“哗啦——”
<br/>　　苍穹在轰鸣，九天在震动，天地仿佛被划开一道口子，嘹亮的剑鸣声仿佛从九霄之上传来，破开无尽空间，杀向烛九阴与祝融两大祖巫。
<br/>　　“太一小儿，也敢如此嚣张，看我不将你打成真正的扁毛雀！”祝融猛然怒吼，惊天动地，直接化身数万丈之高的祖巫真身，轰然间气势狰狞而发，古老而沧桑，狂暴而恐怖。
<br/>　　无尽的火焰从九天之上被他召唤而来，化身无尽炼狱，焚灭无数空间，多少妖兽葬身火海之中，无穷火海化为各种凶兽，冲向东皇太一，祝融化身火神，执掌诸天神火，挥动着无数火球，杀向东皇太一。
<br/>　　同时，时间祖巫烛九阴同样化身数万丈之高的祖巫真身，他仿佛来到时间的尽头，带着一股明灭不定的气息，一道道乳白色光辉，洒向洪荒大地，只见无尽生灵在迅速衰老，不过片刻时间，便化为飞灰，飘散空中。
<br/>　　同时，他手中演化出一道道时光轮，时光轮转，飞舞穹天之间，光轮所过之处，一个个妖族强者被带走无尽寿命，成片的妖族高手在倒下。不过眨眼之间，被时光轮带走无尽妖族的命。
<br/>　　“巫族小儿！尔敢！”东皇太一见此，目眦尽裂，脸色发狂，带着无穷凶气的混元剑，划破无尽虚空，斩向烛九阴。
<br/>　　轰！
<br/>　　“太一小儿，想走就走，问过我没有？哈哈！”就在此时，祝融挥动着神火拳头砸向东皇太一，阻断了太一的去路，太一无奈，只得继续与祝融厮杀。
<br/>　　“混蛋，既然你想死，就别怪本皇心狠手辣！啊！”太一发狂，手持凶剑，带着无尽气势，熊熊的太阳真火，与祝融拼杀起来。
<br/>　　在另外一处战场，妖帝帝俊挥动着手中的金色王剑，与祖巫帝江拼杀在一起，两者身上都受了不同程度的伤，虚空之中，金色血滴与淡青色血滴飞溅。
<br/>　　帝江祖巫虽然也有准圣中期强度的肉身，但奈何不修元神，拼杀还是敌不过妖帝帝俊，但巫族又是喜欢以战养战，战斗之中，帝江将巫族的强大战斗天赋发挥得淋漓尽致。尽管不敌帝俊，但他还是能够撑得住，之后又有强良祖巫加入到这处战场之中，与帝江联手，同时拼杀帝俊，三者之间，一时之间还分不出胜负。
<br/>　　另一边妖师鲲鹏与共工祖巫、风之祖巫天吴，三者之间同样斗得不可开交，鲲鹏因之前被帝俊太一算计，所以并没有发挥出全部力量。但尽管如此，他还是与两大祖巫僵持了下来。
<br/>　　“杀！”
<br/>　　鲲鹏化身出本体，鲲之大，不知其几万里，化而为鸟，其名为鹏，其翼若垂天之云，鲲鹏展翅，凤舞九天，张口一吸，便将无数生灵吞噬腹中。
<br/>　　“噗嗤——！”
<br/>　　天吴祖巫与公共祖巫一时不查，被这滔天一击扇中，直接倒飞数万里，胸口坍塌，大口大口的鲜血横流。
<br/>　　“岂有此理！鲲鹏小儿，你在找死！”两大祖巫显现出祖巫真身，数万丈自高的祖巫真身，带着无尽江河湖海，幻化出无数水龙，杀向鲲鹏，同时，天吴祖巫召唤出巨大龙卷风，吞噬无尽生灵，吹响鲲鹏。三者一时间战斗得不分胜负。
<br/>　　后土祖巫与玄冥祖巫两大女祖巫，与羲皇伏羲在战斗，伏羲手持伏羲琴，弹奏一曲战曲，铁血铮铮，战歌高昂，控制人心，杀人于无形之中，只是如今他的两色苍白，显然是心力过多运用，导致心力不稳。
<br/>　　他的对手，后土脚踏大地，无尽的大地之力加持其身，使得她的力量无穷无尽，永不枯竭。同样，玄冥祖巫，操控雨水，雨从天降，降下的不仅仅只是意境，同样也是一道道催命符。雨中带有生死杀机，落谁谁死，端是恐怖。
<br/>　　最后，不周山的娲皇女娲还是忍不住出手了，女娲不可能眼睁睁地看着自己的同胞被巫族无情地杀戮，所以，她出手了。
<br/>　　女娲一出手，便极为狠辣，一点也看不出身为女子的优柔寡断。也是，任谁看着自己的同族被无情屠杀，他出手时还能忍住不杀。若真是如此，那才是笑话。那样的人能够在洪荒活下来，才是一个奇迹。
<br/>　　各大妖族妖神，还有余下的祖巫、大巫，都在拼杀。种族之战，不是你死，就是我亡。除非一方倒下，或者出现特殊情况，否则它不到最后决不罢休。
<br/>　　大战在继续，谁也不知道它要到何时才会停止？也许是下一瞬，也许是成百上千年。但，无论结局如何，这一场战争，都不算真正的胜利。
<br/>　　这是劫，天地降劫，最终赢的，始终是天地。
