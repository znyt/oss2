　　在官道隘口停了很多马车，叶少阳这辆车的车夫下去打听了一下，带回来一个令人沮丧的消息：
<br/>　　两省交界的地段，有几股小的军阀纠合在一起，趁着皖军主力被抽调到到北京打仗，趁机进犯，跟留在安徽的一部分守军交上了火。
<br/>　　这种小规模的战争，对百姓的生活本没什么大的影响，但是巧的是，两方交火的地方，就是马车们通行官道所在的这个县城，皖军现在已经将县城戒严，任何车辆和百姓都无法出城。
<br/>　　赶上这种事，大家都没办法，于是就在县城住下来，等着时局好转。
<br/>　　叶少阳带着翠云找了一个客栈住下，在县城转悠，发现一切照旧，根本没有战争的迹象，不过城门紧闭，外面不是传来轰隆隆的炮声，说明打仗的地方是在城外。
<br/>　　这座皖南的小县城，跟翠云住的上蔡县也差不多，不过因为离宣州比较近，卖的东西也都是文房四宝之类的，但是买东西的多，逛街的却不多，街上还有很多拖家带口的流民，在街上乞讨。
<br/>　　还有一些穿制服的官兵，在县城里到处晃悠，随意盘查，据说是为了防止有敌方派特务进城搞破坏。叶少阳也被盘查了一次，不过好在他身上有居民证，这种居民证，在当时是相当于身份证一类的东西，上面有政府盖章，证明是某某人，家住在什么地方。
<br/>　　叶少阳这个穿越者，本来身上是没有这种东西的，不过之前在帮秋镇长家捉鬼之后，在翠云的提醒下，找秋镇长开了一个居民证，地址填的就是翠云家。秋镇长是一方父母官，开具这样一个居民证只是举手之劳，就这样，叶少阳在这个时代也算是有了合法的身份。
<br/>　　叶少阳一干人在这个县城一呆就是几天，叶少阳心急如焚，但也没有办法可想，只能干等。
<br/>　　等到第五天，送他们来的马车夫实在等不了了，表示自己不能在这里干耗，必须得回去了。乘客们一听都傻眼了，差点把马车夫暴揍一顿。但是马车夫态度很坚决，也愿意退一部分路费，并且已经帮他们打听了，从县城出去，有小路可以离开安徽，进入江西，他们可以从这条山路走过去，到了江西之后，再雇马车去往各自的目的地。
<br/>　　大伙没办法，也只能接受退钱，然后翠云带头跟马车夫谈判，最后要回了几乎一半的路费。
<br/>　　车夫赶着马车离开之前，特意跟他们说了一下，他打听出来的那条路很好找，但是城外山上有土匪，过路需要小心谨慎，千万别撞见土匪。
<br/>　　车夫走后，八个乘客在一起商量了一下，有乘客经常往来皖赣，对周围情况是有点熟悉的，知道路怎么走，但是正如车夫所说，附近山里住着几窝土匪，有点冒险，于是大家商量，当即就有三个乘客表示不去了。
<br/>　　叶少阳知道翠云是不想回去了，想了想，说道：“不然你就先留在这县城等我，我自己过去办事，等办完了我一定回来找你。”
<br/>　　“不，我跟着你！”翠云很坚定。
<br/>　　“这……你不怕土匪？”
<br/>　　翠云咬了咬嘴唇，然后摇了摇头。“你路上没人照顾，我不放心。”
<br/>　　叶少阳有些无奈，劝了她一番，但翠云态度十分坚决，让叶少阳很没办法，最后仔细思索了下，也只好同意了。
<br/>　　他对翠云的安危，实际上不是特别担心，毕竟自己虽然法力尽失，拳脚功夫还是有的，真要遇到土匪，对付十个八个应该不成问题。最主要是那个见多识广的老先生告诉他们，虽然说是有可能遇到土匪，但是几率还是很小的。而且内地的土匪也不像东北那么彪悍，一般都是求财，万一真遇到了，只要舍弃一部分财物，活命应该是没问题。
<br/>　　“只是……”那个老先生望着翠云的脸，说道，“恕老汉冒昧了，小嫂子长得俊，万一遇到土匪……”
<br/>　　他没有说下去，但是意思是明显的，叶少阳想到了从电视上看到的、略有喜感的一个名词：压寨夫人。
<br/>　　翠云笑道：“这个不用担心，我有办法！”
<br/>　　确定了要走这条山路之后，大伙约好了一个时辰后在这边见面，然后各自回自己住的客栈整理行李。
<br/>　　叶少阳在自己房间整理好行李之后，去找翠云，结果翠云让他过会儿在来，叶少阳等了差不多有半个小时，见她还不出来找自己，忍不住又去找她，在门外等了一会，屋里响起脚步声，然后房门打开，叶少阳以为是翠云，刚要开口，愣了一下，面前是一个身材臃肿、脸色发黑、长满了雀斑的中年妇女。
<br/>　　“阿姨，你……”叶少阳本想问她是从哪来的，突然又觉得有点面熟，仔细一看，顿时乐了，“我靠，我差点没认出来！”
<br/>　　翠云笑起来，说道：“像真的吗？”
<br/>　　叶少阳又仔细打量了一会，确实很难看出是化了妆的，顿时啧啧称奇，问她是怎么做到的。
<br/>　　“我家以前住过一个革/命党，是个姐姐，她很年轻漂亮，但是走南闯北不方便，就把自己化妆成很丑的样子，我当时觉得好玩，就跟她学了这种技法，其实不难，这些雀斑，是把黑豆压碎兑水搅拌，贴上去的，脸上抹了黄米汁，黄米汁干了之后会起皮，看上去跟皱纹一样。”
<br/>　　“所以你刚才去街上买杂粮，是为了这个？”叶少阳感到很好玩，目光落在她臃肿的腰身上，说道：“那怎么连腰都粗了？”
<br/>　　“这个更简单啊，我把毯子裹在腰上了，用布条裹的像一点，就是这样了。”
<br/>　　翠云笑着看他，说道：“这下你放下了，我这种没姿色的中年妇人，土匪不会看得上的。”
<br/>　　叶少阳觉得也是，土匪虽然大部分都是单身狗，对女人的要求估计低一点，不过也不至于会喜欢上这样一个大妈。
<br/>　　离开客栈之后，翠云又买了一块方巾，将一头秀发扎起来，看上去更像是一个中年妇女，身上穿的也是最普通的妇女穿的布衣，看上去就是个不起眼也不好看的中年大妈。
