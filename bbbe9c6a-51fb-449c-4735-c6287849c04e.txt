<br/>				“当然，这字迹错得了吗？”灰狼咧嘴一笑，道:“那易修染往鬼医身边凑，一看就是狼子野心，你说，主子不收拾他，收拾谁呢？”<br/>　　<br/>　　任翔看了看手中的字条，而后收了起来，道:“我知道了。”他站了起来，问:“你何时回去？”<br/>　　<br/>　　“我去瞧一眼那易修染后就回去。”<br/>　　<br/>　　灰狼说着，又道:“对了，主子过些日子要陪鬼医回去一趟，凤夜和赵阳两个小屁孩还活着，鬼医打算带他们回去，还有好像关习凛准备跟叶箐成亲，估计着这趟回去的话没那么快回来。”<br/>　　<br/>　　“嗯，我知道了。”他看向灰狼，道:“跟我来吧！”<br/>　　<br/>　　灰狼见此，咧着嘴笑着跟了上去。来到这里，其实他还挺想知道易修染那货在这里被整成什么样的。<br/>　　<br/>　　跟着任翔来到清风楼的二楼处，上了阁楼推门进去，当灰狼看见那斜椅在椅子上喝着酒的身影披着一件透明的薄纱时，不由的咧着嘴笑了起来。<br/>　　<br/>　　只是，笑容才一咧开，下一刻，他就瞪起了眼叫了一声:“怎么没绑起来？你居然就这样放着他这样在这里？你不怕他逃了？”<br/>　　<br/>　　房内的易修染没有被捆绑着，只是一身的灵力气息被封印住，如同寻常的凡人一般不起眼。但，他那出色的容颜以及气势，还有那举止投足间的贵气，却是生生的让人忽略了他身上只是简单的披着那件薄纱。<br/>　　<br/>　　他在房中喝着酒，那举止间的潇洒姿态，仿佛不是被人困在这里，而是成了这里的主人一般悠哉自在。<br/>　　<br/>　　也难怪灰狼看到后会怪叫一声，毕竟当初他可是被捆绑着的。<br/>　　<br/>　　易修染抿了口酒，目光冷冷的朝灰狼瞥了一眼:“凤九让你来的？”<br/>　　<br/>　　“你想多了，鬼医已经记不起你是哪号人物了，又怎么会让我过来呢！”灰狼轻哼一声走了进去，道:“实话告诉你吧！今天我来这里可是奉了我家主子的命令过来的，告诉你，最好不要再去招惹鬼医，那可不是你招惹得起的！”<br/>　　<br/>　　听见这话，易修染目光微眯，他盯着灰狼，道:“轩辕墨泽？他回来了？”<br/>　　<br/>　　“不错。&lt;&gt;”<br/>　　<br/>　　灰狼双手环胸，笑道:“你在这里不知道吧？前段时间魔族的人攻上星云仙宗，鬼医与魔主大战，最后我家主子也正好赶回来了，连魔主都被他打成重伤而逃，你就该掂记着点自己的身份的实力。”<br/>　　<br/>　　闻言，易修染没有说话，只是敛下了眼眸，不知在想着什么。<br/>　　<br/>　　“行了，我也要走了。”灰狼说着，看向一旁的任翔。<br/>　　<br/>　　任翔笑了笑:“要我送你就开口，看着我做什么？”声音一落，他迈步往外走去，对他道:“走吧！我送送你。”<br/>　　<br/>　　“这还差不多。”灰狼咧嘴一笑，道:“其实我还想找你喝几杯酒的，不过担心着路上耗时太久，所以还是算了，以后有机会我请你吧！”<br/>　　<br/>　　两人一边往外走去，一边聊着，出了清风楼的阁楼，往后院走去没多久，就听护卫匆忙来报。<br/>
