　　“大周天……不会吧。”毛小方震惊地不行，“天底下，真的有大周天吐纳心法？”
<br/>　　叶少阳拍了拍自己的胸脯，“别人不知道，反正我真的会。你也别跟我客气了，这心法是我偶然间得到的，不是我门派专属法术，传授给你，也没关系。”
<br/>　　毛小方不淡定了，虽然没喝什么酒，但是因为激动，一张脸涨得通红，突然对叶少阳抱拳，咬牙说道：“如果能学会的话，我不拒绝。多谢兄弟了！”
<br/>　　“吃好喝好，然后我教你！”
<br/>　　吃完饭，喝完酒，叶少阳将大周天吐纳心法的法门传授给他，等他完全理解之后，自己就让他尝试着吐纳，自己去睡觉了，醒来之后，看见毛小方怔怔地坐在床头发呆，问道：“这心法怎么样？”
<br/>　　“我……我吐纳了两个周天，气息未乱，收获的真气是小周天心法的好几倍！”
<br/>　　“当然了，”叶少阳表示自己早就知道这个结果，“我之前说了，以你的天赋，绝对能驾驭这套心法，不过你刚用，一天最多吐纳两个周天，不可以更多，将来可以增加次数。”
<br/>　　毛小方摸着下巴，沉吟道：“我之前感受了一下，以我的道心，一天最多也只能吐纳四个周天，我不贪多，这已经是非常大的奇迹了！”
<br/>　　随后想起什么，问叶少阳：“你每天可以吐纳几个周天？”
<br/>　　“八个。”
<br/>　　毛小方当场就震惊了，因为激动，声音几乎都颤抖起来：“大周天吐纳心法收获的真气，至少是小周天的三倍还多，就算是三倍，你一天吐纳八个周天，这就等于……普通人修行十几天，至少十几天！这……太不可思议了！”
<br/>　　“有什么不可思议的，不然你以为我怎么晋升的灵仙。”叶少阳有点沾沾自喜，随后想到自己的处境，顿时又有点不爽，叹息道，“只是现在丹田被毁，连这种小角色都想欺负我，等另外两位来了，估计更会不把我看在眼里。”
<br/>　　不被人看在眼里，在一些风轻云淡的宗师眼中，根本就是无所谓的事，青云子就是这样，而且瞒了一辈子，如果最后不是为了两个徒弟出头，可能到死都不会有人知道他真正的实力，但是叶少阳不同，他很不喜欢这种充当配角的感觉。
<br/>　　毛小方端起酒杯，敬了他一杯，说道：“有什么关系，你早晚会恢复的，到那个时候，你一定要在这些人面前露一手，扬眉吐气，到时候叫上我，我要看看目瞪口呆的样子！”
<br/>　　叶少阳喃喃道：“是啊，我也想啊，希望这一天早点来。”
<br/>　　在客栈里等了两天，第三天的早晨，叶少阳被妙心敲响了房门，告诉他卢晓清和陈晓宇已经来了，在曹雨兴的房间里，既然大家要一起行动，她还是希望能见个面。
<br/>　　叶少阳也觉得没什么，反正就算在讨厌，就当是多了两个曹雨兴。于是跟他过去。
<br/>　　“对了，我一个师姐也来了，这样我们又多了一个帮手。”
<br/>　　“你师姐？”叶少阳有点纳闷，“我记得你说过……你不是单传来着，哪里来的师姐？”
<br/>　　“并不是我真正的师姐，是峨眉山的俗家弟子，跟我关系特别好，这次听说我在这里，也跟着他们一起来了，不过她性格十分冰冷，你不要介意。”
<br/>　　叶少阳连忙说哪里。
<br/>　　毛小方却是突然问道：“那个，妙心师妹，你口中这位峨眉山俗家弟子，可是梧桐？”
<br/>　　妙心一愣，说道：“你怎么会知道？”
<br/>　　毛小方笑道：“梧桐师姐，名声太大，提到峨眉山二代弟子，第一个想到的就是她了，你又说她冷冰冰的，跟江湖传闻很是吻合，其次，他也是七大弟子之一，跟另外两个一起出现，也算情理之中，因此，贫道冒昧猜测。”
<br/>　　妙心点头说道：“你倒是很聪明的，正是梧桐师姐，你们跟我一起来吧。”
<br/>　　在走廊上过去的途中，叶少阳想着毛小方说的话，估计这个什么梧桐，也是个冰山美人类的，他实在是不喜欢跟这种妹子打交道，不过也只有硬着头皮进去了。
<br/>　　屋里只有曹雨兴，和两个二十多岁的男子，一僧一道，不过都是常服打扮，看上去也算是器宇轩昂，坐在茶几前面，曹雨兴在殷勤地给他们倒茶。
<br/>　　见面之后，妙心简单介绍了一下，那个和尚是卢晓清，道士是陈晓宇，两人都出身在一个叶少阳没听过名字的门派。
<br/>　　当介绍到叶少阳的时候，陈晓宇摆手说道：“知道了，听雨兴师弟刚说过了，茅山的外门弟子。”鄙夷之情溢于言表。
<br/>　　叶少阳也没搭理他，突然一抬头，看到窗外贴着一张五颜六色的鬼脸，一怔之下，立刻冲过去，那“鬼脸”却蹦了进来，两只手只扑叶少阳的面门。叶少阳反应极快，虽然不知道对方是什么东西，但是反手就是一掌拍过去，带着七成法力，打在那东西脸上，发出了一个响亮的巴掌，叶少阳自己手被震的生疼，不过那东西也被他震翻在地上，怪叫一声，起身又要朝叶少阳扑过来。
<br/>　　“劣奴，不得无礼！”陈晓宇呵斥了一声，那东西这才站在地上，叶少阳这才看清，是一只猴子，不过脸上红白相间，应该是一种狒狒，看上去跟鬼似的，也难怪自己刚才会认错了。
<br/>　　这东西坐在地上，一只手捂着脸，冲着叶少阳龇牙咧嘴，神情十分怨毒，似乎十分不服叶少阳刚才打了自己一巴掌。
<br/>　　“这是我养的妖仆，名为猴王。”陈晓宇走过来，对它轻声呵斥道：“一边去！”
<br/>　　猴王立刻有点委屈，跳到窗台上，狠狠瞪了叶少阳一眼，跳了出去。
<br/>　　妖仆……这家伙原来还养了妖仆。叶少阳有点不能接受。
<br/>　　虽然豢养邪物当仆从的事，从古时候就是有的，法术界并没有禁制，不过真正这么做的还真不多，毕竟豢养邪物这种事不是特别光彩，所以一般正修的法师都不会做这种事，叶少阳自己是个例外，还养了那么多……
