　　第432章
<br/>　　黄三元带来的消息，程大雷的确第一次知道，但并没有太出乎他的预料。
<br/>　　帝位之争，涉及的是帝国最高权力，在这个字眼上，天下又有几个人把持得住。这世上，有人沉迷于美色，有人嗜好美食，有人痴迷山川美景……但这一切的一切，在最高权力前，都显得浅薄。不仅仅是因为权力能带来什么，更重要的原因是，所谓最高权力是唯一的。而唯一的代表的含义是，这种东西只能你有，其他人都无法染指。
<br/>　　最高权力，自然带来最激烈的追逐，长安城争斗得不可开交，却也是不难想到。
<br/>　　“林少羽那边情况怎样？”程大雷问。
<br/>　　“很不好。”
<br/>　　这次是黄三元和高飞虎同时回答。
<br/>　　能斩掉林少羽的脑袋，就有资格坐上那世间唯一的龙椅。各路诸侯支持的代言人，都陷入疯狂之中，对于逃亡的起义军展开围追堵截。
<br/>　　他的幽州大本营，在第一时间就被击溃，留守幽州的人，被全部屠尽。准备逃回幽州的起义军顿时成了无家可归之人，只能带着残兵败将狼狈逃窜，而沿途守着他们的，则是一个个磨刀霍霍的强军。
<br/>　　攻破长安城时，林少羽俨然是天下第一名将，不管他是正义或邪恶，他都做到了世上从未有人做到的事情。
<br/>　　不过，他这天下第一陨落的速度实在有些太快了。短短时间，便从高高在上，成了无家可归的流浪犬。
<br/>　　程大雷敲打着桌案，一声接着一声，沉重且缓慢。
<br/>　　“大当家，咱们该怎么办？”黄三元。
<br/>　　当高飞虎和黄三元汇报的时候，和珅也在倾听，脑中飞速思考着这件事带来的变化。总得来说，这是一件好事，长安越乱，便越就方便他这边的行动。
<br/>　　不过黄三元的话，令和珅愣了愣。什么意思，这件事和咱们有什么关系。咱们不应该抓紧时间研究攻占凉州城的事情么。
<br/>　　却见程大雷整个人都兴奋起来，竟迫不及待从座位上坐起来，在房间内来回踱步，很显然正在思考着什么。
<br/>　　“管，当然要管了，都是自家兄弟，就算他真的做错了什么事，也该是咱们自己处理。能让别人去管么。”程大雷道。
<br/>　　黄三元和高飞虎眼睛一亮，二人今天过来，等的就是程大雷这句话。
<br/>　　“哈哈，我便知道么，大当家还是大当家，不会让自家兄弟落在别人手中。”
<br/>　　“大当家，你说怎么办吧，我们已准备好了，就等着大当家一声令下。”
<br/>　　和珅终于感觉到，某些不好的事情已经发生。自己想要说什么，却已经有些插不上嘴，三人表现得都很兴奋，话说得密不透风。
<br/>　　程大雷挥挥手：“将徐神机叫过来，说我有要事找他商量。”
<br/>　　这个时候，和珅才终于找到机会，道：“大当家，您真要去救林少羽，可凉州城的事情……”
<br/>　　“凉州城的事慢慢再议，反正也不着急这一天两天。我答应你，等我救林少羽回来，便按你的计划行事，由你全权指挥。”
<br/>　　程大雷也很是兴奋，兴奋的原因，不知是因为可以出去转一转，还是因为能够暂时逃避和珅带来的压力。
<br/>　　和珅觉得自己完全无法理解程大雷的所作所为。凉州垂手可得，可程大雷却要不远万里，去救天下人人皆欲杀的林少羽。
<br/>　　他感觉脑袋在嗡嗡作响，机智如他，一时竟也不知该说什么才好。
<br/>　　“这个……那个，大当家，你是否再考虑考虑，这个林少羽的事情，和我们没什么关系呐。”
<br/>　　程大雷白了他一眼：“怎么可以说没有关系呐，他可是我们的兄弟呐。自家兄弟的事，我能不管么。”
<br/>　　说着，程大雷伸手拍了拍他的肩膀，道：“这次是林少羽出事，下次如果你出事，也是一样的。”
<br/>　　和珅立在原地，脑海中一片空白。因为程大雷的话，狠狠的颠覆了他的世界观。外面关于程大雷的传言很多，太过匪夷所思的，如每天喝几斤人血，吃多少人肉的事情就不提了。总之，如果排个恶人榜，往前数一百年，往后数一百年，程大雷都可以榜上有名。
<br/>　　但今天和珅惊讶的发现一个事实，与外面的传言截然相反，程大雷不仅不是一个恶人，反而有可能是个……好人。
<br/>　　问题的关键在于，和珅不是呐。
<br/>　　而在这个弱肉强食的世道，很显然一个恶人要比好人活得更好。大当家心怀如此妇人之仁，又能够走多远呢。
<br/>　　程大雷在房间内踱步，与高飞虎几人交谈，时不时发出爽朗的笑声。
<br/>　　和珅没有参与他们的讨论，默默退出了房间。
<br/>　　徐神机很快到了，他现在过得很是清闲。几乎算是山寨内的清静散人，程大雷什么事也不让他管，他就乐得逍遥自在。虽然徐神机对于自己的能力没有一个正确的认识，但关于懒惰这一点，他自己也承认。
<br/>　　“大当家，你找我什么事？”徐神机边走进来边道。
<br/>　　“你来得正好。”程大雷看了他一眼：“我有件事要与你商量。”
<br/>　　“哈哈，大当家有什么解决不了的事，与我直说无妨，我帮大当家参谋参谋。”
<br/>　　“有你在我就放心啦。”程大雷点点头，问：“徐军师，现在天下人人想要少羽的命，除之而后快，依你看，这件事该怎么办？”
<br/>　　徐神机几乎没什么犹豫：“少羽出事，我们怎么可能不管，当是要将少羽救回来。”
<br/>　　“嗯。”程大雷道：“徐军师，你是山寨的第一军师，现在的状况，你拿个主意吧。”
<br/>　　“这个……呃……哈哈。”徐神机面色红润，胡子都要快因高兴而翘起来：“让我想想，让我想想，呵呵，我心中确实也有了主意。为救少羽，我们确实牺牲再多也在所不辞，我们直接倾巢而动，一举攻破长安。”
<br/>　　程大雷摇了摇头：“倾巢而动，路上消耗的时间太多，等我们到的时候，少羽可能已被人杀死。”
<br/>　　“大当家的意思是？”
<br/>　　“我……亲自去。”
<br/>(本章完)
