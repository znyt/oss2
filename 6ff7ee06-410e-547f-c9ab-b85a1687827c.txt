　　第244章
<br/>　　距离程大雷离开长安，已经有十几天时间，目的地一日一日近了。这天夜里，大家在野地露营。
<br/>　　一路行来，都是绕着城郭而行，满天下发了程大雷的通缉令，赏银高达五千两，他们也只能露营。
<br/>　　“前面是峡山，穿过峡山，到琴川大概还要走一个月。”
<br/>　　“这样也快了，就是不知道琴川是什么情况。咱们先歇着吧，秦蛮，你守上半夜，我守下半夜。”
<br/>　　秦蛮有露营的经验，先用炭火烧出一片空地，诸人把干粮烘烤着吃了。然后把篝火熄灭，在白地上铺上毯子，这样不用担心土地的寒气。
<br/>　　几人行了一天，都有些累了，没多久便窝着身子睡着。秦蛮拄着一杆大枪蹲在地上，百无聊赖的守夜。
<br/>　　其实也没有什么要提防的，无非是防一些野兽而已，不过，秦蛮已经在周围撒上硫磺，一般的小兽受不了这气味，轻易不会过来。
<br/>　　大概有一个时辰，程大雷起身将大枪扎在地上，去林地中方便。
<br/>　　几乎是他刚离开不久，三十步外的一棵大树上，密密的树叶中探出一个脑袋。
<br/>　　他手中握着一柄弩，弩箭顶端闪着绿芒，很显然淬了剧毒。
<br/>　　将弩箭瞄准程大雷，他窝在毯子上，鼾声平稳。
<br/>　　嗖！
<br/>　　破空声，如等待许久的毒蛇猛地窜起，男人嘴角抿起一丝弧度。
<br/>　　这时，骤变突声。
<br/>　　安睡的程大雷忽然伸出手，将飞过来的那支弩箭稳稳接住。
<br/>　　男人脸上表情一怔，猛然感觉一道杀机从背后袭来，他回过头来，看到秦蛮竟突然出现在自己身后。
<br/>　　一枪砸断他脚下的树枝，立足不稳，其人从树上跌了下来。
<br/>　　此人还没回过神来，枪已经压在他肩膀上，此人是个刺客，精于暗杀，然而正面对敌不如秦蛮。
<br/>　　程大雷和徐神机、徐灵儿过来。
<br/>　　“兄弟，跟着我们三天了，终于舍得出来了，说说吧，什么路子来的？”
<br/>　　程大雷说着，目光已落在男人身上。
<br/>　　姓名：王秀（默默无闻的优秀刺客）
<br/>　　年龄：32
<br/>　　技能：蛇行，毒术
<br/>　　隐藏属性：无
<br/>　　王秀看着周围四人神采奕奕的模样，瞬间明白，自己落入一个圈套中。当猎人布下陷阱时，却不知自己也同样落入一个更大的陷进中，猎人，猎物，身份可以随时转换。
<br/>　　忽然，王秀脸上露出一抹阴森森的笑容。
<br/>　　“程大雷，你不会活着走到琴川。”
<br/>　　说罢，他嘴里冒出一股黑水，秦蛮一惊：“不好，他咬破了毒囊。”
<br/>　　程大雷用剑鞘砸得他下巴脱臼，可已经来不及了，毒血冒出来，他的生机已经断绝。
<br/>　　如此决绝，令四人心生寒意。
<br/>　　“究竟什么人要杀我，我平时没得罪过什么人呐？”程大雷琢磨着。
<br/>　　“咦，大当家，你是不是对自己有什么误解，还没得罪过什么人，你得罪的人还少嘛。”
<br/>　　“……”程大雷。
<br/>　　秦蛮去搜他的身，程大雷提醒道：“别碰到他嘴里的毒血。”
<br/>　　一无所获，此人身上并没有任何表示身份的信息。徐神机将掉在地上的小弩捡起来，道：
<br/>　　“这好像是朝廷的制式武器？”
<br/>　　程大雷点点头，这一点他也同样看出来了。那么，是朝廷派的人？目前得到的信息太少，还分析不出什么。
<br/>　　“接下来的路小心些，我想还会有人来。”
<br/>　　夜幕里，寂静无声，只有群山巍峨，偶尔有几只夜莺寂寥飞过。
<br/>　　无人知晓，在这样的夜里，山坡，树梢，桥头隐着一个个人，这些人与黑夜融为一体，互相却也见不着，是以布谷鸟的名号传讯。
<br/>　　“来了么，来了么？”
<br/>　　“还没看见，没看见。”
<br/>　　“守好，守好，千万不能让他从这里过去。”
<br/>　　程大雷在黎明出发，天气太热的缘故，也就是黎明和黄昏天气凉爽时可以赶路。
<br/>　　“大当家，前面就到峡山了。”徐神机。
<br/>　　“小心提防起来，若朝廷不肯放过我们，派大军追来，凭我们几个人，怕是挡不住。”
<br/>　　正说着，秦蛮忽然一惊，道：“大当家，有动静！”
<br/>　　不用他说，程大雷也已经发现了，哒哒的马蹄声，草动树摇，从周围的山林中窜出一大波人。
<br/>　　这些人粗布麻衣，手中握着各种兵器，有胖有瘦，有高有矮，模样千奇百怪。
<br/>　　程大雷四人立刻围成一圈，程大雷握紧剑柄，眼绽杀机。
<br/>　　为首一个书生儒衫的男人走到前面，道：“请问你们那个是程大雷？”
<br/>　　“不认识。”程大雷和徐神机异口同声斩钉截铁。
<br/>　　书生男上下打量程大雷，忽然呵呵笑道：“佩剑背斧，模样平平无奇，不会错的。”
<br/>　　这平平无奇四个字从何谈起。
<br/>　　“这位风姿卓绝的定是神机妙算徐神机。”书生男笑着道：“还有这位方脸阔面，一身黑衣的定是大枪将秦蛮了。”
<br/>　　“某乃峡山风云寨寨主黄飞虎，特在此地迎接程当家。”
<br/>　　程大雷不屑道：“你们竟想到冒充山贼杀人？”
<br/>　　“呃，冒充二字从何说起，我们真是风云寨的，也真是在等程当家。”
<br/>　　声音落地，忽然有一洪亮的声音响起。
<br/>　　“那义薄云天，平平无奇的程当家在哪里，俺老胡来接你了。”
<br/>　　只见远处又有一拨人马赶到，为首一个大胡子冲在最前方。
<br/>　　“程当家在哪里，程当家在哪里？”
<br/>　　“呔，姓胡的，这里没有你的事，是我先接到程当家的。”
<br/>　　“你先见到就是你的，你叫他名字他会答应么？”大胡子。
<br/>　　“胡八刀，你是想和我争了。”
<br/>　　“便是与你争又如何，我们乱石寨什么时候怕过你们，程当家非去我乱石寨不可。”
<br/>　　程大雷、徐神机、秦蛮、徐灵儿面面相觑，皆感觉眼前的事情有些难以理解。
<br/>　　“程大哥在哪里，程大哥在哪里，俺萧弥峰来见你了。”一个黄衣大汉领着大队赶来。
<br/>　　“程将军，程将军……”一个五短身材的汉子赶到。
<br/>　　没多长时间，程大雷面前围了十几个队伍，有唤大哥的，有唤寨主的，有唤将军的，不一而足。看每个人的情绪，要是有妹子的话，必须让嫁给程大雷。
<br/>　　关键是，程大雷一个都不认识。
<br/>　　
<br/>(本章完)
