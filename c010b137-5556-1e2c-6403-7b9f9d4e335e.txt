　　群星闪烁。
<br/>　　就当楚风眠抓住了那第五枚星辰的一刻，在楚风眠的背后，群星之力，爆发而起。
<br/>　　一枚枚星辰之力，凝聚在了一起，化为了两个大字。
<br/>　　七杀！
<br/>　　这两个大字，将星辰之力，凝聚到了极致，如同一道利刃，斩杀一切。
<br/>　　在楚风眠的面前，那一本剑道古籍，猛然之间飞了出来，翻到最后一页，最新的一页，开始凝聚。
<br/>　　这一页的威力，几乎不亚于刚刚凝聚出的，属于四方飓风剑的那一页。
<br/>　　七杀两个大字，浮现而出。
<br/>　　这两个字血红的大字，一眼看到，都足以让人恐惧无比。
<br/>　　九玄仙术，不，应该是九玄仙术的第五式，七杀，在剑道古籍之中，彻底的凝聚完成。
<br/>　　楚风眠缓缓的睁开双眼，在他的嘴角之上，浮现出了几分笑意。
<br/>　　领悟出了这两道剑术，在跟少帝的交锋之中，楚风眠也会更有几分把握。
<br/>　　如今。
<br/>　　时间，已经是过去了两天半的时间。
<br/>　　楚风眠睁开双眼，只见樊老已经是站在了他的身旁。
<br/>　　樊老，已经是将那一条灵石长河，全部吞噬炼化，如今樊老身上的气息，已经是强横了不少。
<br/>　　已经是不亚于之前跟少帝交手之时的状态了。
<br/>　　虽然只是恢复了一成左右的实力，但是樊老现在的实力，也无比可怕了。
<br/>　　毕竟樊老的真正实力，可是天人境，哪怕是恢复了一成的力量，也拥有半步天人境的实力。
<br/>　　“领悟如何？”
<br/>　　樊老看到楚风眠苏醒过来，也是好奇的询问道，他的眼神之中，露出了几分好奇之色，隐隐预约，有着几分期待。
<br/>　　虽然说樊老，已经是将楚风眠，当做是第一剑帝的传承者了。
<br/>　　就连四方飓风剑，也教给了楚风眠，不过他也有些怀疑，楚风眠能不能领悟这一剑，。
<br/>　　毕竟，四方飓风剑，可是许多天人境的剑修，为未必足以领悟出的一剑。
<br/>　　“已经领悟完了。”
<br/>　　楚风眠随意笑着说道。
<br/>　　“勉强可以施展了。”
<br/>　　“哦？”
<br/>　　听到楚风眠的话，樊老的眼神之中，突然露出了几分怀疑之色。
<br/>　　论起难度，这四方飓风剑，要比起领悟三阳烈炎剑，还要难上数倍，他也有些怀疑，在这短短三日之内，楚风眠居然可以领悟完四方飓风剑？
<br/>　　看到樊老眼神之中的怀疑，楚风眠轻笑一声，手心一捏，祖龙至尊剑，当即从天涯剑匣之中出鞘。
<br/>　　同时打出的，还有数十道剑气。
<br/>　　轰隆！
<br/>　　剑气围绕在楚风眠手中的灵剑之上，当即一剑对着樊老斩杀而去。
<br/>　　无数剑气，跟风之灵力，凝聚在一起，化为飓风，碾碎一切。
<br/>　　这剑气化为了飓风，樊老都是不由的退后几步，躲避其锋芒。
<br/>　　“这是？四方飓风剑？”
<br/>　　看到这席卷而来的飓风剑气，樊老跟随第一剑帝许久，当然一眼就看的出来，这一招就是四方飓风剑。
<br/>　　只是樊老都有些没有想到，楚风眠居然是真的在这短短三日时间之中，领悟出这四方飓风剑。
<br/>　　四方飓风剑的难以领悟，樊老可是知晓的。
<br/>　　曾经第一剑帝，也试想过想要将他的剑术，传承下去。
<br/>　　但是无数天才，都难以达到第一剑帝的要求。
<br/>　　其中最为简单的要求之一，便是对剑道的领悟。
<br/>　　三阳烈炎剑的要求，是一天。
<br/>　　四方飓风剑的要求，则是五天。
<br/>　　三阳烈炎剑，楚风眠虽然已经领悟，但是那个时候樊老毕竟没有跟随在楚风眠的身边。
<br/>　　所以也不知道，楚风眠到底是花费了多久，才将三阳烈炎剑领悟的。
<br/>　　但是这一次，四方飓风剑就是之前樊老刚刚传授给楚风眠的。
<br/>　　现在短短三日，不，两日半的时间，楚风眠就已经是将四方飓风剑，领悟完成的，甚至足以施展出来。
<br/>　　这比起第一剑帝，曾经定下的要求时间，还要快上一倍。
<br/>　　樊老的眼神之中惊骇无比。
<br/>　　他跟随第一剑帝，见识过一个又一个时代之中资质绝顶的剑道天才。
<br/>　　可没有任何一人，可以跟楚风眠媲美。
<br/>　　逼退了樊老之后，楚风眠也是心神一动，那剑气飓风，轰然消散。
<br/>　　他出手，自然也不想伤了樊老。
<br/>　　刚刚只是樊老的眼神之中，那种怀疑，让楚风眠有些不爽，所以才特意施展出这四方飓风剑来，让樊老震惊一下。
<br/>　　现在看着樊老的表情，的确已经是被彻底的震惊到了。
<br/>　　“两日半的时间……”
<br/>　　樊老盯着楚风眠，许久说不出话来。
<br/>　　楚风眠的剑道资质，比起他想象之中的，还要可怕的多。
<br/>　　“少主的剑道资质，真是天下无双，老夫也是坐井观天了，不该怀疑少主，请少主赎罪。”
<br/>　　樊老看着楚风眠，恭敬的说道。
<br/>　　他对楚风眠的称呼，也已经是改成了少主，之前的轻视，如今已经是彻底的消失了。
<br/>　　之前樊老，认定楚风眠为第一剑帝的传承人，其实还有着几分被逼迫的意味。
<br/>　　毕竟在这关键时刻，樊老已经是找不出一位比起楚风眠更加出色的剑修了，所以只能够是楚风眠。
<br/>　　可如今，在看到楚风眠两日半的时间内，就将四方飓风剑，领悟出来之后。
<br/>　　这樊老心中对楚风眠的轻视，已经是彻底的消失了，彻底认定楚风眠为少主，足以继承第一剑帝的剑术了。

<br/>　　楚风眠笑了笑，也没有再说些什么。
<br/>　　两日半修炼成四方飓风剑，就让樊老震惊到如此地步了。

<br/>　　“好，有了少主，第一剑帝的剑术传承，绝不会断，这一次那太初剑锋，还有人皇罐碎片，少主一定可以得到。”
<br/>　　樊老的眼神之中都露出无比自信。
<br/>　　他在这净土佛寺之中，一直血邪魔帝的骸骨，在这净土佛寺之中的一道道剑气，也都是他的眼线。
<br/>　　在这净土佛寺之中，除了少帝之外，还有着大量的魔族强者。
<br/>　　不过以现在楚风眠的实力，凭借这四方飓风剑的厉害，哪怕是九劫古帝巅峰的魔族强者，楚风眠也足以说杀就杀。
