　　度心诀！
<br/>　　威能恐怖如斯！度心诀，乃元始所创元心印中延伸出来的一个神秘法诀，为心之大道的一个用法。
<br/>　　度心诀，可度尽世间诸仙，从本源内心入手，在其内心最深处，顺其自然的度化，十分诡异，强悍无比，这比后世佛门的普渡术还要狠，让人丧失自主性，如同傀儡，不，这比傀儡要高级多了。
<br/>　　一旦被度心诀度化之人，从此后，将其主人尊为最亲近之人，将一生遵从其主人，永世不变，哪怕元始要求他灭了他的亲人，他也会毫不犹豫地去做。
<br/>　　简而言之，被度心诀度化之人，是让一个强者心甘情愿的……做狗！
<br/>　　这可比什么控魂术，控心术、毒药等控人手段要高级多了，这简直无法比较，一个是以手段强行控制他人生死，另一个是从内心入手，顺其自然地度化他人，一高一低，立马分晓。
<br/>　　随着元始修为的提高，心道参悟的也越来越高，元心印的威力也就越来越强，心道秘术也会越来越多。
<br/>　　因心之一道没有参考，元始按照自己的遗愿将其分为几个境界，分别为：凡心境、冰心境、主宰境、红尘境、返真境等。
<br/>　　凡心境，身陷凡尘苦海游，心向大道彼岸行！
<br/>　　凡心境，意指身虽为凡身，却有一颗不惧艰难险阻、矢志不移的强烈向道之心。
<br/>　　冰心境，心若冰清，天塌不惊。万变犹定，神怡气清。忘我守一，六根大定。……内外无物，若浊冰清。尘垢不沾，俗相不染。……水流心不惊，云在意俱迟。一心不赘物，古今自逍遥。
<br/>　　主宰境，心的主人，唯有自己！一切外在，休想撼动。
<br/>　　若说冰心境，是内心深处和外在隔离开来，保持冰心之境的话，那么主宰境，却是一种超然。堵不如疏，心道达到主宰境的强者，不会刻意压制自己的情感，任由一切情感、外物冲击，一切尽在本人掌控之中。
<br/>　　红尘境，红尘如狱，众生如难，用红尘之气炼心，用众生之难筑心，心处红尘苦海之中，却仍能照破山河万朵。
<br/>　　返真境，心若琉璃，返璞归真，一举一动，大道自然。
<br/>　　返真境之后，仍有境界，只是元始还没有达到而已。元始至今才刚刚突破至红尘境，今后的境界，也只是他猜测推演出来的的，至于是否真实如此，那只有他今后经历过了，才会知道。
<br/>　　以元始如今红尘境的心道修为，元心印已经强大到一个不可思议的地步。什么读心术，控心术，度心诀，这只它延伸出来的法诀。但，就算是延伸出来的，其威力也不可小觑。
<br/>　　而敖玉作为第一个被元始用度心诀度化之人，显然是具有极大的运气。洪荒生灵何止亿万万，但只有他一人被选到，说他运气不逆天都不行。
<br/>　　“敖玉。”
<br/>　　“属下在！”
<br/>　　“你先将你龙族的一些情况报于本座，之后再将你所知龙族的道经功法尽数道来。”
<br/>　　“是，尊主！
<br/>　　龙族分五部，分别为天龙部、地龙部、玄龙部、黄龙部、烛龙部，每部……”
<br/>　　接下来，敖玉将龙族的一些基本情况讲给元始听，这也让元始了解到了龙族的强大之处，而作为与龙族等同势力的凤凰族、麒麟族又如何？不用想元始也知道，他们就算有所不如，也不比龙族差多少。
<br/>　　当敖玉说道龙族的众多宝贝时，就连元始都快要流口水了，这让他不得不感慨，龙族真是富有啊，虽然这些宝物大多对元始无用，但不妨碍他收集宝物啊，毕竟谁也不会嫌弃自己宝物多。
<br/>　　最后，敖玉又将他知道的龙族功法说了出来，包括他的血脉传承功法。这些功法对元始《玄黄不灭》第四重有用，故而，他要多收集一些。
<br/>　　“好了，敖玉，你现在的任务是回到龙族，为本座收集一百滴不下于你之血脉等级的龙族精血，至少三滴龙祖精血。
<br/>　　至于如何收集，那就是你的事了，本座不管过程，只管结果。”
<br/>　　在敖玉讲完后，元始想到自己即将修炼的《玄黄不灭》第四重所需要的龙脉之气，于是对着敖玉吩咐道。
<br/>　　至于为何不直接收集龙脉之气，而要收集真龙精血？那是因为元始在用过四象神兽精血后，发现神兽精血比龙脉之气好用多了。
<br/>　　之前推演功法之时是因为没有神兽精血，只好推演用龙脉之气，现在有了更好的真龙之血，何必用那次一等的龙脉之气呢。
<br/>　　况且，就算要用龙脉之气，也是要用能够化龙而出的龙脉。那样的龙脉，需要天时地利人和，还需要种种不可思议的逆天机缘，才能化形而出，是何等稀少。那样的东西，他哪知道何处会有？用更多精力去找那样的龙脉之气，还不如用真龙精血好用呢。
<br/>　　“是，尊主。”敖玉恭敬道。
<br/>　　“好了，你去吧。”元始对着敖玉挥了挥手，道。
<br/>　　“属下遵命。”敖玉行了一礼，转身向远处飞去。
<br/>　　“任务已经给你了，至于你能不能完成，就看你的造化了。呵呵！”元始对着远去的敖玉轻声笑道。
<br/>　　之后，元始便不再多管，而是向着东海深处，他推算到的方位飞去。
<br/>　　……
<br/>　　“你们这些废物，太子出去了这么久都不知道，还留着你们干什么！来人，拖出去砍了！”东海深处，龙宫之中，一个妩媚妖娆的贵妇坐在大殿之上，对着几个跪在地上的下人大声斥责道。
<br/>　　“娘娘饶命！娘娘饶命！小的们下次再也不敢了。”只见几个侍从跪倒在地，向着贵妇大声哭喊求饶。
<br/>　　“娘娘！娘娘！殿下回来了！”就在此时，一个侍女跑步过来对着贵妇小声喊道。
<br/>　　“回来了？玉儿回来了！在哪里？”贵妇站起身来，连声道。
<br/>　　只是还不等侍女回答，宫门之外就传来一阵痛哭声：
<br/>　　“母后，你可要给我做主啊！”
<br/>　　话音刚落，就见敖玉一身伤痕，满是虚弱，踉踉跄跄地走了进来。
<br/>　　“怎么了？玉儿！你这是怎么了？！到底是谁将你伤成这样的？”只见贵妇一脸担忧地扶起敖玉，紧接着就大声问道。
<br/>　　“呜呜呜！母后，你要给我做主啊，我本因在龙宫了待闷了，想出去逛逛，我只不过出去不远，就遇到麒麟族的人在我龙族地盘上撒野。我上去教训他几句，谁知——”敖玉一把鼻涕一把泪地说道。
<br/>　　“怎么了？”贵妇急声问道。
<br/>　　“谁知那麒麟仗着自己法力高强，呜呜，不仅没有听从我的话，还对我龙族百般侮辱，而且还杀了我的侍卫，抽了我的真龙精血啊！呜呜！”敖玉哭声道。
