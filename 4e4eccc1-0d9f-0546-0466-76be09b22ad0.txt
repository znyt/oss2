　　第185章
<br/>　　次日午时，戎族发动总攻。
<br/>　　一辆马车推着林少羽，他被绑在木桩上，做十字架状。
<br/>　　身后是戎族大军，手中握着弯刀、长矛。这就是金问道的计划，以林少羽为盾，向前稳定推进。
<br/>　　六百步，五百步，四百步……已进入射程，众人能更清晰看清林少羽如今的模样。
<br/>　　身上伤痕累累，血结成痂，有苍蝇飞虫围着他转，他嘴巴被绳索累着，模糊不清发出的声音是：
<br/>　　“杀了我，杀了我。”
<br/>　　杀了他的确可以解决很多事情，而在如今的距离，只需一箭就能完结他的生命。然后大家就能像以前一样活下去，毕竟大家已经守了二十多日，还该能继续守下去。
<br/>　　可是，谁有勇气射出这一箭？
<br/>　　即便明知道被戎族攻进来就是数千人的死亡，而只要杀死林少羽就能避免这些事情发生，但是，谁有勇气杀戮自己的弟兄？
<br/>　　守军的手摁在城墙上，骨节暴起，牙齿叩在一起，发出咯吱咯吱的声响。
<br/>　　三百步，更近……
<br/>　　城门缓缓打开，此刻一人一马从城内走出。
<br/>　　程大雷。
<br/>　　只有他一个人。
<br/>　　戎族发出一片整齐的惊呼，皆不知他要做什么。是开门投降……还是血战到底？
<br/>　　这两者都不是一个人能办到的。
<br/>　　程大雷身背鬼面斧，腰佩匹夫剑，头发披散开来，如仙如鬼。
<br/>　　他出得城门，没有停步，座下马继续前行。
<br/>　　二百步，一百步，五十步……
<br/>　　他在五十步外停下。
<br/>　　此时他已在戎族弓箭的射程，只要金问道一声令下，万箭齐发，他立刻会被打成筛子。
<br/>　　但金问道没有下令，因为今日程大雷当真有些奇怪。
<br/>　　林少羽距离程大雷最进，他口不能言，只能摇摇头，随之又点点头。
<br/>　　摇头是说你不该来，点头则是谢谢你能来。
<br/>　　“狗贼，去死吧！”
<br/>　　孟子云大喝一声，提着刀冲出来。他对程大雷恨之入骨，而程大雷的本事他知道，几乎相当于一个废物。今日离得怎么近，孟子云怎能不杀他。
<br/>　　急奔而来，在靠近程大雷马头时身体跃起，刀在空中闪过白光，凌厉的杀意向程大雷倾泻而下。
<br/>　　程大雷拔剑，收剑，若一切未曾发生，只有血从剑鞘中渗出来。
<br/>　　孟子云落地，喉咙裂开一道血线，随之浓烈的血涌出来，渗入黄土中，像极了一位风格强烈画家笔下的油画。
<br/>　　世间竟有如此快的剑，世间竟有如此浓烈的杀意。
<br/>　　戎族众将皆感觉脑后一寒，仿佛这剑是落在自己身上。
<br/>　　金问道此刻才注意到，程大雷身上竟然穿着蓑衣，他奇怪的看着程大雷。
<br/>　　“你这幅打扮做什么？”
<br/>　　“因为今日有雨。”
<br/>　　“有雨？”
<br/>　　金问道奇怪的睁大眼睛，看了眼天上的日头，春夏之交，日头高悬头顶，汗水一层层涌出来，这人竟然说今日有雨。
<br/>　　程大雷搭在一起的眼睛睁开：“本将军由神仙梦中传艺，师父是南极仙翁，天蓬元帅是我师弟，太白金星和我拜把子，本将军一口仙气就能吹得你们魂飞魄散。”
<br/>　　“啊！”
<br/>　　金问道忽然觉得莫名其妙，从程大雷今天一出来他就感觉莫名其妙，似乎自己被程大雷拉进一种奇怪的气场中。
<br/>　　就连金问道身后的众将也发出了笑声，这货该不会是吓疯了吧。想来想去，似乎只有如此才能解释他目前的奇怪行为。
<br/>　　面对众人的嘲笑，程大雷表情不变，他抬起左手，似乎神赏赐甘霖给世人。
<br/>　　“今日且看本将军赏你半日风雨。”
<br/>　　风不动，树不摇，蝉鸣越来越噪。
<br/>　　金问道愣了愣，随之在马上捧腹大笑。
<br/>　　“你这个人呐，还真有意思，实在没法子了，就幻想自己会神仙法术，还真是，还真是……哈哈。”
<br/>　　程大雷表情不变，微微侧头。
<br/>　　“该有风来。”
<br/>　　一丝凉意从众人脸上扫过，金问道一愣，看到树叶轻轻晃动起来。
<br/>　　“然后云聚。”
<br/>　　刚才还响晴薄日的天空，瞬间彤云密布，天色突然暗了几分。
<br/>　　“随之雷鸣。”
<br/>　　咔嚓，紫雷在空中闪过，紧跟着轰隆隆的雷鸣铺天盖地而来。
<br/>　　“当有雨落。”
<br/>　　瓢泼大雨扑面而来，像龙王发怒，天河泄洪，遥遥之水从九天之上而来。
<br/>　　金问道一声惊呼，险些从马上跌下来。他终于发现些什么，程大雷今日所作所为，作为人来讲的确有些奇怪，可若在『人』前再加一个仙字……那就是理所应当。
<br/>　　难道他真被神仙传艺，难道他真能呼风唤雨？
<br/>　　刚才程大雷所言像个笑话，然而如果他所说的是真的，那就不是可笑，而是可怖。
<br/>　　金问道尚且如此，其他人就更不必说了。
<br/>　　“啊，天蓬元帅真是他师兄，现在他师弟降下大雨来救他了！”
<br/>　　“这人是神仙呐，这人是神仙！”
<br/>　　金问道身后的兵丁一片大乱。
<br/>　　“呔！”程大雷拎出斧子，一声大吼：“尔等凡夫俗子，竟敢冒犯天威，不怕遭天谴嘛！杀，吃我杀！”
<br/>　　程大雷冒雨拎着大斧头冲过来。
<br/>　　戎族兵丁一时心慌意乱，竟然没有反应过来。
<br/>　　程大雷冲到马车前，一斧头便将十字架劈开。
<br/>　　此时金问道才反应过来，大手挥着：“杀，杀了他。”
<br/>　　装神弄鬼糊弄得了一时，糊弄不了一世，这世间真不缺不信神不信鬼的人。瞬间就有四五员大将向程大雷将程大雷团团围住。
<br/>　　有杜茂。
<br/>　　杜茂的狼牙棒硬撞上程大雷的斧头，只感觉双臂一麻，手中的狼牙棒竟然握不紧。心中一惊：此贼竟然如此了得，当初被他诓了不成。
<br/>　　“狗贼，我与你一天二地仇，三江四海恨，今日不是你死……”话还没说话，他座下马就塌了架。
<br/>　　程大雷收回斧头，口中大喊：“子龙何在！”
<br/>　　一队人从城门中窜出来，为首一员小将，白盔白甲，座下白龙马，首次亮银枪，如一道白色闪电般杀将过来。
<br/>　　“赵子龙，参见。”
<br/>　　能够七进七出，单骑冲锋的人自然不该是程大雷，而是那七进七出的常山赵子龙。
<br/>　　龙胆技能发作，一夫冲锋，万夫莫敌。
<br/>　　
<br/>(本章完)
