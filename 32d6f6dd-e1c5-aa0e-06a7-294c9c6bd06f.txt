<br/>        刚才是她第一次见他的父亲，还有他的后母，陆氏帝国那座庞大的基业他会回去继承吗，他会回帝都吗？
<br/>
<br/>        她知道帝都留给他的记忆是痛苦的，同样也是不善的，儿时的陆寒霆就是那些名门子弟眼里的怪物，神经病，他们不愿意容纳接受他，都不是他的朋友。
<br/>
<br/>        夏夕绾又想起陆寒霆的亲生母亲柳璎珞来，她没有机会一堵柳璎珞的真容，一堵这位帝都第一美人的绝色风采来。

<br/>        夏夕绾有些担心，现在他的身体怎么样了，失眠障碍还有心理疾病，有没有好转一些，还是持续恶化？
<br/>
<br/>        夏夕绾辗转反侧，更加的睡不着，她索性坐起身，又翻开了妈咪留下来的那本医典，想找一下有没有跟陆寒霆病情有关的信息。
<br/>
<br/>        但是她失望了，并没有。
<br/>
<br/>        翻到最后一页时，夏夕绾纤长的羽捷突然一颤，只见最后一页上写着上古禁术，以毒攻毒，以血炼药。
<br/>
<br/>        夏夕绾停在了这一页，她将这一行小字来回看了好几遍。

<br/>        ……

<br/>        但是很不幸，她走在半途的时候就觉得视线一黑，什么都看不见了。
