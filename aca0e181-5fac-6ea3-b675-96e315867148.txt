        &lt;h3&gt;第三卷第二百七十二章  易倾秋的喜与惊&lt;/h3&gt;
<br/>
<br/>        易倾秋没有想到陈子昂竟然决定让她担任副总经理之职，这让她心里又惊又喜，毕竟她年龄还小，论资排辈都轮不上她，但是她的心里也是雀跃的。
<br/>
<br/>        她给陈天天打电话：“天天。你下午有没有事情？”
<br/>
<br/>        易倾秋和陈天天年纪相仿，都是唯恐天下不乱的性格，所以俩个人能说到一起，玩到一起。
<br/>
<br/>        “我不要上课啊，下午还有一篇论文要写。”只要易倾秋给陈天天打电话，保准就是吃喝玩乐，要不就是怼人。
<br/>
<br/>        “不是，天天，我有喜悦想和你分享。”易倾秋喜滋滋的语气感染了陈天天。
<br/>
<br/>        “怎么了，脱单了？不，我不要狗粮，我昨晚就已经吃得饱饱的了。”陈天天幽怨的喊着。
<br/>
<br/>        “你昨晚吃狗粮了？谁如此大胆，削他，拆散他，敢给我们天天吃狗粮。”易倾秋火上浇油。
<br/>
<br/>        “我敢削嘛，一个个都是大咖，不但是大咖，还是我的长辈，我奶奶我爷爷，我姑姑，我叔叔，嘶，算了不说了，见面说，你在哪里？”陈天天想起见张倩楠时被张倩楠怼得无话可说的时候心里就憋屈的很。
<br/>
<br/>        “我还能在哪里？在给你卖命啊。”易倾秋将手机夹在肩膀上，侧着头边收拾着办公桌上的文件，边给陈天天打电话。
<br/>
<br/>        “给我卖命？得了吧？别给我镀金了，和我有什么关系呢？”陈天天手中敲打着键盘，她今天得将这篇论文写完。
<br/>
<br/>        “我可是在你们家公司上班，不是给你卖命是给谁卖命？”
<br/>
<br/>        “我们家公司和我有毛线关系？我先写论文，半小时之后见，我们去通怡坊吃小龙虾。”陈天天看了一下时间，再有二十分钟她的论文就可以完成。
<br/>
<br/>        “好，一会见。”易倾秋挂了电话，将手头的工作整理好，明天是需要交接的，只要明天的任命一发布，她就是公司副总了。
<br/>
<br/>        陈天天将车开到他们家公司的楼下，给易倾秋发了语音：“秋富婆，天天已经到楼下，速速来接驾！”
<br/>
<br/>        易倾秋微微一笑，陈天天这个性格似乎和他父亲一点也不像，陈子霖稳重老成，生的孩子却是皮豆豆。
<br/>
<br/>        易倾秋下楼看见陈天天眼巴巴的看着她：“等急了？”
<br/>
<br/>        “我已经等了你五分钟。上车。”陈天天似乎不满意易倾秋让她等了五分钟，噘着嘴。
<br/>
<br/>        “大小姐，我是从楼上跑下来的。”易倾秋故意喘了口气。
<br/>
<br/>        “好吧，原谅你了。”陈天天开着车载着易倾秋朝大唐通易坊驶去。
<br/>
<br/>        大唐通易坊是西城新兴的酒吧一条街，也是江家旗下产业，集吃喝玩乐一体，白天以吃为主，晚上以酒吧为主，整条街上古色古香，陈天天找了个地方将车停好，拉着易倾秋的手走进一家以龙虾为主的餐厅，俩个人点了一个蒜香龙虾，一份麻辣龙虾。
<br/>
<br/>        “我每次来都想吃两份不一样的，每种口味都想尝一尝。”陈天天是个小吃货，这一家的龙虾味道不错，成了她的据点，隔三差五就会带着朋友来吃一顿。
<br/>
<br/>        “你看看你，那小胖爪子，就知道吃吃吃，不怕胖死你啊。”易倾秋看着陈天天白白嫩嫩肉乎乎的小手拿着小勺子搅动着冰粉取笑陈天天。
<br/>
<br/>        “还行吧，一米七的身高，五十公斤，不算是太胖吧？”陈天天迷一样的自信，确实他们陈家人一个个生得眉清目秀，身材匀称，只是她脸部有点像她的奶奶，是鹅蛋脸，所以就显得有点肉肉的。
<br/>
<br/>        “你越来越自恋了。”
<br/>
<br/>        “那可不？我陈天天原本就是如花似玉的美少女，为什么不自恋？说，有什么好事情要分享给我？”陈天天吃了一口冰粉，含糊不清的问。
<br/>
<br/>        “我明天就取代了你父亲的位置，我成了公司的副总。”易倾秋想看到陈天天惊讶的表情。
<br/>
<br/>        “果然是陈疯子一到西城，西城就不一样了，她这是想要你给她卖命呢。”陈天天看着易倾秋那开心的样子泼了一瓢冷水给她。
<br/>
<br/>        “士为知己者死，我就想给她卖命，我才到公司不到两年的时间，能给我这样的位置是认可我的人品和能力。”易倾秋毫不犹豫的怼回去。
<br/>
<br/>        “那我的父亲呢？”陈天天问易倾秋，今天的会议她的父亲肯定没有参加。
<br/>
<br/>        “西城的老总。”
<br/>
<br/>        “老总不是江俞轩吗？陈子昂一直都不相信我爸爸的能力，让外人任老总都不让我爸爸任老总。”陈天天为父亲打抱不平。
<br/>
<br/>        “据说江总和陈总在一起搭档很多年了，俩个人配合默契，无往不利，陈总在北京，让江总任西城老总，也是对西城给予厚望啊，况且不都是你们陈家的产业？”易倾秋没有想到陈天天会有这样的想法。
<br/>
<br/>        “呵，配合默契？没见得他们有多默契，我爸爸从一开始就在西城，现在不是将西城做得很好？凭什么江俞轩一来就是西城老总？而且江俞轩还是兼职，陈子昂明显就是偏心，不相信自己的亲哥哥。”
<br/>
<br/>        “天天，你是不是误会什么了？”易倾秋睁大了眼睛，以前的陈天天不是这样的。
<br/>
<br/>        “我没有误会什么，我心里就是不爽，昨天吃饭的时候，有一个女人对我吆三喝四的，要不是我们家请客，我就请她滚出去了，而且那个女人据说是江俞轩的老婆。”陈天天义愤填膺。
<br/>
<br/>        “江俞轩的老婆？”从来都没有听说江俞轩有老婆，难道真的是有钱人喜欢隐婚？
<br/>
<br/>        “是啊，之前还一直以为和我小姑是一对呢，不过这样也好，我就无所顾忌了。”
<br/>
<br/>        “你想干什么？”
<br/>
<br/>        “追求江俞轩，我挺喜欢他的。”
<br/>
<br/>        “人家都有老婆了，你......”
<br/>
<br/>        “只要锄头舞得好，不怕墙头没有草。哈哈哈。”陈天天哈哈哈大笑，别的不说，就想要那个张倩楠不舒服。
<br/>
<br/>        “你疯了，你不怕你爹敲断你的腿？”易倾秋像看疯子一样看着陈天天。
<br/>
<br/>        “婚姻不分对错，爱情不论早晚，更不是拱手相让，我只是想让江俞轩和陈子昂明白这个道理而已。好了，别用那样的眼神看着我，来，恭喜你荣升老总。”陈天天收起笑脸，拿起一只小龙虾和易倾秋碰了一下：“来，干虾！”
<br/>
<br/>        “其实，天天，我叫你出来，不仅仅是庆祝，我们西城公司面临着危机，有可能会被人变相打压。”
<br/>
<br/>        “你是说那个叫夏颜的。”
<br/>
<br/>        “聪明！”
<br/>
<br/>        “你想要我怎么配合你？另外，那夏颜和我的生母关系很好。”
<br/>
<br/>        “你生母？”易倾秋将要说的话恰在喉咙处，没有说出来。
<br/>
<br/>        “是啊，是我妈的干女儿。”
