　　“噗嗤！”
<br/>　　太快了！
<br/>　　昆吾老祖仅仅只是惊骇，即便强大如他，也根本来不及逃避开来。
<br/>　　那“一点”瞬间就划过一条黑色轨迹，穿过了昆吾老祖的头颅眉心。虽然他头上也有一层强大的法力保护着，也有自己堪比准圣后期强大的肉体保护着！可，此刻那一点带着可怕的穿透力，不但穿透了空间，就连时间都隐隐受到影响。
<br/>　　刷！
<br/>　　道一印所过之处，仅仅留下一道黑线，穿透了昆吾老祖的头颅。
<br/>　　双方身体交错而过。
<br/>　　元始站在了昆吾老祖的后方。
<br/>　　昆吾老祖呆呆站在那里，他的头颅似乎没有任何伤口，可实际上是那攻击太快了，甚至影响到时空层面，才会出现肉眼看不见任何伤痕的迹象。
<br/>　　“死在这样的大道神通之下，我无话可说。”昆吾老祖喃喃道，“这是什么大道神通，超越了时间与空间，竟能以大罗之境瞬杀混元强者？”
<br/>　　太可怕了。
<br/>　　以大罗杀混元，不错，就是以大罗杀混元，他虽然现在是大罗圆满之境，但毕竟转劫之前曾是混沌魔神，又有生死石印这样可怕的法宝，战力早就堪比准圣圆满了。更何况眼前此人是瞬杀他，那就说明，此人能够杀死混元强者。
<br/>　　这绝对不是任何一种他知道的大道，仅仅还只是雏形，就有如此可怕！
<br/>　　因为各种法则奥妙都还未实质性显化，这应该还只是大道雏形。
<br/>　　可仅仅达到雏形就如此可怕！那等他完善后，真正走出这样的大道，那又应该有多强？恐怕只有那苍茫第一的盘古神尊才有如此可怕吧。
<br/>　　能死在这样的大道雏形之下，自己的确该荣幸了。
<br/>　　“蓬！”
<br/>　　紧接着，他的眉心处突兀的有鲜血缓缓流淌，他的神魂也早已破碎，不得轮回，他的内脏及识海，在这一刻，尽皆爆裂。他身体一软，直接倒在地上。
<br/>　　……
<br/>　　“呼！”
<br/>　　长呼一口气，元始转过身来，看着倒在地上的昆吾老祖，已经死了不能再死了。他眼中闪过一丝复杂之色，与昆吾老祖这一战，虽然不是元始这一生最为凶险的一战，但却是他重生以来最为艰难的一战。
<br/>　　昆吾老祖本身战力并没有超过准圣，达到混元强者的地步，就拿他本身实力来说，比起元始之前在不周之顶遇到的黑暗魔神还要弱一筹，但，他有一个比黑暗魔神还厉害的法宝——生死石印。
<br/>　　这是一件蕴含生死大道法则的灵宝，以生死之力形成世界，再以世界之力锁定敌人，生死互转，世界爆炸。想要逃脱世界的束缚，必须要有打破世界的力量，但，又有几人能有如此实力？
<br/>　　而且，生死法则也恰好是元始接触不深的法则，毕竟生死可是涉及到轮回之谜的法则。
<br/>　　不过，经此一战，元始也成功施展出来自己一直没有成功的一道印法——道一印。
<br/>　　道一印的成功创出，不仅仅意味着元始再创一道大道神通，实力大增，而且还意味着元始的大道更进一步。
<br/>　　元始之道，就在于万道合一，走出最为重要的一点，掌握最为初始的元。以“元”衍万道、万法、万物。
<br/>　　而现在，道一印的出现，意味着众道合一踏出了最为关键的一步。
<br/>　　大道的突破，元始颇为高兴，但令他更高兴的是，自己又得到了两件先天灵宝，虽然不说很富裕，但也终于脱贫了。
<br/>　　元始大手一抓，便将悬浮在空中的两件灵宝抓到手中，一件是昆吾老祖常用的昆吾神剑，另一件就是元始最为看重的生死石印。
<br/>　　昆吾剑被元始直接收了起来，这剑虽然是极品先天灵宝，但元始并不怎么看重。先不说他自己又不是剑修，就是比它威力大的灵宝元始也有，现在又多了一件，这固然值得高兴，但对元始实力的增加也不是太大。
<br/>　　他最感兴趣的还是手中另一件灰色石印，也就是令他差点阴沟里翻船的生死石印。
<br/>　　拿到石印，收起昆吾老祖的尸首，元始准备到山谷之中再仔细研究这枚石印。
<br/>　　只是，他刚要走，但看到此处一地破败不堪，周围万里之内，寸草不生，大地残破断裂，山河破碎，地火翻滚。
<br/>　　这简直就是一片地狱。
<br/>　　“呃？被玩坏了。”元始无语道。
<br/>　　到了他们这种级别的战斗，出现这种情况十分正常，毕竟，每一个人的实力都已经接近了混元，战斗起来，毁天灭地在正常不过了，这还是洪荒世界等级太高，就连周围空间都十分坚固，不到准圣的实力，连震动空间都十分艰难，更不要说穿梭空间、破碎空间了。
<br/>　　要是换一个低级的世界，现在就不是这个样子了，而是早已出现灭世的景象了。
<br/>　　“罢了，既然是本座搞出来的，那此地就与本座有一段因果了。若是一直如此，那样不好。既然如此，那么——”
<br/>　　“五行五帝，给我出——”
<br/>　　“皇帝土皇，平乱大地，梳理地脉！”
<br/>　　随着话音一落，只见一尊头戴帝冠，身穿黄色帝袍的神邸出现在虚空，他大手一挥，那破碎的大地就以一种肉眼可见的速度恢复起来，地脉凝聚，山川，山峰、山谷、大地，一切变得美好起来。
<br/>　　“赤帝火皇，沟通地火，平乱山火！”
<br/>　　“黑帝水皇，江河湖泊，行云布雨！”
<br/>　　“白帝金皇，金矿山石，镇压大地！”
<br/>　　“青帝木皇，花草树木，滋养万物！”
<br/>　　紧接着，虚空再次出现四道不同的神邸，他们一出现，这片空间就变得热闹起来，原先暴乱的地火，瞬间就变得温顺无比，顺着一个方向流去。天降大雨，渐渐演变江河湖泊，山川河流。
<br/>　　青帝木皇不知从何处找来一些草木，迅速播到各处，木皇之气一出，这些树木花草迅速长大了起来，转眼间，便长成参天大树，树木成精，灵药密布，百草丰茂，花香四溢。
<br/>　　前后不过片刻时间，原先因大战而变得破败不堪的大地眨眼间就变得鸟语花香，山河绚丽，日光明媚，树木郁郁葱葱，溪流潺潺，山势挺拔。
<br/>　　这就是一个人间仙境！
<br/>　　这是何等的可怕神通，翻手之间，造化天地，滋生万物。这简直太可怕了。
<br/>　　“这样就顺眼多了。呵呵！”
<br/>　　看着眼前大变样的天地，元始满意一笑。
