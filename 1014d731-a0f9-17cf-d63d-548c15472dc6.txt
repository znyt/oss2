　　第333章
<br/>　　大队人马涌入琴川关，这次进攻的人包括蛤蟆寨的一千人，云中龙三家的六七百人，再加上樊梨花的一些人，总兵力超过两千。
<br/>　　这些人闯入琴川关后，见人就杀，手中有用菜油做的简易燃烧瓶，四处放火。
<br/>　　极短时间内，琴川关就火光漫天，喊杀声惨叫声响成一片。
<br/>　　关内守军仓皇迎战，根本摸不清程大雷有多少人，有些人连兵器都没摸着，盔甲杂乱披在身上，还没弄清楚是怎么会事，往往就是身首异处。
<br/>　　程大雷的人也很乱，却是乱而有章，关于各人执行如何任务，这是早就安排好的。过去的这段时间，程大雷可不是闲极无聊一直和严笛逗闷子。他在严笛面前有多放松，蛤蟆寨就有多紧张。
<br/>　　秦蛮带人去占领马圈，将琴川关的马都放出来，张肥高飞豹带人四处放火，赵子龙占领兵器库，关鱼带人去攻占粮仓。
<br/>　　而程大雷的任务最重，他与樊梨花、云中龙领着二三百弟兄直扑将军府。
<br/>　　只要能在短时间内击杀莫鸣米，也就确定了此战的胜局。
<br/>　　快，必须快，必须在城内守军没有反应过来时，完成战斗。若给了守军反应时间，令他们稳住局面，程大雷这两千人，还真是不够杀的。
<br/>　　城内各个建筑的位置，程大雷事先已经摸清楚，情报自然是从严笛处得来的。
<br/>　　这时间，莫鸣米还在和严笛吃酒，二人都喝了不少，酒已喝得半醉，话也越说越多。
<br/>　　莫鸣米拍着严笛的肩膀，情真意切道：“军师，在所有人里面，我最信任的可就是你啊。因为我知道，谁都有可能背叛我，但只有你不会。”
<br/>　　“将军，从我跟着你那天，我这条命就是将军的。”严笛将杯中酒一饮而尽。
<br/>　　“好，好，好！”莫鸣米连赞三声，道：“军师，你该安个家了，我的小妾里你随便挑一个，屋里放个女人，知冷知冷，晚上也有人与你暖脚……”
<br/>　　说着说着话，二人都动了真感情，可下一刻，外面就传来喊杀声。
<br/>　　“报！”
<br/>　　一名仆人跌跌撞撞跑进屋子，气喘吁吁，结结巴巴道：“将军，蛤蟆寨的人，蛤蟆寨的人杀了过来！”
<br/>　　“什么！”
<br/>　　莫鸣米和严笛同时都是一惊，严笛脸色瞬间变得不正常了。
<br/>　　“他们有多少人？”
<br/>　　“不知道，乱哄哄一片，在城里不停杀人，至少有上万人！”
<br/>　　“他们哪里有这么多人？”莫鸣米暗暗嘀咕着。
<br/>　　“将军，如今该怎么办，您快拿个主意。”严笛。
<br/>　　莫鸣米一时之间没回过神，仍沉浸在震惊中。
<br/>　　他们是如何突破城门，又是从何处召来上万兵丁……太多问题令莫鸣米想不明白。
<br/>　　最想不明白的还是一件事：程大雷为何要打过来！
<br/>　　获胜之后，难道他不该安稳过日子，在琴川一地与自己平起平坐，难道还不是他的目的么。谁给了他野心，要来攻打自己，然后……消灭自己。
<br/>　　这一刻莫鸣米才发现自己错了，错得太离谱了。程大雷的目的从不是和自己平起平坐，那不是他的野心。
<br/>　　有些错，一次都不能犯，因为犯了就是死。
<br/>　　“将军，将军……”严笛连唤几声。
<br/>　　莫鸣米猛地抬起头来，双目如火炬。
<br/>　　“取我的兵器来！”
<br/>　　“军师莫慌，既然他们敢来，我就让他们有来无回。”
<br/>　　莫鸣米道了一声，踏出屋子，只见外面火光漫天，喊杀声惨叫声不绝于耳。
<br/>　　手下说得不错，要搞出这样大场面，兵力至少过万。
<br/>　　此刻已有许多兵丁涌到将军府，人群慌慌乱乱，有些人兵器都没拿。
<br/>　　“将军，现在我们怎么办！”
<br/>　　“大家从西门逃吧！对方人太多了！”
<br/>　　莫鸣米站在台阶上，忽然冷哼一声，高声喝道：“慌什么，要死我和你们一起死，现在随我一起杀敌。”
<br/>　　见到莫鸣米后，众人总算有了主心骨，大概齐又有了队形。莫鸣米粗略看了一眼，见自己面前的人大概有二三百人。
<br/>　　“弟兄们，随我杀！”
<br/>　　程大雷此刻也正带人杀到将军府，冲破大门，正好和莫鸣米撞了个正对面。
<br/>　　二人目光相对，都恨不得生吞活剥了对方。
<br/>　　莫鸣米将牙齿咬得咯吱咯吱响：“姓程的，你竟然敢来打我，不想活了么！”
<br/>　　“哼，别废话啦，这琴川关是你的么，俺老程才是明帝封的琴川关守将。你丫算个什么东西，雀占鸠巢也太久了，早该给老子腾地方了。”
<br/>　　程大雷骂了一通，看着面前琴川关的守兵。
<br/>　　“兄弟们，亲人们，你们平日被姓莫的欺负惨了吧，我是特意来救你们的。现在整个琴川关都已被我占了，弟兄们，只要你们杀了姓莫的，大家以后一起吃香的喝辣的。”
<br/>　　别说这些守军，就是莫鸣米听到后也是一激灵。难道蛤蟆寨的人真的已经占了琴川关。
<br/>　　“弟兄们别听他蛊惑。”莫鸣米气急败坏，这程大雷，到这时候还满口胡说八道：“姓程的，动手吧，今天咱们两个只能活一个。”
<br/>　　“慢！”
<br/>　　程大雷一挥手。口中高喝道：“严军师，还不动手，更待何时！”
<br/>　　严笛和莫鸣米同时一楞，莫鸣米的目光看向严笛。
<br/>　　“将军，莫听他挑拨，我没有。”严笛急忙道。
<br/>　　“我当然信得过你，这小子的话我怎么会相信。”
<br/>　　“哈，严军师，你忘了这些是什么了。”程大雷从怀里扬出一叠纸，飘飘扬扬，有几张落在莫鸣米手中。
<br/>　　严笛的笔迹莫鸣米怎么会不认得，他看得清清楚楚，上面自己每天吃什么饭，吃多少都写得明明白白。
<br/>　　他不愿意相信这是真的，可证据确凿，他却又不得不信。
<br/>　　怪不得程大雷能够神不知鬼不觉的杀进来，原来是在身边安插了间谍啊。
<br/>　　背叛不可怕，可怕的是，你最信任的人背叛了你。
<br/>　　看到莫鸣米露出失神的模样，程大雷嘴角露出一抹冷笑，杀机淋漓。
<br/>　　“弟兄们随我杀！”
<br/>　　手中大斧抡开，程大雷一马当先，直扑莫鸣米。
<br/>
