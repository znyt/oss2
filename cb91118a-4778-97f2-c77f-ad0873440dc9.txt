　　在这丛林之中，楚风眠灵识扫过，足以发现在其中有着无数妖兽。
<br/>　　在这无数丛林的最中心，一座巨大无比的山峰上，有着一座大门，其中蕴含这极为恐怖的龙威。
<br/>　　这颜古龙帝的洞府，应该正是在这山峰之上。
<br/>　　只是，一旦是想要接近这颜古龙帝的洞府，必须要对付那下方的无数妖兽。
<br/>　　这些妖兽，可不是什么弱小的存在，其中楚风眠察觉到，在其中最为强大的几尊妖兽，实力更是达到了生死境五重左右的实力。
<br/>　　远远超越了在场的武者，楚风眠对付起来，都极为困难。
<br/>　　白萱暗地里对着楚风眠传音道。
<br/>　　“我们上一次进入其中，就是碰到了这些妖兽，只能暂时退避。”
<br/>　　楚风眠心中暗自点了点头。
<br/>　　他也察觉到，这些妖兽，隐约之间正是都在守护这丛林中心的山峰。
<br/>　　一旦是想要穿越丛林，这些妖兽肯定会群起攻之。
<br/>　　“诸位，这颜古龙帝的洞府，就是在这丛林的最中心的山峰之中，不过在此之前，我们必须要对付了那些妖兽才行。”
<br/>　　司空轩冷冷的开口道，目光闪烁，不知道是在算计这什么。
<br/>　　“这些妖兽，我们跟他们战斗，只怕也不一定能够对付的了他们，所以我们必须要有人去引走他们才行。”
<br/>　　……
<br/>　　司空轩的话，让众人直接沉默起来。
<br/>　　按照司空轩的意思，在场的人，必须要有人去当做诱饵，去引诱这些妖兽。
<br/>　　引诱的危险性先不提，谁有能够保证引诱了这些妖兽之后，也可以进入到颜古龙帝的洞府中去？
<br/>　　想要在这些妖兽的口中活下来，都是极难的，更何况还要甩开他们。
<br/>　　麟木突然开口道。
<br/>　　“这颜古龙帝的洞府，本来就是司空兄先发现的，我们自当听从司空兄的安排。”
<br/>　　罗辰在一旁，也是开口附和说道。
<br/>　　“我也愿意。”
<br/>　　云公子也是点了点头道。
<br/>　　“不错，没有司空兄，我们也来不到这里，就全凭司空兄安排了。”
<br/>　　这三人，纷纷表态，听起来虽然大义凌然，但是实际上却是早就串通一气了。
<br/>　　这让一旁白萱的脸色，都变的极为难堪起来，这些人如今表态，自然是跟司空轩已经是传统一气的。
<br/>　　司空轩是绝不可能让他们去引诱妖兽送死的，要去的人，只有她跟楚风眠。
<br/>　　“怎么？白萱，楚风眠，你们二人不准备答应？”
<br/>　　云公子突然站出来，大义凌然的说道。
<br/>　　“我们能够来到这颜古龙帝的洞府，也都是多亏了司空兄，怎么现在你们想要过河拆桥？”
<br/>　　“做人可不能没有良心！”
<br/>　　罗辰在一旁也是附和说道。
<br/>　　“更何况，只是去负责引走妖兽罢了，等我们进入到了洞府之中，自然会帮助你们也进去的。”
<br/>　　“不错，白萱，楚风眠，你们二人听从我的安排，我也不会让你们吃亏的。”
<br/>　　司空轩冷声说道，隐约之间，也是一股灵力压迫而来，冷冷开口道。
<br/>　　“怎么样，考虑清楚了没有？”
<br/>　　“你们……”
<br/>　　白萱气的脸色通红，刚想开口说些什么，却是被楚风眠一把打断。
<br/>　　“我们二人，自当遵从。”
<br/>　　楚风眠开口道，脸色却是露出了几分不甘，但是似乎也抵不住颜古龙帝洞府的诱惑，留了下来。
<br/>　　听到楚风眠开口，司空轩的脸上露出几分诡异笑意道。
<br/>　　“那好，现在就由我来分配任务，一会就由我跟云兄二人，留在最后，以防不测，由我来负责开启这颜古龙帝的洞府。”
<br/>　　“麟兄跟罗兄，便是负责注意看看有没有漏网之鱼的妖兽，前去对付。”
<br/>　　“至于这引诱妖兽的事，就交给白萱跟楚兄了，楚兄毕竟可是有着真龙血脉，对付几只小小妖兽，应该不成问题，我们率先进入洞府，等楚兄解决了那些妖兽，也可以跟随进来。”
<br/>　　司空轩开口的一刻，大义凌然，似乎是公正无比。
<br/>　　但是实际上，却是将最为危险的任务，交给了楚风眠跟白萱，至于他们，则是捡个现成的，前去颜古龙帝的洞府。
<br/>　　听着司空轩的话，白萱的脸色已经是气的通红，想不到跟随司空轩前来，居然是选择将她，当做诱饵。
<br/>　　就连楚风眠的心中，都是杀机涌现，这司空轩将众人先带到这颜古龙帝的洞府之中，才撕破脸皮。
<br/>　　现在就算是走，也无路可走，要独自离开的话，将要一个人面对危险的乱葬海，危险无比，根本不可能活着离开。
<br/>　　“司空兄好算计，让这二人先进入到颜古龙帝的洞府来，再对付他们，他们现在就算是走，也只有死路一条，只能是听从司空兄的安排。”
<br/>　　云公子冷笑连连，向着司空轩传音道。
<br/>　　“我就说司空兄为什么带来了这样两个累赘，原来是早就想好了的。”
<br/>　　“哼，白萱那个贱人，我追求她许久，她居然敢拒绝我，今日便是给她一个教训的时候。”
<br/>　　司空轩脸色阴寒的开口道。
<br/>　　“至于那小子，他的血脉有些特殊，正好是让他去引走那些妖兽。”
<br/>　　“我们四个人联合起来，就不怕他们二人不听话。”
<br/>　　司空轩悄悄传音一声，便是转过头来，看向楚风眠冷冷的开口道。
<br/>　　“楚风眠，你先下去，看看到底有多少妖兽。”
<br/>　　这司空轩的一句话，简直就是命令了，他开口的一刻，那云公子，麟木，罗辰三人，都是催动身上的灵力。
<br/>　　一股股庞大力量，直接便是压迫过来，其中的威胁之意，不言而喻。
<br/>　　“司空轩，你不要太过分了！”
<br/>　　白萱再也忍受不住，怒声吼道。
<br/>　　“我们跟你来是合作的，不是作为你的奴隶为你做事的！”
<br/>　　“怎么白萱，你敢反抗？”
<br/>　　司空轩还没开口，麟木便是率先一步的冷哼一声道。
<br/>　　“这安排可是我们大家都同意的，怎么你不服气？”
<br/>　　云公子也是脸上露出几分寒意，露出几分阴冷笑容道。
<br/>　　“司空兄的安排，自然有他的道理，既然大家都是为了这颜古龙帝的洞府来的，你还是要乖乖听话才行。”
<br/>　　“走。”
<br/>　　在这威逼之下，楚风眠似乎是无法反抗了，只能是身形一动，带着白萱，直接向着下方的丛林中飞去。
